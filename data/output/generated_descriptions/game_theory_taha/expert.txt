name: Two-Player Zero-Sum Game
description: The Two-Player Zero-Sum Game is a classical problem in game theory where two players, often referred to as the row player and the column player, compete against each other in a game where one player's gain is equal to the other player's loss. The game is represented by a payoff matrix, where each entry in the matrix corresponds to the payoff for the row player when they choose a particular row and the column player chooses a particular column. The goal of each player is to maximize their expected payoff, assuming the other player is also playing optimally.

variables: $x_1, x_2, ..., x_{rows}, v$ 
These variables represent the mixed strategy for the column player, where $x_i$ is the probability of choosing column $i$, and $v$ is the expected value of the game for the column player.

constraints: $\forall i \in \{1, 2, ..., rows\}: v - \sum_{j=1}^{cols} (x_j \cdot game_{j,i}) \leq 0$, 
$\forall i \in \{1, 2, ..., rows\}: x_i \geq 0$, 
$\sum_{i=1}^{rows} x_i = 1$ 
These constraints ensure that the expected value of the game for the column player is less than or equal to the expected payoff for each possible row choice by the row player, that the probabilities of choosing each column are non-negative, and that the probabilities sum to 1, respectively.

objective: maximize $v$ 
The objective of the problem is to maximize the expected value of the game for the column player, which corresponds to finding the optimal mixed strategy for the column player.
