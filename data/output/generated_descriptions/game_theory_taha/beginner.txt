Game Theory Problem (2 Player Zero Sum Game)
This problem is about two players competing in a game where one player's win is the other player's loss. 
The main variables are: 
- x, which represents the strategy or moves of the first player, 
- v, which represents the value or outcome of the game, 
- game, which is a table showing the possible outcomes of the game for each combination of moves.
The basic restrictions are: 
- the first player's moves (x) must add up to 1, 
- each move (x) must be a non-negative number, 
- the outcome (v) must be less than or equal to the sum of the products of each move and its corresponding outcome in the game table.
The goal is to maximize the outcome (v) of the game for the first player.
