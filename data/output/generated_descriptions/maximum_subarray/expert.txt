name: Maximum Subarray Problem

description: The Maximum Subarray Problem is a classic problem in computer science and mathematics that involves finding a contiguous subarray within a one-dimensional array of integers that has the largest sum. The problem allows for the subarray to be empty, in which case its sum is considered to be 0. This is useful when all elements in the array are negative, as the maximum subarray sum would then be 0, corresponding to an empty subarray.

variables: $n, a_1, a_2, ..., a_n, ans_1, ans_2, ..., ans_n, tot\_sum, from, to, from\_answer, to\_answer$ 
These variables represent the following: $n$ is the number of elements in the array, $a_1, a_2, ..., a_n$ are the elements of the input array, $ans_1, ans_2, ..., ans_n$ represent the elements of the maximum subarray, $tot\_sum$ is the sum of the elements in the maximum subarray, and $from$, $to$, $from\_answer$, $to\_answer$ are indices used to define the start and end of the maximum subarray.

constraints: $a = [-1, -2, 3, 5, 6, -2, -1, 4, -4, 2, -1]$, $from, to \in [1, n]$, $from < to$, $tot\_sum = \sum_{k=from}^{to} a_k$, $(tot\_sum > 0 \leftrightarrow from\_answer = from \wedge to\_answer = to)$, $(tot\_sum \leq 0 \leftrightarrow from\_answer = 0 \wedge to\_answer = 0)$, $ans_k = a_k$ if $from\_answer \leq k \leq to\_answer$, $ans_k = -999$ otherwise
These constraints define the input array, the relationship between the indices and the sum of the subarray, and the conditions under which the maximum subarray is considered non-empty.

objective: maximize $tot\_sum$ 
The objective of the problem is to maximize the sum of the elements in the subarray, which corresponds to finding the maximum subarray.
