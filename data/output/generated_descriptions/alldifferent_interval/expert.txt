name: All Different Interval Problem
description: The All Different Interval problem is a constraint satisfaction problem where the goal is to assign values to a set of variables such that each variable belongs to a distinct interval, and the intervals are defined by a given size.
variables: $x = (x_1, x_2, ..., x_n)$, $intervals = (intervals_{i,1}, intervals_{i,2})$, $size\_interval$ - These are the decision variables of the problem, where $x_i$ represents the value assigned to the $i^{th}$ variable, $intervals_{i,1}$ and $intervals_{i,2}$ represent the lower and upper bounds of the $i^{th}$ interval, and $size\_interval$ represents the size of each interval.
constraints: $alldifferent(x)$, $intervals_{i,1} < intervals_{i-1,2}$, $intervals_{i,2} - intervals_{i,1} = size\_interval - 1$, $x_i \geq intervals_{i,1}$, $x_i \leq intervals_{i,2}$, $intervals_{k,1} = size\_interval*k$, $intervals_{k,2} = size\_interval*k+size\_interval-1$ - These constraints ensure that all variables take distinct values, the intervals do not overlap, the size of each interval is consistent, each variable is within its assigned interval, and the intervals are defined by the given size.
objective: The objective of the problem is to find a satisfying assignment of values to the variables, there is no explicit objective function to minimize or maximize, the goal is to satisfy all the constraints.
