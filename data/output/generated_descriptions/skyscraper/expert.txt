name: Skyscraper Puzzle
description: The Skyscraper Puzzle is a classic constraint satisfaction problem where the goal is to reconstruct a 2D grid representing the heights of skyscrapers in a city, given certain clues about the number of skyscrapers visible from different directions.
variables: $x_{i,j}, cu_i, cl_i, rl_i, rr_i$ where $i, j \in \{1, \ldots, n\}$. These variables represent the height of the skyscraper at position $(i, j)$, the number of skyscrapers visible from above column $i$, the number of skyscrapers visible from below column $i$, the number of skyscrapers visible from the left of row $i$, and the number of skyscrapers visible from the right of row $i$, respectively.
constraints: $\forall i \in \{1, \ldots, n\}: \text{all\_different}([x_{i,j} | j \in \{1, \ldots, n\}]) \land \text{all\_different}([x_{j,i} | j \in \{1, \ldots, n\}])$, $\forall i \in \{1, \ldots, n\}: \text{num\_skyscrapes}([x_{i,j} | j \in \{1, \ldots, n\}], rl_i) \land \text{num\_skyscrapes}([x_{i,n-j+1} | j \in \{1, \ldots, n\}], rr_i)$, $\forall i \in \{1, \ldots, n\}: \text{num\_skyscrapes}([x_{j,i} | j \in \{1, \ldots, n\}], cu_i) \land \text{num\_skyscrapes}([x_{n-j+1,i} | j \in \{1, \ldots, n\}], cl_i)$. These constraints ensure that each row and column contains each height exactly once, and that the number of skyscrapers visible from each direction matches the given clues.
objective: satisfy
