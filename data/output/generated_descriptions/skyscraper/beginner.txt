Skyscraper Puzzle
The Skyscraper Puzzle is about placing buildings of different heights in a grid, following certain rules. 
The main variables are: 
- x: a 2D grid representing the heights of the buildings, 
- cu and cl: arrays representing the number of skyscrapers visible from the top and bottom of each column, 
- rl and rr: arrays representing the number of skyscrapers visible from the left and right of each row.
The basic restrictions are: 
- each row and column must contain each height exactly once, 
- the number of skyscrapers visible from the top, bottom, left, and right of each row and column must match the given values.
The goal is to find a solution that satisfies all the constraints, which means finding a valid arrangement of buildings in the grid.
