name: Checker Puzzle Problem

description: The problem involves placing n checkers on an n x n checkerboard with certain restrictions. Each column must have exactly one checker, and for the first i rows, they cannot have exactly i checkers. The goal is to count the number of ways to place the checkers under these conditions.

variables: $x_{i,j}, sum\_rows_i$ 
These are decision variables where $x_{i,j}$ represents whether a checker is placed at row i and column j (1 if a checker is placed, 0 otherwise), and $sum\_rows_i$ represents the number of checkers in row i.

constraints: $\sum_{i,j} x_{i,j} = n$, $\forall j \sum_{i} x_{i,j} = 1$, $\forall i sum\_rows_i = \sum_{j} x_{i,j}$, $\forall i \in [1..n-1] \sum_{j=1}^{i} sum\_rows_j \neq i$ 
These constraints ensure that the total number of checkers is n, each column has exactly one checker, the number of checkers in each row is calculated correctly, and the first i rows do not have exactly i checkers.

objective: The objective of this problem is not to minimize or maximize a specific value, but rather to count the number of feasible solutions that satisfy all the given constraints.
