Job-Shop Scheduling Problem
The problem is about scheduling jobs on machines in a way that minimizes the time it takes to complete all the jobs. Imagine a factory with several machines and many jobs that need to be done. Each job needs to be processed on several machines, one after the other, and each machine can only work on one job at a time.

The main variables are:
- Jobs (n): the number of jobs that need to be done
- Machines (m): the number of machines available to do the jobs
- Sigma: the order in which each job needs to be processed on the machines
- P: the time it takes to process each job on each machine
- X: the time when each job starts on each machine
- Y: a variable that helps decide which job to do first on each machine
- Z: the total time it takes to complete all the jobs

The basic restrictions or rules are:
- Each machine can only work on one job at a time
- Each job needs to be processed on several machines in a specific order
- A job can only start on a machine after it has finished on the previous machine

The goal of the problem is to minimize the total time (Z) it takes to complete all the jobs by finding the best order to do the jobs on each machine.
