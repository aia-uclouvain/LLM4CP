name: N-Queens Problem
description: The N-Queens problem is a classic problem in the field of constraint programming and computer science. It involves placing N queens on an NxN chessboard such that no two queens attack each other. A queen can attack another queen if they are in the same row, column, or diagonal. The problem is to find a configuration of queens on the board that satisfies these constraints.
variables: $queens = (q_1, q_2, ..., q_n)$, where $q_i$ is a decision variable representing the column in which the $i^{th}$ queen is placed, and $q_i \in \{1, 2, ..., n\}$. These variables represent the positions of the queens on the board.
constraints: $\forall i, j \in \{1, 2, ..., n\} \text{ where } i < j: (q_i \neq q_j) \land (q_i + i \neq q_j + j) \land (q_i - i \neq q_j - j)$, which ensure that no two queens are in the same column or on the same diagonal.
objective: The objective of the problem is to find a satisfying assignment of values to the decision variables, i.e., a configuration of queens on the board that satisfies all the constraints. The problem does not have a specific objective function to minimize or maximize, but rather seeks to find a feasible solution.
