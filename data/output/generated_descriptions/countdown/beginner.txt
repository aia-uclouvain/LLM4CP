Countdown Problem
The Countdown problem is about using basic math operations like addition, subtraction, multiplication, and division to combine a set of given numbers to get as close as possible to a target number.
The main variables are the given numbers, the target number, and the operations used to combine the numbers. In plain language, these variables mean: 
- The given numbers are the set of numbers that can be used to try to reach the target number.
- The target number is the number that we are trying to get as close to as possible.
- The operations are the basic math functions like addition, subtraction, multiplication, and division that can be used to combine the given numbers.
The basic restrictions or rules of the problem are: 
- Each given number can only be used once.
- The operations must be used in a specific order to combine the numbers.
- The result of each operation must be used as the input for the next operation.
The goal of the problem is to find a combination of operations that gets as close as possible to the target number, or exactly reaches it if possible.
