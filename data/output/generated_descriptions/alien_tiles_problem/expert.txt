name: Alien Tiles
description: The Alien Tiles problem is a constraint programming problem where the goal is to find a matrix of clicks on a grid of tiles, each of which can be one of a given number of colors, such that the total number of clicks is maximized and no faster way of reaching the same state exists.
variables: $clickMatrix_{i,j}$, where $i$ and $j$ are indices of the grid, $1 \leq i,j \leq gridSize$, and $clickMatrix_{i,j}$ represents the number of clicks on the tile at position $(i,j)$, which is a decision variable that determines the number of times each tile is clicked.
constraints: $\forall simplerClickMatrix \in \mathbb{Z}^{gridSize \times gridSize}$, $(\sum_{i,j} clickMatrix_{i,j}) \leq (\sum_{i,j} simplerClickMatrix_{i,j}) \vee \exists x,y \in dGridSize. ((\sum_{index} clickMatrix_{index,y} + clickMatrix_{x,index}) - clickMatrix_{x,y}) \mod colours \neq ((\sum_{index} simplerClickMatrix_{index,y} + simplerClickMatrix_{x,index}) - simplerClickMatrix_{x,y}) \mod colours$, which ensures that either the total number of clicks in the current solution is less than or equal to the total number of clicks in any other possible solution, or there exists a tile where the effect of the clicks is different between the two solutions.
objective: Maximize $\sum_{i,j} clickMatrix_{i,j}$, which represents the total number of clicks on all tiles.
