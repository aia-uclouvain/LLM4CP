name: Minesweeper
description: The Minesweeper problem is a classic puzzle where a player is presented with a grid of cells, some of which contain hidden mines. The player is given clues about the number of mines adjacent to each cell, and the goal is to determine the location of all the mines.
variables: $x_{i,j} \in \{0,1\}$ for $i = 1,\ldots,r$ and $j = 1,\ldots,c$, where $x_{i,j} = 1$ if there is a mine in cell $(i,j)$ and $x_{i,j} = 0$ otherwise. These variables represent the presence or absence of a mine in each cell of the grid.
constraints: $\sum_{(i',j') \in N(i,j)} x_{i',j'} = game_{i,j}$ for $i = 1,\ldots,r$ and $j = 1,\ldots,c$, where $N(i,j)$ is the set of neighboring cells of $(i,j)$. These constraints ensure that the number of mines adjacent to each cell matches the given clue.
objective: The objective of the problem is to find a feasible solution that satisfies all the constraints, as there is no explicit objective function to minimize or maximize. The goal is to determine the location of all the mines, given the clues provided in the grid.
