Kaprekar's Constant Problem
The problem is about finding a special number, known as Kaprekar's constant, which is calculated by repeatedly applying a simple operation to a number with a certain number of digits.
The main variables are: 
- y: the original number with n digits
- x: the result of the operation applied to y
- base: the base of the number system (e.g. 10 for decimal)
- n: the number of digits in y
The basic restrictions or rules are: 
- y must have n digits
- y must be a decreasing sequence of digits (e.g. 9, 8, 7, ...)
- the operation applied to y involves reversing the digits, subtracting the reversed number from the original, and getting a new number x with the same digits as y (but possibly in a different order)
The goal of the problem is to find the value of x, which is Kaprekar's constant for the given base and number of digits.
