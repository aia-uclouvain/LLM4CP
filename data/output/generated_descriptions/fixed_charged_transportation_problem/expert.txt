name: Fixed-Charge Transportation Problem (FCTP)
description: The Fixed-Charge Transportation Problem is a classical transportation problem with an additional fixed cost imposed on each transportation link if there is a positive flow on that link. The problem involves finding the optimal amount of goods to transport from a set of sources to a set of customers while minimizing the total transportation cost, which includes both variable and fixed costs.
variables: $x_{i,j}, y_{i,j}$ where $x_{i,j}$ represents the amount shipped from source $i$ to customer $j$ and $y_{i,j}$ is a binary variable indicating whether some amount is shipped from $i$ to $j$ (1) or not (0).
constraints: $\sum_{j} x_{i,j} = supply_i$, $\sum_{i} x_{i,j} = demand_j$, $x_{i,j} \geq 0$, $x_{i,j} \leq min(supply_i, demand_j) \cdot y_{i,j}$, which ensure that the supply at each source is met, the demand at each customer is satisfied, the amount shipped is non-negative, and the amount shipped is zero if the link is not used.
objective: minimize $\sum_{i,j} (varcost_{i,j} \cdot x_{i,j}) + \sum_{i,j} (fixcost_{i,j} \cdot y_{i,j})$, which represents the total transportation cost, including both variable and fixed costs.
