Logical Design Problem
The Logical Design Problem is about designing a logical circuit with a minimal number of inverters (NOT gates) to satisfy certain constraints.

The main decision variables are:
- `in1`: a 2D array representing the input to each gate
- `out`: a 2D array representing the output of each gate
- `nor`: a 1D array representing whether each gate is an inverter (NOT gate) or not

The key constraints are:
- Each gate can have at most one input from each other gate
- Each gate can have at most one output to each other gate
- The output of a gate cannot be the same as its input
- The output of a gate is determined by its input and whether it is an inverter
- The input to each gate is determined by the output of other gates and the `alpha` array, which represents the connections between gates
- The number of inverters is minimized

The problem's objective is to minimize the number of inverters (`nor`) in the circuit while satisfying all the constraints. The `alpha` array represents the connections between gates, and the `number` variable represents the total number of inverters in the circuit. The goal is to find a solution that minimizes `number` while satisfying all the constraints.
