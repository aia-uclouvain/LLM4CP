name: Resource-Constrained Project Scheduling Problem (RCPSp)
description: The problem is to schedule a set of tasks with given durations and precedence constraints to minimize the total project duration, while considering the limited resources available. The tasks represent different activities in a construction project, such as erecting walls, carpentry, and painting. Each task has a specific duration and must be completed before certain other tasks can start. The goal is to find the optimal schedule that completes the project as quickly as possible.
variables: $start_1, start_2, ..., start_{10}, end_1, end_2, ..., end_{10}, makespan, limitx$ - These variables represent the start and end times of each task, the total project duration (makespan), and the limit on the number of resources available (limitx). The start and end times are decision variables that determine the schedule of the tasks, while the makespan and limitx are auxiliary variables used to formulate the objective and constraints.
constraints: $\forall t \in \{1, 2, ..., 10\}: end_t = start_t + duration_t$, $\forall p \in \{1, 2, ..., 13\}: prec(precedences_{p,1}, precedences_{p,2}, start, duration)$, $cumulative(start, duration, height, limitx)$, $maximum(makespan, end)$ - These constraints ensure that the end time of each task is calculated correctly, the precedence constraints are satisfied, the cumulative resource usage does not exceed the limit, and the makespan is the maximum end time of all tasks.
objective: Minimize $makespan$ - The objective is to minimize the total project duration, which is represented by the makespan variable.
