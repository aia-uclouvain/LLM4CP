Strimko problem
The Strimko problem is about filling an n x n grid with numbers from 1 to n, such that each row, column, and a set of predefined "streams" (connected paths in the grid) contain each number exactly once.

The main decision variables in this problem are the values of the grid cells, represented by the 2D array `x`, where `x[i, j]` is the value in the cell at row `i` and column `j`. The `streams` array represents the predefined streams, where `streams[i, 2*j+1]` and `streams[i, 2*j+2]` are the row and column indices of the `j`-th cell in the `i`-th stream.

The key constraints in this problem are: 
- Each row and column must contain each number from 1 to n exactly once, which is a classic Latin square constraint. 
- Each stream must also contain each number from 1 to n exactly once. 
- Some cells in the grid may have predefined values, represented by the `placed` array, which must be respected in the solution.

The objective of the Strimko problem is to find a solution that satisfies all the constraints, which is a satisfaction problem (i.e., a feasibility problem) rather than an optimization problem. In other words, the goal is to find a valid assignment of values to the grid cells, rather than to minimize or maximize a specific objective function.
