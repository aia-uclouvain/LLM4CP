Strimko problem
The problem is about filling in a square grid with numbers so that each row, column, and special path (called a "stream") contains each number only once.
The main variables are: 
- n, which is the size of the square grid
- x, which is the grid itself where each cell can hold a number from 1 to n
- streams, which defines the special paths in the grid
- num_placed and placed, which are used to specify some numbers that are already filled in the grid.
The basic restrictions are: 
- each row must contain each number from 1 to n exactly once
- each column must contain each number from 1 to n exactly once
- each special path (or "stream") must also contain each number from 1 to n exactly once
- some numbers may already be filled in the grid and must not be changed.
The goal of the problem is to fill in the entire grid so that all these rules are satisfied.
