Lexicographically Ordered Matrix Problem
The problem is about finding a matrix of integers that satisfies the lexicographically ordered constraint, where both adjacent rows and adjacent columns are lexicographically ordered.
The main decision variables are the elements of the matrix `x`, which represent the integers in the matrix.
The key constraints are that adjacent rows and adjacent columns must be lexicographically ordered, meaning that for any two adjacent rows or columns, the first row or column is lexicographically less than or equal to the second row or column. In plain language, this means that when comparing two rows or columns, the first row or column is considered less than or equal to the second row or column if the first element of the first row or column is less than the first element of the second row or column, or if the elements are equal, the second elements are compared, and so on.
The problem's objective is to find a matrix that satisfies the lexicographically ordered constraint, rather than minimizing or maximizing a specific objective function. The model uses the `satisfy` keyword, which means that the goal is to find a solution that satisfies all the constraints, rather than optimizing a specific objective.
