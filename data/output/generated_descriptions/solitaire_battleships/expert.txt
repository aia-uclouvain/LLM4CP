name: Solitaire Battleships
description: Solitaire Battleships is a puzzle where a partially filled board and the number of ships in each row and column are given, and the goal is to fill the board with ships of different sizes and types, ensuring that the ships do not overlap and that the row and column tallies are respected.
variables: $s_{i,j}, t_{i,j}, cp_i, cn_i$ 
These variables represent the following: $s_{i,j}$ is a binary variable indicating whether the cell at row $i$ and column $j$ is occupied by a ship segment, $t_{i,j}$ is an integer variable representing the type (size) of the ship fragment at the cell, $cp_i$ and $cn_i$ are integer variables counting the number of positive and negative ship segments of type $i$, respectively.
constraints: $s_{i,j} \in \{0,1\}, t_{i,j} \in \{-s,0,s\}, cp_i \geq 0, cn_i \geq 0, \sum_{j} s_{i,j} = rowsum_i, \sum_{i} s_{i,j} = colsum_j, t_{i,j} \neq 0 \leftrightarrow s_{i,j} = 1, cp_i + cn_i = surfaces_i$ 
These constraints ensure that: the cell occupancy variables are binary, the ship type variables are within the allowed range, the counts of ship segments are non-negative, the row and column tallies are respected, the ship type variables are consistent with the cell occupancy variables, and the total number of ship segments of each type is correct.
objective: satisfy 
The objective of the problem is to find a feasible solution that satisfies all the constraints, as there is no explicit objective function to minimize or maximize.
