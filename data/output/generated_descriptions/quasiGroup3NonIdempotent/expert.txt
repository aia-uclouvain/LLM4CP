name: Quasigroup Existence Problem
description: The Quasigroup existence problem is a classic problem in constraint programming that involves finding an m-order quasigroup, which is an mxm multiplication table of integers 1..m, where each element occurs exactly once in each row and column, and certain multiplication axioms hold.
variables: $quasiGroup_{i,j}, qgDiagonal_i$ 
These are decision variables where $quasiGroup_{i,j}$ represents the value at the i-th row and j-th column in the quasigroup table, and $qgDiagonal_i$ represents the value at the i-th position on the diagonal of the quasigroup table.
constraints: 
$\forall i \in nDomain: qgDiagonal_i = quasiGroup_{i,i}$, 
$\forall row \in nDomain: all\_different([quasiGroup_{row,col} | col \in nDomain])$, 
$\forall col \in nDomain: all\_different([quasiGroup_{row,col} | row \in nDomain])$, 
$\forall i,j \in nDomain: quasiGroup_{quasiGroup_{i,j},quasiGroup_{j,i}} = i$, 
$all\_different(qgDiagonal)$, 
$\forall i,j \in nDomain: (i \neq j) \rightarrow (quasiGroup_{i,j} \neq quasiGroup_{j,i})$, 
$\forall i,j \in nDomain: (quasiGroup_{i,i} = j) \rightarrow (quasiGroup_{j,j} = i)$, 
$\forall i \in nDomain: quasiGroup_{i,n-1} + 2 \geq i$
These constraints ensure that the quasigroup table is properly constructed, with each row and column containing each element exactly once, and the multiplication axioms holding.
objective: The objective of the problem is to find a satisfying assignment for the decision variables, i.e., to find an m-order quasigroup that satisfies all the constraints, rather than minimizing or maximizing a specific value. The problem is a satisfaction problem.
