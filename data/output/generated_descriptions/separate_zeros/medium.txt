**Array Separation Problem**
The Array Separation Problem is about rearranging the elements of a given array such that all zeros are placed either at the beginning or at the end of the array, while maintaining the relative order of non-zero elements.

**Decision Variables**
The main decision variables in this problem are:
- `s`: the original array
- `s2`: the array with zeros placed at the beginning
- `s3`: the array with zeros placed at the end
These variables represent the arrays before and after the separation of zeros.

**Key Constraints**
The key constraints in this problem can be described as follows:
- The original array `s` is given and must be copied to `s2` and `s3` with the zeros separated.
- In `s2`, all non-zero elements from `s` are placed after the zeros, in the same order as they appeared in `s`.
- In `s3`, all non-zero elements from `s` are placed before the zeros, in the same order as they appeared in `s`.
- The relative order of non-zero elements is maintained in both `s2` and `s3`.

**Objective**
The objective of the Array Separation Problem is not to minimize or maximize a specific value, but rather to satisfy the constraints of separating zeros in the array while maintaining the relative order of non-zero elements. The problem is a satisfaction problem, where the goal is to find a solution that meets all the given constraints.
