name: Array Separation Problem
description: The problem is to separate zeros from non-zero elements in a given array, while maintaining the relative order of non-zero elements. The goal is to create two new arrays, one with zeros placed first and another with zeros placed last.
variables: $s, s2, s3$ - These are decision variables representing arrays of integers, where $s$ is the original array, $s2$ is the array with zeros placed first, and $s3$ is the array with zeros placed last.
constraints: $s = [1,0,5,0,3]$, $separate\_z\_first(s,s2,0)$, $separate\_z\_last(s,s3,0)$ - These constraints define the original array $s$, and ensure that $s2$ and $s3$ are created by separating zeros from non-zero elements in $s$, while maintaining their relative order.
objective: The problem does not have a specific objective function to minimize or maximize, as it is a constraint satisfaction problem aiming to find a solution that satisfies the given constraints.
