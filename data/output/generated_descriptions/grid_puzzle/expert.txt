name: ABC Grid Puzzle
description: The ABC Grid Puzzle is a constraint satisfaction problem where a given grid of size n x n must be filled with letters A, B, and C. The grid initially contains some letters and empty spaces. The goal is to fill the empty spaces with letters A, B, and C such that each row and column contains the same number of each letter if they are different, and at least one row or column contains the same letter.
variables: $x_{i,j}$, where $i \in [1,n]$ and $j \in [1,n]$. These variables represent the letter to be placed in the $i^{th}$ row and $j^{th}$ column of the grid, with $x_{i,j} = 1$ corresponding to letter A, $x_{i,j} = 2$ corresponding to letter B, and $x_{i,j} = 3$ corresponding to letter C.
constraints: 
$\forall i \in [1,n], \forall j \in [1,n]: grid_{i,j} > 0 \rightarrow x_{i,j} = grid_{i,j}$, 
$\forall i \in [1,n]: global\_cardinality([x_{i,j} | j \in [1,n]], s, [gcc\_rows_{i,k}| k \in [1,m]]) \wedge same\_except\_0([gcc\_rows_{i,k} | k \in [1,m]])$, 
$\forall j \in [1,n]: global\_cardinality([x_{i,j} | i \in [1,n]], s, [gcc\_cols_{j,k}| k \in [1,m]]) \wedge same\_except\_0([gcc\_cols_{j,k} | k \in [1,m]])$, 
$\sum_{i \in [1,n]} (bool2int(same([x_{i,j} | j \in [1,n]])) + bool2int(same([x_{j,i} | j \in [1,n]]))) = 1$. 
These constraints ensure that the given letters in the grid are not changed, each row and column contains the same number of each letter if they are different, and at least one row or column contains the same letter.
