name: Olympic Puzzle
description: The Olympic Puzzle is a classic problem in constraint programming where we have a set of variables arranged in a specific configuration, and we need to assign each variable a different integer from 1 to 10 such that for any three variables in a specific triangular formation, the absolute difference between the two top variables is equal to the bottom variable.
variables: $X_1, X_2, X_3, X_4, X_5, X_6, X_7, X_8, X_9, X_{10}$, which are decision variables representing the values to be assigned to each position in the puzzle, with the goal of finding a valid assignment that satisfies the given constraints.
constraints: $X_1 = 3, \forall i \in \{1, 2, ..., 10\} X_i \in \{1, 2, ..., 10\}, X_1 \neq X_2 \neq ... \neq X_{10}, |X_2 - X_3| = X_1, |X_4 - X_5| = X_2, |X_5 - X_6| = X_3, |X_7 - X_8| = X_4, |X_8 - X_9| = X_5, |X_9 - X_{10}| = X_6$, which ensure that each variable is assigned a unique value, and the absolute difference between specific pairs of variables is equal to another variable, as per the problem's requirements.
objective: The objective of the problem is to find a satisfying assignment, i.e., to find values for the variables that satisfy all the constraints, with no specific objective function to minimize or maximize.
