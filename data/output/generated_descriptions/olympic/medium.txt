Olympic Puzzle
The Olympic Puzzle is a classic constraint programming problem that involves assigning different integers to a set of variables, arranged in a specific configuration, such that certain constraints are satisfied.

The key constraints in this problem are that for any three variables arranged in a triangular formation, the absolute difference between the values of the two upper variables must be equal to the value of the lower variable.
