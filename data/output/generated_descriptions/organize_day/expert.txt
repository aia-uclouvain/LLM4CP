name: Scheduling Problem
description: The problem is to schedule a set of tasks with given durations, such that certain tasks must be completed before others, and no two tasks can overlap. The tasks are Work, Mail, Shop, and Bank, and they need to be scheduled between 9am and 17pm. The goal is to find a schedule that satisfies all the constraints.
variables: $Begins = [Begins_1, Begins_2, Begins_3, Begins_4], Ends = [Ends_1, Ends_2, Ends_3, Ends_4]$ where $Begins_i$ and $Ends_i$ represent the start and end times of task $i$, respectively. These variables determine the schedule of the tasks.
constraints: $\forall i \in \{1, 2, 3, 4\}: Ends_i = Begins_i + Durations_i$, $\forall i, j \in \{1, 2, 3, 4\} \text{ where } i < j: no\_overlap(Begins_i, Durations_i, Begins_j, Durations_j)$, $\forall i \in \{1, 2\}: Ends_{BeforeTasks_{i,1}} \leq Begins_{BeforeTasks_{i,2}}$, $Begins_1 \geq 11$ where $no\_overlap(s_1, d_1, s_2, d_2) = (s_1 + d_1 \leq s_2) \vee (s_2 + d_2 \leq s_1)$. These constraints ensure that each task's end time is calculated correctly, no two tasks overlap, tasks that must be completed before others are scheduled accordingly, and the first task starts after 11am.
objective: The objective of the problem is to satisfy all the constraints, i.e., find a feasible schedule. There are also alternative objectives provided, such as minimizing the start time of certain tasks or the sum of the start times of all tasks, but the primary objective is to find a schedule that satisfies all the constraints.
