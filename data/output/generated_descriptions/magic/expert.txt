name: Magic Square Problem
description: The Magic Square Problem is a classic problem in recreational mathematics where the goal is to arrange distinct integers in a square grid such that the sum of the numbers in each row, column, and diagonal is the same. A normal magic square contains the integers from 1 to n^2, where n is the order of the square.
variables: $x_{i,j,k}, s, square_{i,j}$, where $x_{i,j,k}$ is a binary variable that equals 1 if cell $(i,j)$ contains integer $k$, $s$ is the magic sum, and $square_{i,j}$ is the value of cell $(i,j)$ in the magic square. These variables represent the assignment of integers to cells in the square and the magic sum that the rows, columns, and diagonals should add up to.
constraints: $\sum_{k=1}^{n^2} x_{i,j,k} = 1, \sum_{i=1}^{n} \sum_{j=1}^{n} x_{i,j,k} = 1, \sum_{j=1}^{n} \sum_{k=1}^{n^2} k \cdot x_{i,j,k} = s, \sum_{i=1}^{n} \sum_{k=1}^{n^2} k \cdot x_{i,j,k} = s, \sum_{i=1}^{n} \sum_{k=1}^{n^2} k \cdot x_{i,i,k} = s, \sum_{i=1}^{n} \sum_{k=1}^{n^2} k \cdot x_{i,n-i+1,k} = s, s \geq 0, s \leq n^3$, which ensure that each cell contains exactly one integer, each integer is assigned to exactly one cell, the sum of each row, column, and diagonal equals the magic sum, and the magic sum is within a valid range. 
objective: The objective of the problem is to find a feasible solution that satisfies all the constraints, which means finding a magic square of order $n$ with a magic sum $s$. The problem does not have a specific objective function to minimize or maximize, but rather seeks to find a valid assignment of integers to cells in the square that satisfies the magic square constraints.
