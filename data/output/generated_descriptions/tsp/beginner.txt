Traveling Salesman Problem
The problem is about finding the shortest route for a salesman to visit a set of cities and return to the starting point, without visiting any city more than once. 
The main variables are: 
- n, the number of cities 
- E, the set of roads between cities 
- c, the distance between each pair of cities 
- x, which represents whether the salesman travels from one city to another 
- y, the number of cars the salesman has after leaving a city and before entering another. 
The basic restrictions are: 
- The salesman must visit each city exactly once 
- The salesman must return to the starting point 
- The salesman can only travel between cities that are connected by a road. 
The goal is to minimize the total distance traveled by the salesman.
