name: Common Partition Problem
description: The Common Partition Problem is a constraint satisfaction problem that involves finding a common partition between two sets of variables. The problem is defined by two collections of variables, two sets of partitions, and two integers representing the number of variables in each collection that take a value in a partition derived from the other collection. The goal is to find an assignment of values to the variables such that the number of variables in each collection that take a value in a partition derived from the other collection is equal to the given integers.
variables: $a, b, x_1, x_2, x_3, x_4, y_1, y_2, y_3, y_4, y_5, y_6, partitions_1, partitions_2, partitions_3$, where $a$ and $b$ are integers representing the number of variables in each collection that take a value in a partition derived from the other collection, $x_i$ and $y_j$ are variables representing the values assigned to the variables in the two collections, and $partitions_i$ are sets of integers representing the partitions.
constraints: $a = \sum_{i=1}^3 x\_count_i \* bool2int(y\_count_i > 0)$, $b = \sum_{j=1}^3 y\_count_j \* bool2int(x\_count_j > 0)$, $x\_count_i = \sum_{j=1}^4 bool2int(x_j in partitions_i)$, $y\_count_i = \sum_{j=1}^6 bool2int(y_j in partitions_i)$, $x = [2, 3, 6, 5]$, $y = [5, 6, 3, 3, 7, 1]$, $partitions = [{1, 3}, {4}, {2, 6}]$, where the first two constraints define the common partition relationship between the two collections, the next two constraints count the occurrences of each variable in the partitions, and the last three constraints define the given values for the variables and partitions.
objective: The objective of the problem is to satisfy the given constraints, there is no explicit objective function to minimize or maximize.
