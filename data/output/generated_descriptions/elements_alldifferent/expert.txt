name: Elements Alldifferent Problem
description: The Elements Alldifferent Problem is a constraint satisfaction problem where we have two collections of items, and the goal is to ensure that each item in one collection is equal to one of the entries in the other collection, and all the variables in the first collection take distinct values.
variables: $element\_table_{i,j}, items_{i,j}$ for $i = 1..n, j = 1..2$, where $element\_table_{i,j}$ and $items_{i,j}$ are decision variables representing the values in the element table and items collection, respectively.
constraints: $\forall i \in \{1..m\} \exists j \in \{1..n\} (items_{i,1} = element\_table_{j,1} \wedge items_{i,2} = element\_table_{j,2}) \wedge all\_different(items_{i,1})$ for $i = 1..m$, where the first constraint ensures that each item in the items collection is equal to one of the entries in the element table, and the second constraint ensures that all the variables in the items collection take distinct values.
objective: The objective of the problem is to find a satisfying assignment of values to the decision variables, i.e., to satisfy all the constraints, as the problem is a constraint satisfaction problem and does not involve optimization.
