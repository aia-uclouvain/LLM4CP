name: Enigma Puzzle 1396: Planets
description: The problem is to assign distinct positive integers to the letters of the alphabet such that the sum of the values of the letters in the names of some members of the solar system matches the given scores, and then find the score for the word "PLANETS".
variables: $p, l, u, t, o, r, a, n, s, e, i, m, j, h, v, c, y, planets$ - These are decision variables representing the values assigned to each letter of the alphabet and the score for the word "PLANETS".
constraints: $all\_different(letters) \wedge p + l + u + t + o = 40 \wedge u + r + a + n + u + s = 36 \wedge n + e + p + t + u + n + e = 29 \wedge s + a + t + u + r + n = 33 \wedge j + u + p + i + t + e + r = 50 \wedge m + a + r + s = 32 \wedge e + a + r + t + h = 31 \wedge m + o + o + n = 36 \wedge v + e + n + u + s = 39 \wedge m + e + r + c + u + r + y = 33 \wedge s + u + n = 18 \wedge p + l + a + n + e + t + s = planets$ - These constraints ensure that each letter is assigned a unique value, the sum of the values of the letters in each planet's name matches the given score, and the score for the word "PLANETS" is calculated correctly.
objective: The objective of the problem is to find the value of $planets$, which is the score for the word "PLANETS" calculated by summing the values of its letters.
