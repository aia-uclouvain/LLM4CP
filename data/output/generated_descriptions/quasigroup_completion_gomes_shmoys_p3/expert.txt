name: Quasigroup Completion Problem
description: The Quasigroup Completion Problem is a classic problem in constraint programming that involves completing a partially filled table, known as a Latin square, such that no row or column contains the same two numbers. The problem is to determine whether the remaining blanks in the table can be filled in to obtain a complete Latin square.
variables: $x_{i,j}$, where $i$ and $j$ are indices representing the rows and columns of the table, respectively. These variables represent the values to be assigned to each cell in the table, and are decision variables that need to be determined.
constraints: $cp2d(x, array2d(1..n, 1..n, ...)) \wedge k\_all\_different(x)$, where $cp2d$ is a constraint that ensures the table is a valid Latin square, and $k\_all\_different(x)$ is a constraint that ensures all values in each row and column are distinct. These constraints ensure that the completed table is a valid Latin square.
objective: The objective of the problem is to find a feasible solution, i.e., to determine whether it is possible to complete the partially filled table to obtain a valid Latin square. There is no explicit objective function to minimize or maximize.
