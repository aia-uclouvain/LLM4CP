name: Stretch Circuit Problem
description: The Stretch Circuit Problem is a constraint satisfaction problem that involves finding a sequence of values that satisfies certain conditions. The sequence is divided into stretches, where a stretch is a set of consecutive variables that take the same value. The problem requires that each stretch has a minimum and maximum length, and that the values of the variables in each stretch are the same. The sequence is considered circular, meaning that the last variable is connected to the first variable.
variables: $x_1, x_2, ..., x_n, val_1, val_2, ..., val_m, lmin_1, lmin_2, ..., lmin_m, lmax_1, lmax_2, ..., lmax_m$. These variables represent the sequence of values ($x_i$), the set of possible values for each stretch ($val_i$), and the minimum and maximum lengths of each stretch ($lmin_i$ and $lmax_i$).
constraints: $lmin_s \leq lmax_s \forall s \in 1..m$, $(1 + (j-i)) \mod n \geq lmin_s$, $(1 + (j-i)) \mod n \leq lmax_s$, $x_k = x_{k-1} \forall k \in i+1..j$ if $i \leq j$, $x_k = x_{k-1} \forall k \in i+1..n-1$ and $x_k = x_{k-1} \forall k \in 1..j$ if $i > j$, $x_{(i-1) \mod n} \neq x_i$, $x_{(j+1) \mod n} \neq x_j$. These constraints ensure that the minimum length is less than or equal to the maximum length for each stretch, that the length of each stretch is within the allowed range, and that the values of the variables in each stretch are the same and different from the values of the variables outside the stretch.
objective: The objective of the problem is to satisfy all the constraints, there is no explicit objective function to minimize or maximize. The problem is a constraint satisfaction problem, where the goal is to find a sequence of values that satisfies all the conditions.
