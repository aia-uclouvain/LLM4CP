name: Penguin Problem
description: The Penguin problem is a classic example of nonmonotonic reasoning, which involves drawing conclusions based on incomplete information and default assumptions. The problem states that birds typically fly, but there are exceptions, such as penguins, which are abnormal birds. Given that Tweety is a penguin and Toto is an eagle, the goal is to determine which of these two birds can fly.
variables: $flies_{Tweety}, flies_{Toto}$, where $flies_x$ is a binary decision variable representing whether bird $x$ can fly or not.
constraints: $\forall t \in Bird: does\_fly(t, flies_t)$, where $does\_fly(t, flies_t)$ is a predicate that ensures $flies_t = 1$ if and only if $t$ is a bird that is not abnormal. This constraint is based on the default assumption that birds fly unless they are abnormal.
objective: The objective of this problem is not to minimize or maximize a specific value, but rather to satisfy the given constraints and determine the values of the decision variables, specifically which bird can fly. The problem is a satisfaction problem, where the goal is to find an assignment of values to the decision variables that satisfies all the constraints.
