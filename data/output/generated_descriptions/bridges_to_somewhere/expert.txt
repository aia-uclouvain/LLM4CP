name: Bridges to Somewhere problem
description: The problem involves building bridges between two towns, Hometown and Worktown, to minimize the total commuting distance of five residents. The residents live in houses represented by letters "A" through "E" in Hometown and work in offices represented by matching letters in Worktown. The bridges can only be built in approved areas, and each resident will take the shortest path from home to work, traveling only in up, down, left, or right directions.
variables: $x, b\_ix, tot\_dist$ 
These variables represent the following: $x$ is an array of decision variables representing the bridges to be built, $b\_ix$ is an array of decision variables representing the nearest bridge for each resident, and $tot\_dist$ is the total commuting distance of all residents.
constraints: $tot\_dist = \sum_{p=1}^{num\_p} (min([dist1[p,x[b]] + 1 + dist2[p,x[b]] | b in 1..num\_bridges\_to\_build]))$, 
$min\_index(min\_b\_ix, [dist1[p,x[b]] + 1 + dist2[p,x[b]] | b in 1..num\_bridges\_to\_build])$, 
$alldifferent(x)$, 
$increasing(x)$
These constraints represent the following: the total commuting distance is calculated by summing the minimum distances for each resident, the nearest bridge for each resident is determined, the bridges to be built must be distinct, and the bridges to be built must be in increasing order.
objective: minimize $tot\_dist$
