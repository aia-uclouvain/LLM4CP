Bertrand Russell Puzzle
The problem is about finding the correspondence between two sets of meaningless symbols, {a, b, c, d} and {w, x, y, z}, based on given logical conditions.
The main decision variables are the mappings between the two sets of symbols, represented by the arrays abcd and wxyz, where each element is a variable that can take a value from 1 to 4, corresponding to the indices of the symbols in the other set.
The key constraints are:
- If a is not mapped to x, then c must not be mapped to y.
- If b is mapped to either y or z, then a must be mapped to x.
- If c is not mapped to w, then b must be mapped to z.
- If d is mapped to y, then b must not be mapped to x.
- If d is not mapped to x, then b must be mapped to x.
The objective of the problem is to find a satisfying assignment of the symbols, such that all the given constraints are met, which is achieved by using the solve satisfy statement in the Minizinc model.
