name: Optimal Partial Design (OPD)
description: The Optimal Partial Design problem is a combinatorial optimization problem that involves finding a binary matrix with specific properties. The goal is to construct a matrix with a given number of rows and columns, where each row sums to a specified value, and the dot product between any pair of distinct rows is minimized.

variables: $x_{i,j}, s_{i,j,k}, m_{i,j}, opd, \lambda$ 
These variables represent the elements of the binary matrix, where $x_{i,j}$, $m_{i,j}$ are the values at row $i$ and column $j$, $s_{i,j,k}$ are auxiliary variables used to compute the dot product between rows, $opd$ is a set of sets representing the rows of the matrix, and $\lambda$ is the maximum dot product between any pair of distinct rows.

constraints: $\sum_{j=1}^{b} x_{i,j} = r, \sum_{j=1}^{b} m_{i,j} = r, \sum_{j=1}^{b} x_{i,j} * x_{j,k} = s_{i,j,k}, \sum_{j=1}^{b} m_{i,j} * m_{j,k} \leq \lambda, |i \cap j| \leq \lambda$ 
These constraints ensure that each row sums to the specified value $r$, the dot product between any pair of distinct rows is computed correctly, and the dot product is less than or equal to the maximum allowed value $\lambda$.

objective: minimize $\lambda$ 
The objective of the problem is to minimize the maximum dot product $\lambda$ between any pair of distinct rows, which represents the optimal partial design.
