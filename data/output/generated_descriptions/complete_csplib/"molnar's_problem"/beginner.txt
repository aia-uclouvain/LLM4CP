Molnar problem
The problem is about finding a special kind of square matrix, which is like a table with the same number of rows and columns. 
The main variables are x and y, where x represents the value in each cell of the matrix, and y represents the square of the value in each cell.
The basic restrictions are that each value in the matrix must be between 2 and a certain number d, and the determinant of the matrix (a mathematical calculation) must be 1. Also, the square of each value in the matrix must also have a determinant of 1. Additionally, the values in the matrix must be arranged in a specific order.
The goal of the problem is to find a matrix that satisfies all these restrictions, with no specific objective to minimize or maximize, just to find a valid solution.
