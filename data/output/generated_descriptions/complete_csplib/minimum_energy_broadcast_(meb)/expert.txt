name: Minimum Spanning Tree Problem
description: The problem is to find a tree that connects all nodes in a graph with the minimum total edge cost, where each node can have a different depth in the tree, and the root of the tree is a specified initial node.
variables: $parents: V \rightarrow V, depths: V \rightarrow D, optVar: \mathbb{Z}$, where $V$ is the set of nodes, $D$ is the set of possible depths, and $\mathbb{Z}$ is the set of integers. These variables represent the parent of each node in the tree, the depth of each node in the tree, and the total cost of the tree, respectively.
constraints: $parents(initialNode) = initialNode, \forall (child, parent) \in parents: (child \neq initialNode) \rightarrow (parent \neq child \wedge linkCosts((parent, child)) \neq 0), \forall (child, parent) \in parents: child \neq initialNode \rightarrow depths(child) > depths(parent)$, which ensure that the initial node is its own parent, each node has a valid parent, and the depth of each node is greater than its parent's depth.
objective: minimize $optVar = \sum_{parent \in V} \max_{child \in V} (linkCosts((parent, child)) \cdot \mathbb{I}(parent = parentI))$, where $\mathbb{I}$ is an indicator function, which represents the total cost of the minimum spanning tree.
