name: Lam's Problem
description: Lam's Problem is a classic problem in constraint programming that involves finding a binary matrix that satisfies certain conditions. The problem is defined as finding a matrix where each row and column sum equals 11, and for any two distinct rows, there is exactly one column where both rows have a 0.
variables: $m_{r,c}$, where $r,c \in Index$, $m_{r,c}$ is a boolean decision variable representing the entry in the $r^{th}$ row and $c^{th}$ column of the matrix, indicating whether the entry is 1 (true) or 0 (false).
constraints: $\sum_{c \in Index} m_{r,c} = 11, \forall r \in Index$, $\sum_{r \in Index} m_{r,c} = 11, \forall c \in Index$, $\sum_{c \in Index} \neg (m_{r_1,c} \lor m_{r_2,c}) = 1, \forall r_1,r_2 \in Index, r_1 \neq r_2$, which represent the row sum constraints, column sum constraints, and the constraint that for any two distinct rows, there is exactly one column where both rows have a 0.
objective: The objective of the problem is to find a feasible solution, i.e., a binary matrix that satisfies all the given constraints, as there is no explicit objective function to minimize or maximize.
