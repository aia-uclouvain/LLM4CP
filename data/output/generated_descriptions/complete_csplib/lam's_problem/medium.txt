Lam's Problem
Lam's Problem is a classic constraint programming problem that involves finding a binary matrix that satisfies certain conditions. The problem is about constructing a matrix where each row and column has a specific number of true values, and each pair of rows has a specific relationship.

The main decision variables in this problem are the elements of the matrix `m`, which is a binary matrix indexed by `Index`. Each element `m[r, c]` represents a boolean value indicating whether the cell at row `r` and column `c` is true or false.

The key constraints in this problem are:
- Each row must have exactly 11 true values.
- Each column must have exactly 11 true values.
- For each pair of distinct rows, there must be exactly one column where both rows have a true value.

The objective of Lam's Problem is not to minimize or maximize a specific function, but rather to find a feasible solution that satisfies all the given constraints. In other words, the goal is to construct a binary matrix that meets the specified conditions.
