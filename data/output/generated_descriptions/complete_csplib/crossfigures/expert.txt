name: Crossfigure Problem
description: The Crossfigure problem is a numerical puzzle where a grid of numbers needs to be filled in, following certain rules and clues. The grid has some black boxes, which are fixed to the value of 0. The clues are given in the form of equations, where the solution to the equation corresponds to a number in the grid.
variables: M[i,j], A1, A4, ..., A30, D1, D2, ..., D28 - These are the decision variables, where M[i,j] represents the value in the grid at row i and column j, and A1, A4, ..., A30 and D1, D2, ..., D28 represent the values of the across and down clues, respectively. They are integers between 0 and 9999.
constraints: across(M, A1, 4, 1, 1), ..., down(M, D28, 2, 8, 8), A1 = 2 * A27, ..., D28 = D21 + 27 - These are the constraints, which include the relationships between the grid values and the clue values, as well as the equations given by the clues. The across and down constraints ensure that the numbers in the grid correspond to the correct clue values.
objective: The objective of the problem is to find a solution that satisfies all the constraints, i.e., to fill in the grid with numbers that correspond to the correct clue values, while following the given rules and equations. There is no explicit objective function to minimize or maximize.
