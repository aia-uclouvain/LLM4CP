Tail Assignment 
This problem is about assigning flights to airplanes in a way that follows certain rules. 
The main variables are: 
- n_flights: the total number of flights 
- n_planes: the total number of airplanes 
- maint_freq: how often an airplane needs to visit a maintenance depot 
- route: the sequence of flights assigned to each airplane 
The basic restrictions or rules are: 
- Each airplane can only make flights that it is capable of making 
- Each airplane must visit a maintenance depot after a certain number of flights 
- The flights assigned to each airplane must be in a valid order, meaning that each flight can be made after the previous one 
- Each flight can only be made once 
- The route for each airplane must start with a flight that can be made after the last flight the airplane made previously 
The goal of the problem is to find a valid assignment of flights to airplanes that follows all the rules.
