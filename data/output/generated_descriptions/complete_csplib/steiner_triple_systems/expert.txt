name: Steiner Triplets
description: The Steiner triplets problem is a classic problem in combinatorial design theory, which involves finding a set of triples, where each triple contains three distinct elements from a given set, such that any two triples have at most one element in common.
variables: $x_{ij}$, where $i$ represents the index of the triple and $j$ represents the index of the element in the triple. These variables are typically represented as a matrix or an array, where $x_{ij}$ is true if the $j$th element is in the $i$th triple, and false otherwise. In some models, the variables are represented as sets, where each set represents a triple.
constraints: $\sum_{j=1}^{n} x_{ij} = 3 \forall i$, which ensures that each triple contains exactly three elements; $\sum_{i=1}^{nb} x_{ij} = (n-1)/2 \forall j$, which ensures that each element appears in exactly $(n-1)/2$ triples, although this constraint is not always explicitly stated; $\sum_{j=1}^{n} (x_{ij} \wedge x_{i'j}) \leq 1 \forall i, i'$, which ensures that any two triples have at most one element in common.
objective: The objective of the problem is to find a set of triples that satisfy all the constraints, which is essentially a feasibility problem, as there is no explicit objective function to minimize or maximize.
