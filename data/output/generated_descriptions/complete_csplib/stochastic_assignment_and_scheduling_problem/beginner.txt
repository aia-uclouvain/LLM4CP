Stochastic General Assignment Problem
The problem is about assigning tasks to machines and scheduling them in a way that minimizes the expected time it takes to complete all tasks, considering that the actual time it takes to complete each task is not known in advance and can vary.
The main variables are:
- Tasks: the individual tasks that need to be completed
- Machines: the machines that can be used to complete the tasks
- Start time: the time at which each task starts
- Duration: the time it takes to complete each task
- Optional tasks: some tasks can be completed in different ways, and the model decides which way to use
The basic restrictions or rules of the problem are:
- Each task must be assigned to a machine
- Each task must be completed after its preceding tasks
- The total time available for each machine is limited
- The actual time it takes to complete each task is not known in advance and can vary
The goal of the problem is to minimize the expected time it takes to complete all tasks, considering the uncertainty in the task durations.
