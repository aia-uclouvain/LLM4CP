name: Diamond-free Degree Sequences
description: The problem is to find all unique degree sequences for a given number of nodes in a simple undirected graph, such that each node has a degree greater than zero, the degree of each node is modulo 3, the sum of the degrees is modulo 12, and there exists a simple diamond-free graph with that degree sequence.
variables: $x_{ij}$, $degrees_i$ where $i, j \in \{1, 2, ..., n\}$, $x_{ij}$ represents the adjacency matrix of the graph, and $degrees_i$ represents the degree of the $i^{th}$ node. 
constraints: $\sum_{j=1}^{n} x_{ij} + \sum_{j=1}^{n} x_{jk} + \sum_{j=1}^{n} x_{il} + \sum_{j=1}^{n} x_{jl} + \sum_{j=1}^{n} x_{kl} + \sum_{j=1}^{n} x_{jl} \leq 4$ for all $i, j, k, l \in \{1, 2, ..., n\}$ where $i < j < k < l$, $x_{ij} = x_{ji}$ for all $i, j \in \{1, 2, ..., n\}$, $degrees_i = \sum_{j=1}^{n} x_{ij}$ for all $i \in \{1, 2, ..., n\}$, $degrees_i > 0$ for all $i \in \{1, 2, ..., n\}$, $degrees_i \mod 3 = 0$ for all $i \in \{1, 2, ..., n\}$, $\sum_{i=1}^{n} degrees_i \mod 12 = 0$, $x_{ii} = 0$ for all $i \in \{1, 2, ..., n\}$.
objective: The problem does not have a specific objective function to minimize or maximize, but rather to find all feasible solutions that satisfy the given constraints.
