Progressive Party Problem
The problem is about organizing a party where several guest boats need to visit a set of host boats over multiple periods. Each host boat has a limited capacity, and each guest boat has a certain number of crew members. The goal is to find a schedule that allows all guest boats to visit host boats without exceeding their capacities and without any two guest boats meeting more than once.

The main variables are:
- Hosts: the set of boats that will host the guest boats
- Sched: the schedule of which host boat each guest boat will visit at each period

The basic restrictions or rules of the problem are:
- Each host boat can only host a certain number of guest boats at a time, based on its capacity
- Each guest boat can only visit a host boat once
- No two guest boats can meet more than once at the same host boat
- The schedule must be repeated over multiple periods

The goal of the problem is to minimize the number of host boats needed to accommodate all the guest boats, while satisfying the above restrictions.
