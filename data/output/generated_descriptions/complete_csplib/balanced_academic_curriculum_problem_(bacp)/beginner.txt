Balanced Academic Curriculum Problem
The problem is about creating a schedule for a set of university courses that have prerequisites, meaning some courses must be taken before others. Each course has a certain number of credits, and students can only take a certain number of credits and courses per period. The goal is to assign each course to a period in a way that satisfies these rules and minimizes the maximum number of credits taken in any period.

The main variables are:
- Course period: the period to which each course is assigned
- Number of courses per period: the number of courses assigned to each period
- Number of credits per period: the total number of credits of all courses assigned to each period

The basic restrictions or rules are:
- Each course must be assigned to a period
- A course cannot be scheduled before its prerequisites
- The number of credits per period must be within the allowed range
- The number of courses per period must be within the allowed range

The goal of the problem is to minimize the maximum number of credits taken in any period, which means finding a schedule where the workload is as balanced as possible across all periods.
