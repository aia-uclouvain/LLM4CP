Tank Allocation Problem
The Tank Allocation Problem is about allocating different types of cargo to a set of tanks, where each tank has a limited capacity and certain constraints apply. The goal is to maximize the number of empty tanks while ensuring that all cargo is allocated and the constraints are satisfied.

The main decision variables in this problem are the allocation of cargo to tanks, represented by the matrix "allocation" where each entry allocation[tank] represents the type of cargo (or no cargo, represented by 0) allocated to tank t.

The key constraints in this problem are: 
- each tank can only be allocated one type of cargo or be empty, 
- the total volume of each cargo type allocated to the tanks must be greater than or equal to the volume of that cargo that needs to be shipped, 
- no tank can be allocated a cargo that is in its set of impossible cargos, 
- no tank can be neighbors with a tank containing an incompatible cargo.

The objective of the problem is to maximize the number of empty tanks, which is achieved by maximizing the sum of tanks where the allocation is 0 (representing no cargo). This means that the solver will try to allocate cargo to the minimum number of tanks necessary to satisfy the constraints, leaving as many tanks empty as possible.
