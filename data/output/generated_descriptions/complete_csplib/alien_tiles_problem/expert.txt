name: Alien Tiles
description: The Alien Tiles problem is a constraint programming problem where the goal is to find a matrix of clicks that maximizes the total number of clicks, with the constraint that there is no faster way to reach the same state. The problem is defined on a grid of size gridSize x gridSize, where each cell can have one of colours possible colors. The clicks are applied to the grid, and the color of each cell is determined by the sum of clicks affecting it modulo the number of colors.
variables: $clickMatrix_{i,j}$, where $i, j \in \{1, \ldots, gridSize\}$, $clickMatrix_{i,j} \in \{0, \ldots, colours-1\}$. These variables represent the number of clicks applied to each cell in the grid.
constraints: $\forall simplerClickMatrix \in \{0, \ldots, colours-1\}^{gridSize \times gridSize}$, $(\sum_{i,j} clickMatrix_{i,j}) \leq (\sum_{i,j} simplerClickMatrix_{i,j}) \vee \exists x, y \in \{1, \ldots, gridSize\}$, $((\sum_{index} clickMatrix_{index,y} + clickMatrix_{x,index}) - clickMatrix_{x,y}) \mod colours \neq ((\sum_{index} simplerClickMatrix_{index,y} + simplerClickMatrix_{x,index}) - simplerClickMatrix_{x,y}) \mod colours$. These constraints ensure that there is no faster way to reach the same state, by comparing the total number of clicks with all possible simpler click matrices, and checking if the resulting color configuration is different.
objective: Maximize $\sum_{i,j} clickMatrix_{i,j}$, which represents the total number of clicks applied to the grid.
