Traffic Lights Problem
The Traffic Lights Problem is a classic constraint programming problem that involves finding a set of consistent traffic light settings for a four-way junction.

The main decision variables in this problem are the settings for each traffic light, represented by the variables V1 to V4 for vehicles and P1 to P4 for pedestrians. These variables represent the color of each traffic light.
