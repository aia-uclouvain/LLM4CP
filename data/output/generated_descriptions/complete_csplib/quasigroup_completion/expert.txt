name: Latin Square Completion
description: The Latin Square Completion problem is a classic problem in constraint programming where we are given a partially filled N x N grid, and we need to fill in the remaining empty cells such that each row and each column contains each number from 1 to N exactly once.
variables: $puzzle_{i,j}$, where $i$ and $j$ are the row and column indices, respectively. These variables represent the value assigned to each cell in the grid, where $puzzle_{i,j}$ is the number assigned to the cell at row $i$ and column $j$.
constraints: 
- $puzzle_{i,j} = start_{i,j}$ if $start_{i,j} > 0$ (initial board respected)
- $\forall i \in \{1, \ldots, N\}, \forall k \in \{1, \ldots, N\}, \sum_{j=1}^{N} puzzle_{i,j,k} = 1$ (every row's number is distinct)
- $\forall j \in \{1, \ldots, N\}, \forall k \in \{1, \ldots, N\}, \sum_{i=1}^{N} puzzle_{i,j,k} = 1$ (every column's number is distinct)
- $\forall i \in \{1, \ldots, N\}, \forall j \in \{1, \ldots, N\}, \sum_{k=1}^{N} puzzle_{i,j,k} = 1$ (each cell must have a number assigned)
These constraints ensure that the completed grid is a valid Latin square, where each row and column contains each number exactly once, and the initial board is respected.
objective: satisfy (the objective is to find a feasible solution that satisfies all the constraints, there is no explicit objective function to minimize or maximize)
