name: Nurse Rostering Problem
description: The Nurse Rostering Problem is a classic problem in the field of constraint programming, which involves creating a schedule for nurses to work at a hospital. The goal is to assign nurses to shifts while satisfying various constraints such as the number of nurses required for each shift, the minimum and maximum number of consecutive shifts a nurse can work, and the requirement for rest days.
variables: $plan1d, plan2d, plan2dT, s\_min\_arrays, s\_max\_arrays$ - These are decision variables that represent the assignment of nurses to shifts. $plan1d$ is a one-dimensional array that represents the assignment of nurses to shifts over the entire planning period, $plan2d$ and $plan2dT$ are two-dimensional arrays that represent the assignment of nurses to shifts for each week and day, respectively. $s\_min\_arrays$ and $s\_max\_arrays$ are helper arrays used to enforce the minimum and maximum consecutive shift constraints.
constraints: 
$plan2d\[week, day\] = plan1d\[(week-1) \* daysPerWeek + day\]$ - This constraint ensures that the one-dimensional and two-dimensional arrays are consistent.
$plan2d\[week, daysPerWeek - 1\] = plan2d\[week, daysPerWeek\]$ - This constraint ensures that the last two days of each week have the same shift.
$plan1d\[day\] != plan1d\[day+1\] -> all\_equal(plan1d\[day+1..day+s\_min\])$ - This constraint ensures that if a nurse works a different shift on consecutive days, they must work the same shift for at least $s\_min$ days.
$all\_equal(plan1d\[day..day+s\_max\]) -> plan1d\[day\] != plan1d\[day + s\_max\]$ - This constraint ensures that a nurse cannot work the same shift for more than $s\_max$ consecutive days.
$count(j in plan1d\[day..day + daysPerWeek \* 2\])(j=0) >= 2$ - This constraint ensures that there are at least 2 rest days every 2 weeks.
$plan1d\[day\] <= plan1d\[day+1\] \/ plan1d\[day+1\] = 0$ - This constraint ensures that the shifts are assigned in a forward rotating order, i.e., a nurse cannot work a shift with a lower value than the previous day unless it is a rest day.
$gcc(plan2d\[, day\], [0,1,2,3], shiftRequirements\[day, ..\])$ - This constraint ensures that the number of nurses assigned to each shift meets the requirements.
objective: The objective of the problem is to find a feasible assignment of nurses to shifts that satisfies all the constraints. There is no explicit objective function to minimize or maximize.
