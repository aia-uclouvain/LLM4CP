Equidistant Frequency Permutation Arrays problem.
The problem is about finding a set of codewords that satisfy certain conditions, specifically that each codeword contains each symbol a specified number of times and that each pair of codewords differs in a specified number of positions.

The main decision variables are the codewords, represented by the variable `c`, which is a set of strings, where each string is a function that maps an index to a character. The characters are represented by the domain `Character`, which is an integer between 1 and `numChars`. The index is represented by the domain `Index`, which is an integer between 1 and `wordLength`, where `wordLength` is the product of the number of symbols and the number of times each symbol appears in a word.

The key constraints are that each codeword must contain each symbol exactly `lam` times, and that each pair of codewords must differ in exactly `dist` positions. In plain language, this means that if you compare any two codewords, they must have exactly `dist` positions where the symbols are different. Additionally, if you count the number of times each symbol appears in a codeword, it must be exactly `lam` times.

The problem's objective is to find a set of `numCodeWords` codewords that satisfy these constraints. The problem does not have a specific objective function to minimize or maximize, but rather it is a feasibility problem, where the goal is to find a set of codewords that satisfy all the constraints.
