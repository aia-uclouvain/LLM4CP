name: Perfect Square Placement
description: The problem involves placing a set of squares of different sizes into a larger square container in such a way that the squares do not overlap and the entire container is filled. The goal is to find a placement that satisfies these conditions.
variables: $x_i, y_i$ for $i = 1, 2, ..., n$, where $x_i$ and $y_i$ are the coordinates of the bottom-left corner of the $i^{th}$ square. These variables represent the position of each square in the container.
constraints: $\forall i: 0 \leq x_i \leq N - s_i$, $\forall i: 0 \leq y_i \leq N - s_i$, $\forall i, j: (x_i + s_i \leq x_j) \vee (x_j + s_j \leq x_i) \vee (y_i + s_i \leq y_j) \vee (y_j + s_j \leq y_i)$, where $N$ is the size of the container, $s_i$ is the size of the $i^{th}$ square, and $n$ is the number of squares. These constraints ensure that each square is within the container, and that no two squares overlap.
objective: The objective is to find a feasible placement of the squares, which means finding values for $x_i$ and $y_i$ that satisfy all the constraints. There is no explicit objective function to minimize or maximize.
