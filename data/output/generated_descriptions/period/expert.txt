name: Period Constraint Problem
description: The Period Constraint Problem is a constraint satisfaction problem that involves finding a sequence of values that satisfies a periodic pattern. The problem is defined by a sequence of variables and a period length, and the goal is to find an assignment of values to the variables such that the sequence repeats itself after the specified period length.
variables: $x_1, x_2, ..., x_n, the\_period_1, the\_period_2, ..., the\_period_{per\_len}$ - These are the decision variables of the problem, where $x_i$ represents the value of the sequence at position $i$, and $the\_period_i$ represents the value of the period at position $i$. The variables $x_i$ are used to represent the sequence, and the variables $the\_period_i$ are used to represent the period.
constraints: $period(per\_len, x)$, $the\_period_i = x_i$ for $i = 1, 2, ..., per\_len$ - These are the constraints of the problem, where $period(per\_len, x)$ is a global constraint that enforces the periodic pattern on the sequence $x$, and $the\_period_i = x_i$ are constraints that link the period variables to the corresponding sequence variables.
objective: satisfy - The objective of the problem is to find a satisfying assignment of values to the variables, i.e., to find a sequence that satisfies the periodic pattern. There is no explicit objective function to minimize or maximize.
