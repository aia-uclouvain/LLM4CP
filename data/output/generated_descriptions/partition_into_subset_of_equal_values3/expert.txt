name: Partition Problem
description: The problem is to divide a given set of numbers into a specified number of subsets such that the sum of the numbers in each subset is equal. The problem requires finding a partition of the set into subsets with equal sums, if such a partition exists.
variables: $x_i$ where $i \in \{1, 2, ..., n\}$, $x_i$ represents the subset to which the $i^{th}$ number belongs. These are decision variables that determine the assignment of each number to a subset.
constraints: 
- $ssum = \sum_{i=1}^{n} s_i \div num\_subsets$
- $\forall p \in \{1, 2, ..., num\_subsets\}, ssum = \sum_{i=1}^{n} s_i \cdot bool2int(x_i == p)$
- $x_1 = 1$ (symmetry breaking constraint)
- $\sum_{i=1}^{n} s_i \mod num\_subsets == 0$ (assertion that the sum of the set is divisible by the number of subsets)
These constraints ensure that each subset has the same sum, that the sum of each subset is equal to the total sum divided by the number of subsets, and that the first number is assigned to the first subset to break symmetry.
objective: The objective of this problem is to find a feasible solution that satisfies all the constraints, which means finding a partition of the set into subsets with equal sums. There is no explicit objective function to minimize or maximize.
