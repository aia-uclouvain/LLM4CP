Home Improvement Problem
The Home Improvement Problem is about determining the most profitable home improvement to make to a house before selling it. The goal is to identify which improvement will add the most value to the house, considering the installation costs of different improvements.

The main decision variables are:
- `x`: an array of variables representing the added value of each improvement to the house.
- `rest`: a 2D array of variables representing the residual value of each house after considering the improvements.
- `sum_rest` and `max_rest`: variables representing the sum and maximum of the residual values, respectively.

The key constraints are:
- Each house's selling price is equal to the sum of the added values of the improvements it has, minus the installation costs, plus the residual value.
- The added values and residual values are bounded within reasonable ranges.

The objective of the problem is to minimize the sum of the residual values (`sum_rest`), which represents the difference between the actual selling prices and the predicted prices based on the improvements. By minimizing this sum, the model aims to find the most accurate prediction of the added values of the improvements, and thus identify the most profitable improvement to make.
