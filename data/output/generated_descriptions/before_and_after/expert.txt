name: Before and After Puzzle
description: The problem involves four kids - Alice, Boris, Claire, and David - who have different numbers of jellybeans. The kids exchange jellybeans in a series of steps, with certain conditions applying to the number of jellybeans exchanged and the final number of jellybeans each kid has. The goal is to determine the initial and final number of jellybeans each kid has.
variables: $x_{t,i}, gifts_i$ where $t \in \{1,2,3\}$ and $i \in \{1,2,3,4\}$ and $gifts_i$ where $i \in \{1,2,3,4\}$. These variables represent the number of jellybeans each kid has at each step ($x_{t,i}$) and the number of jellybeans exchanged in each gift ($gifts_i$).
constraints: $all\_different([x_{1,i} | i \in \{1,2,3,4\}])$, $x_{2,David} = x_{1,David} - gifts_1$, $x_{2,Alice} = x_{1,Alice} + gifts_1$, $x_{2,Claire} = x_{1,Claire} - gifts_2$, $x_{2,Boris} = x_{1,Boris} + gifts_2$, $x_{3,Boris} = x_{2,Boris} - gifts_3$, $x_{3,Alice} = x_{2,Alice} + gifts_3$, $x_{3,David} = x_{2,David} - gifts_4$, $x_{3,Claire} = x_{2,Claire} + gifts_4$, $all\_different(gifts)$, $all\_different([x_{1,i} | i \in \{1,2,3,4\}] ++ [x_{3,i} | i \in \{1,2,3,4\}])$, $x_{3,Alice} = x_{3,David} + 1$, $x_{3,Claire} + 1 = x_{3,Boris}$, $gifts_1 > gifts_3$. These constraints ensure that each kid has a different number of jellybeans at each step, that the number of jellybeans exchanged is consistent with the problem description, and that the final number of jellybeans each kid has satisfies the given conditions.
objective: The objective of the problem is to find a solution that satisfies all the constraints, there is no explicit objective function to minimize or maximize.
