name: Quasigroup Existence Problem
description: The Quasigroup existence problem is a classic problem in constraint programming that involves finding an m-order quasigroup, which is an mxm multiplication table of integers 1..m, where each element occurs exactly once in each row and column, and certain multiplication axioms hold. In this case, the problem is looking for a quasigroup that satisfies axiom 4, idempotency, and other properties.

variables: $quasiGroup_{i,j}, qgDiagonal_i$ 
These are decision variables where $quasiGroup_{i,j}$ represents the value at row i and column j in the quasigroup table, and $qgDiagonal_i$ represents the value on the diagonal of the table at row and column i.

constraints: 
$\forall i \in nDomain: qgDiagonal_i = quasiGroup_{i,i}$, 
$\forall row \in nDomain: all\_different([quasiGroup_{row,col} | col \in nDomain])$, 
$\forall col \in nDomain: all\_different([quasiGroup_{row,col} | row \in nDomain])$, 
$\forall i,j \in nDomain: quasiGroup_{quasiGroup_{j,i},quasiGroup_{i,j}} = i$, 
$\forall i \in nDomain: quasiGroup_{i,i} = i$, 
$all\_different(qgDiagonal)$, 
$\forall i,j \in nDomain: (i \neq j) \rightarrow (quasiGroup_{i,j} \neq quasiGroup_{j,i})$, 
$\forall i,j \in nDomain: (quasiGroup_{i,i} = j) \rightarrow (quasiGroup_{j,j} = i)$, 
$\forall i \in nDomain: quasiGroup_{i,n-1} + 2 \geq i$
These constraints ensure that the quasigroup table is filled in such a way that each row and column contains each element exactly once, the diagonal elements are unique, and the quasigroup satisfies the required axioms and properties.

objective: satisfy 
The objective of this problem is to find a solution that satisfies all the given constraints, i.e., to find a quasigroup that meets all the required properties. There is no objective function to minimize or maximize.
