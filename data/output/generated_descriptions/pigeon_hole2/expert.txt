name: Pigeon Hole Problem
description: The Pigeon Hole Problem is a classic problem in constraint programming that involves placing a certain number of pigeons into a limited number of pigeon holes, with the constraint that each hole can contain at most one pigeon. The problem is to determine if it is possible to place all the pigeons into the available holes.
variables: $pigeon\_holes = [pigeon\_holes_1, pigeon\_holes_2, ..., pigeon\_holes_M]$, where $pigeon\_holes_i$ is a decision variable representing the number of pigeons in the $i^{th}$ hole, and $M$ is the total number of holes. These variables are used to indicate which hole each pigeon is assigned to.
constraints: $\forall i \in \{1, 2, ..., N\} (count(pigeon\_holes, i, 1))$, which ensures that each pigeon is assigned to exactly one hole, and the implicit constraint that $pigeon\_holes_i \in \{0, 1, ..., N\}$, which ensures that each hole can contain at most one pigeon.
objective: satisfy, which means the objective is to find a feasible solution that satisfies all the constraints, rather than minimizing or maximizing a specific value. In other words, the goal is to determine if it is possible to assign all $N$ pigeons to the $M$ holes such that each hole contains at most one pigeon.
