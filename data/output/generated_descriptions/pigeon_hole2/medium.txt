Pigeon Hole Problem
The Pigeon Hole problem is about distributing a certain number of pigeons into a given number of pigeon holes, with the condition that each hole can contain at most one pigeon.
The main decision variables in this problem are the `pigeon_holes` array, which represents the assignment of pigeons to holes. Each element in the array corresponds to a pigeon hole, and its value indicates the pigeon assigned to that hole (or 0 if the hole is empty).
The key constraints in this problem are that each pigeon must be assigned to exactly one hole, and no hole can contain more than one pigeon. This is achieved through the `pigeon` predicate, which ensures that each pigeon is counted exactly once in the `pigeon_holes` array.
The objective of the Pigeon Hole problem is not to minimize or maximize a specific value, but rather to find a feasible solution that satisfies the constraints. In other words, the goal is to determine whether it is possible to assign the pigeons to the holes in a way that meets the conditions, and if so, to find one such assignment. The problem has a solution if and only if the number of pigeons is less than or equal to the number of holes.
