Disjunctive Scheduling Problem
The problem is about scheduling tasks that cannot overlap with each other. Imagine you have several tasks that need to be done, and each task takes a certain amount of time. You want to make sure that these tasks do not happen at the same time.

The main variables are:
- origin: when each task starts
- duration: how long each task takes
- used: whether each task is actually done or not
- end_time: when each task finishes

The basic restrictions or rules are:
- tasks that take time cannot overlap with other tasks
- tasks that do not take time are not considered
- each task has a start time, a duration, and an end time

The goal of the problem is to find a schedule where all tasks that take time do not overlap, and to determine which tasks are actually done, when they start, how long they take, and when they finish.
