Fairies problem
The problem is about a group of fairies visiting children at night and giving them pearls based on the color of a star each child has. 
The main variables are: 
- fairies (there are 5 of them), 
- children (there are 3 of them), 
- colors (there are 9 colors), 
- nights (there are 5 nights), 
- which fairy is attracted to which color, 
- which child has which color star, 
- how many pearls each child gets on each night, 
- which fairy visits on which night.
The basic restrictions are: 
- each fairy can only be attracted to one color, 
- at least one fairy must be attracted to turquoise, 
- exactly one fairy must be attracted to earth, 
- the number of pearls a child gets on a night must match the visits from fairies who are attracted to the child's star color.
The goal is to find a solution that satisfies all these conditions, meaning we want to figure out which fairy is attracted to which color, and which fairy visits on which night, so that everything matches up correctly.
