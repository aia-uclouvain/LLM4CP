name: Fairies problem
description: The Fairies problem is a classic constraint programming problem where a set of fairies are attracted to different colors and visit children on different nights, giving them pearls. The problem is to determine which fairy is attracted to which color, given the number of pearls each child receives on each night and the nights when each fairy visits the children.
variables: $x_{i,k}, visit_{i,m}, star_{j,k}, pearl_{j,m}$, where $x_{i,k}$ is a decision variable representing whether fairy $i$ is attracted to color $k$, $visit_{i,m}$ is a parameter representing whether fairy $i$ visits on night $m$, $star_{j,k}$ is a parameter representing whether child $j$ has a star of color $k$, and $pearl_{j,m}$ is a parameter representing the number of pearls child $j$ receives on night $m$.
constraints: $\sum_{i \in fairy, k \in color} (visit_{i,m} \cdot star_{j,k} \cdot x_{i,k}) = pearl_{j,m} \forall j \in child, m \in night$, $\sum_{k \in color} x_{i,k} = 1 \forall i \in fairy$, $\sum_{i \in fairy} x_{i,5} \geq 1$, $\sum_{i \in fairy} x_{i,9} = 1$, where the first constraint ensures that the number of pearls received by each child on each night is consistent with the visits and stars, the second constraint ensures that each fairy is attracted to a single color, and the last two constraints ensure that at least one fairy is attracted to turquoise and exactly one fairy is attracted to earth.
objective: satisfy, which means that the objective is to find a feasible solution that satisfies all the constraints, rather than minimizing or maximizing a specific value.
