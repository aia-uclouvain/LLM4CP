name: Scheduling Speakers Problem
description: The Scheduling Speakers Problem is a classic constraint programming problem where we have a set of speakers and a set of time slots, and each speaker has a set of available time slots in which they can speak. The goal is to assign each speaker to a time slot such that no two speakers are assigned to the same time slot and each speaker is assigned to one of their available time slots.
variables: $x = [x_1, x_2, ..., x_n]$, where $x_i$ is the time slot assigned to speaker $i$. These variables represent the assignment of speakers to time slots.
constraints: $\forall i \in \{1, 2, ..., n\}, x_i \in available_i$ and $all\_different(x)$, where $available_i$ is the set of available time slots for speaker $i$ and $all\_different(x)$ ensures that no two speakers are assigned to the same time slot. 
objective: The objective of this problem is to find a feasible assignment of speakers to time slots that satisfies all the constraints, as it is a satisfaction problem.
