Cumulative Scheduling Problem
The Cumulative Scheduling Problem is about scheduling a set of tasks with given durations and resource requirements, such that the total resource consumption at any point in time does not exceed a given limit.

The main decision variables are: 
- `origin`: the start time of each task
- `duration`: the length of each task
- `height`: the resource consumption of each task
- `end`: the end time of each task, which is calculated as `origin + duration`
- `limitx`: the maximum allowed resource consumption at any point in time

The key constraints are: 
- Each task has a specific duration and resource consumption
- The end time of each task is the sum of its start time and duration
- The cumulative resource consumption of all tasks at any point in time must not exceed the given limit

The problem's objective is to minimize the maximum end time of all tasks, which means finding a schedule that allows all tasks to be completed as early as possible, while not exceeding the resource limit. Alternatively, the problem can also be formulated to minimize the maximum allowed resource consumption, `limitx`, which means finding the minimum resource capacity required to schedule all tasks.
