name: Set Partitioning and Set Covering Problem
description: The problem involves selecting a subset of alternatives to cover a set of objects, where each alternative covers a specific subset of objects. The goal is to find the minimum-cost subset of alternatives that covers all objects, either exactly once (set partitioning) or at least once (set covering).
variables: $x_i, a_i$ where $i = 1, \ldots, num\_alternatives$ and $a_i \subseteq \{1, \ldots, num\_objects\}$. $x_i$ is a binary decision variable indicating whether alternative $i$ is selected, and $a_i$ is a set of objects covered by alternative $i$.
constraints: $\sum_{i=1}^{num\_alternatives} x_i \cdot \mathbb{I}(j \in a_i) = 1$ for set partitioning, or $\sum_{i=1}^{num\_alternatives} x_i \cdot \mathbb{I}(j \in a_i) \geq 1$ for set covering, for all $j = 1, \ldots, num\_objects$. These constraints ensure that each object is covered exactly once (set partitioning) or at least once (set covering) by the selected alternatives.
objective: Minimize $z = \sum_{i=1}^{num\_alternatives} x_i \cdot costs_i$, where $costs_i$ is the cost of alternative $i$. The objective is to minimize the total cost of the selected alternatives that cover all objects.
