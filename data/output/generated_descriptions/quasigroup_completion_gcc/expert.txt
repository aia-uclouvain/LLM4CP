name: Quasigroup Completion Problem
description: The Quasigroup Completion Problem is a classic problem in constraint programming that involves completing a partially filled Latin square. A Latin square is an n x n matrix filled with n different symbols, each occurring exactly once in each row and exactly once in each column. The problem requires finding a completion of the given partial Latin square such that the resulting matrix is a valid Latin square.
variables: $x_{i,j}$, where $i$ and $j$ are indices ranging from 1 to $n$, representing the value in the $i$-th row and $j$-th column of the Latin square. These variables are decision variables that need to be assigned a value from the set {1, 2, ..., n} to complete the Latin square.
constraints: $\forall i \in \{1, 2, ..., n\}, \forall j \in \{1, 2, ..., n\}, x_{i,j} \in \{1, 2, ..., n\}$ and $\forall i \in \{1, 2, ..., n\}, \forall k \in \{1, 2, ..., n\}, \sum_{j=1}^{n} (x_{i,j} = k) = 1$ and $\forall j \in \{1, 2, ..., n\}, \forall k \in \{1, 2, ..., n\}, \sum_{i=1}^{n} (x_{i,j} = k) = 1$, which ensure that each row and each column contains each symbol exactly once, and the given partial assignments are respected.
objective: The objective of the problem is to find a feasible completion of the partial Latin square, i.e., to assign values to the decision variables $x_{i,j}$ such that all the constraints are satisfied. The problem does not have an explicit optimization objective, such as minimizing or maximizing a particular value.
