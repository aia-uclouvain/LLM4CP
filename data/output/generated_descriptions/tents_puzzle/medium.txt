Tents Puzzle
The Tents Puzzle is a classic constraint programming problem where the goal is to place tents in a grid such that each tree has a tent either horizontally or vertically adjacent to it, and each tent has a tree either horizontally or vertically adjacent to it. The puzzle also specifies the number of tents in each row and column.

The main decision variables in this problem are the values of the grid cells, which can be either a tree (T), a tent (E), or empty (_). The decision variables are represented by the array `x`, where `x[i, j]` represents the value of the cell at row `i` and column `j`.

The key constraints in this problem are:
- Each tree must have a tent either horizontally or vertically adjacent to it.
- Each tent must have a tree either horizontally or vertically adjacent to it.
- No two tents can be adjacent to each other, either horizontally, vertically, or diagonally.
- The number of tents in each row and column must match the given numbers.

The objective of the problem is to find a solution that satisfies all the constraints, which means finding a valid placement of tents in the grid. The problem does not have a specific objective function to minimize or maximize, but rather to find a feasible solution that meets all the constraints.
