Contiguity Constraint Problem
The problem is about finding the best way to select a group of consecutive items from a list of items, where each item has a certain value or weight. 
The main variables are: 
- y, which represents whether an item is selected (1) or not (0), 
- z, which is the total value of the selected items, 
- c, which is the number of selected items, 
- w, which is the weight or value of each item.
The basic restrictions or rules are: 
- the total value of the selected items (z) must be greater than or equal to a certain threshold (A), 
- the selected items must be consecutive, meaning that if an item is selected, all items before and after it must also be selected, 
- each item can either be selected (1) or not (0).
The goal of the problem is to minimize the total value of the selected items (z) while still meeting the threshold requirement and selecting consecutive items.
