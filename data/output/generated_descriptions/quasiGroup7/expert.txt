name: Quasigroup Existence Problem
description: The Quasigroup existence problem is a classic problem in constraint programming that involves finding an order m quasigroup, which is an mxm multiplication table of integers 1..m. The goal is to fill in the table such that each element occurs exactly once in each row and column, and certain multiplication axioms hold. In this specific case, the problem is focused on finding a quasigroup that satisfies axiom 7, which states that (b*a)*b = a*(b*a).
variables: $quasiGroup_{i,j}$, where $i,j \in \{0,1,...,n-1\}$. These variables represent the elements of the quasigroup table, where $quasiGroup_{i,j}$ is the value at row $i$ and column $j$.
constraints: 
$\forall i \in \{0,1,...,n-1\}: all\_different([quasiGroup_{i,j}]_{j=0}^{n-1})$,
$\forall j \in \{0,1,...,n-1\}: all\_different([quasiGroup_{i,j}]_{i=0}^{n-1})$,
$\forall i \in \{0,1,...,n-1\}: quasiGroup_{i,i} = i$,
$\forall i,j \in \{0,1,...,n-1\}: quasiGroup_{i,quasiGroup_{j,i}} = quasiGroup_{quasiGroup_{j,i},j}$,
$\forall i \in \{0,1,...,n-1\}: quasiGroup_{i,n-1} + 2 \geq i$. 
These constraints ensure that each row and column contains each element exactly once, the diagonal elements are equal to their indices, and the quasigroup satisfies axiom 7.
objective: The objective of the problem is to find a feasible solution that satisfies all the constraints, as there is no explicit objective function to minimize or maximize. The goal is to determine whether a quasigroup of order $n$ exists that satisfies the given constraints.
