name: Next Greater Element Problem
description: The Next Greater Element Problem is a constraint satisfaction problem that involves finding the next greater element in a sorted list of variables. The problem requires that a specific variable, var2, is the smallest possible value greater than another variable, var1, in the list, and that the list is sorted in strictly increasing order.
variables: $var1, var2, variables = [x_1, x_2, ..., x_n]$ where $var1$ and $var2$ are integer decision variables representing the values to be compared, and $variables$ is an array of integer decision variables representing the sorted list.
constraints: $var1 = 7, var2 = 8, variables = [3, 5, 8, 9], next\_greater\_element(var1, var2, variables)$ where the constraints specify the values of $var1$ and $var2$, the sorted list $variables$, and the $next\_greater\_element$ constraint ensures that $var2$ is the smallest possible value greater than $var1$ in the list.
objective: The objective of the problem is to satisfy all the given constraints, i.e., to find an assignment of values to the decision variables that satisfies all the constraints, rather than minimizing or maximizing a specific value.
