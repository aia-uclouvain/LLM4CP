Number of Days Problem 
This problem is about finding the best combination of hotel room reservations to book for a certain number of days, given different room options with varying prices and stay lengths. 
The main variables are: 
- x, which represents whether a specific room option is chosen (0 for no, 1 for yes)
- days, which is the total number of days stayed
- total_cost, which is the total cost of the chosen room options
The basic restrictions are: 
- the total number of days stayed must equal a specific target (13 days in this case)
- each room option can only be chosen once
The goal is to minimize the total cost while still meeting the target number of days.
