name: Symmetric Alldifferent Problem
description: The symmetric alldifferent problem is a constraint satisfaction problem where the goal is to assign values to a set of variables such that each variable is assigned a unique value, and for each variable, the value it is assigned is also assigned to it by another variable. This can be interpreted as a graph-covering problem where one has to cover a digraph with circuits of length two in such a way that each vertex of the graph belongs to one single circuit.
variables: $x = [x_1, x_2, ..., x_n]$, where $x_i$ is a decision variable representing the value assigned to the $i^{th}$ node, and $x_i$ can take any value from the set $\{1, 2, ..., n\}$, which represents the indices of the nodes.
constraints: $all\_different(x) \wedge \forall i \in \{1, 2, ..., n\} (x[x_i] = i)$, which means that all values assigned to the variables must be unique, and for each variable $x_i$, the value it is assigned ($x_i$) must also be assigned to it by another variable ($x[x_i] = i$).
objective: The objective of the problem is to find a satisfying assignment of values to the variables, i.e., to find a solution that satisfies all the constraints, and there is no specific value to minimize or maximize.
