Crypto Problem
The Crypto problem is a classic constraint programming problem where the goal is to assign numerical values to letters of the alphabet such that a set of given equations is satisfied. 

The main decision variables in this problem are the assignments of numerical values to each letter of the alphabet, represented by the array `x` where `x[i,j] = 1` means that letter `i` is assigned value `j`. 

The key constraints in this problem are: 
- Each letter can only be assigned one numerical value, and each numerical value can only be assigned to one letter. 
- The sum of the numerical values assigned to the letters in each word must equal the given total for that word. 

The objective of the problem is to find an assignment of numerical values to the letters that satisfies all the given equations. The problem does not have a specific objective function to minimize or maximize, but rather seeks to find a feasible solution that satisfies all the constraints. 

In other words, the goal is to find a mapping from letters to numbers such that the sum of the numbers corresponding to the letters in each word equals the given total for that word, and each letter and number is used only once.
