Lam's Problem
Lam's Problem is a classic constraint programming problem that involves finding a binary matrix that satisfies certain conditions. The problem is about constructing a matrix where each row and column sums up to a specific value, and for any two distinct rows, there is exactly one column where both rows have a zero.

The main decision variables in this problem are the elements of the matrix `m`, which is a binary matrix indexed by `Index`. Each element `m[r, c]` represents whether the cell at row `r` and column `c` is filled (true) or empty (false).

The key constraints in this problem are:
- Each row must have exactly 11 filled cells.
- Each column must have exactly 11 filled cells.
- For any two distinct rows, there must be exactly one column where both rows have an empty cell.

The objective of Lam's Problem is not to minimize or maximize a specific value, but rather to find a feasible solution that satisfies all the given constraints. In other words, the goal is to construct a binary matrix that meets the specified conditions.
