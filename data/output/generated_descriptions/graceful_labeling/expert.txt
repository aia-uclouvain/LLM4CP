name: Graceful Labeling Problem

description: The Graceful Labeling Problem is a classic problem in graph theory and constraint programming. Given a tree with a specified number of nodes and edges, the goal is to assign labels to the nodes from the set {0, 1, ..., m} such that the absolute difference between the labels of adjacent nodes is unique for each edge.

variables: $labels = (labels_1, labels_2, ..., labels_{nbreOfNodes})$, where $labels_i$ is a decision variable representing the label assigned to node $i$, and $labels_i \in \{0, 1, ..., nbreOfNodes-1\}$. These variables represent the possible labels for each node in the tree.

constraints: 
$\bigwedge_{i=1}^{nbreOfNodes} labels_i \in \{0, 1, ..., nbreOfNodes-1\} \wedge 
\bigwedge_{i=1}^{nbreOfNodes} \bigwedge_{j=i+1}^{nbreOfNodes} (adjacencyMatrix_{i,j} = 0 \vee |labels_i - labels_j| \neq |labels_k - labels_h|)$ for all $k, h$ such that $1 \leq k < h \leq nbreOfNodes$ and $adjacencyMatrix_{k,h} = 1$, and 
$\bigwedge_{i=1}^{nbreOfNodes} \bigwedge_{j=i+1}^{nbreOfNodes} labels_i \neq labels_j$. 
These constraints ensure that each node is assigned a unique label, and the absolute difference between the labels of adjacent nodes is unique for each edge.

objective: The objective of the problem is to find a satisfying assignment of labels to the nodes, i.e., to find a solution that satisfies all the constraints. The problem does not have a specific objective function to minimize or maximize.
