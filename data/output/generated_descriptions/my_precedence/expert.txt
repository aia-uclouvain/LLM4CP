name: Precedence Constraint Problem
description: The precedence constraint problem is a constraint satisfaction problem that involves finding an assignment of values to a sequence of variables such that the first occurrence of each value is in non-decreasing order. In other words, the first time a value is used, it must be before the first time a larger value is used.
variables: $x = (x_1, x_2, ..., x_n)$, where $x_i$ is a decision variable representing the value assigned to the $i^{th}$ position in the sequence, and $n$ is the length of the sequence. The variables $x_i$ are integer variables that take values from a finite domain.
constraints: $\begin{aligned} 
&\text{precedence\_me}(x) \\
&\text{contains}(i, x) \rightarrow \exists j \in \{1, 2, ..., n\} (x_j = i \land \forall k \in \{1, 2, ..., j-1\} (x_k \leq i)) \quad \forall i \in \{1, 2, ..., 4\} \\
&\text{contains}(k, x) \quad \forall k \in \{1, 2, ..., i-1\} \\
&x_i \in \{1, 2, 3, 4\} \quad \forall i \in \{1, 2, ..., n\}
\end{aligned}$, where the constraints ensure that the sequence $x$ satisfies the precedence condition, and each value in the domain is used at least once.
objective: The objective of the problem is to find a satisfying assignment of values to the sequence $x$ that satisfies the precedence constraint, with no explicit objective function to minimize or maximize. The goal is to find a feasible solution that meets the given constraints.
