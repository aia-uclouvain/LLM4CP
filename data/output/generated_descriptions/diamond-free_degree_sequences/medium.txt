**Diamond-Free Degree Sequence Problem**

The Diamond-Free Degree Sequence problem is a classic constraint programming problem that involves finding all unique degree sequences for a given number of nodes in a graph, such that the graph is diamond-free and satisfies certain properties.

**Problem Description**

Given a simple undirected graph G=(V,E), where V is the set of vertices and E is the set of undirected edges, the problem is to find all unique degree sequences d1,…,dn, where di≥di+1, such that:

* Each degree di>0 and di is modulo 3
* The sum of the degrees is modulo 12
* There exists a simple diamond-free graph with that degree sequence

A diamond is a set of four vertices in V such that there are at least five edges between those vertices. Conversely, a graph is diamond-free if it has no diamond as an induced subgraph, i.e., for every set of four vertices, the number of edges between those vertices is at most four.

**Decision Variables**

The main decision variables in this problem are:

* x: the adjacency matrix of the graph, where x[i,j] represents the presence or absence of an edge between vertices i and j
* degrees: the degree sequence of the graph, where degrees[i] represents the degree of vertex i

**Constraints**

The key constraints in this problem are:

* The graph is diamond-free: for every set of four vertices, the number of edges between those vertices is at most four
* The graph is undirected: x[i,j] = x[j,i] for all i and j
* The degree of each vertex is greater than zero and modulo 3: degrees[i] > 0 and degrees[i] mod 3 = 0 for all i
* The sum of the degrees is modulo 12: sum(degrees) mod 12 = 0
* Symmetry breaking: the degree sequence is non-increasing, i.e., degrees[i] ≥ degrees[i+1] for all i

**Objective**

The objective of this problem is to find all unique degree sequences that satisfy the above constraints. There is no explicit objective function to minimize or maximize. Instead, the goal is to enumerate all possible solutions that meet the constraints.
