name: Inflexions Problem
description: The Inflexions Problem is a constraint satisfaction problem that involves finding the number of inflexions in a given sequence of values. An inflexion is a point where the sequence changes from increasing to decreasing or vice versa, with the additional condition that the values at the inflexion points are different from their neighbors.
variables: $x = (x_1, x_2, ..., x_n)$, $a$: $x$ is an array of decision variables representing the sequence of values, and $a$ is a decision variable representing the number of inflexions in the sequence.
constraints: $x_i \in [1, 8]$ for $i = 1, 2, ..., n$, $a = \sum_{i=1}^{n-2} \sum_{j=i+2}^{n} (x_i < x_{i+1} \land x_i \neq x_{i+1} \land \forall k \in [i+2, j-1] (x_k = x_{k-1}) \land x_{j-1} \neq x_j \land \neg (x_{j-1} < x_j)) \lor (x_i > x_{i+1} \land x_i \neq x_{i+1} \land \forall k \in [i+2, j-1] (x_k = x_{k-1}) \land x_{j-1} \neq x_j \land \neg (x_{j-1} > x_j))$, $x = (1, 1, 4, 8, 8, 2, 7, 1)$: these constraints define the possible values for the sequence $x$ and the number of inflexions $a$, as well as the condition that the sequence $x$ must match a given sequence.
objective: The objective of the problem is to satisfy the given constraints, specifically to find a sequence $x$ that matches the given sequence and has a certain number of inflexions $a$.
