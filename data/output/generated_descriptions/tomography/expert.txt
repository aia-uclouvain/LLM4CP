name: Tomography Problem
description: The Tomography Problem is a classic problem in constraint programming where a matrix containing zeroes and ones is "x-rayed" vertically and horizontally, resulting in the total number of ones in each row and column. The goal is to reconstruct the original matrix from this information.
variables: $x_{i,j}$, where $i \in \{1, \ldots, r\}$ and $j \in \{1, \ldots, c\}$. These are decision variables representing the presence (1) or absence (0) of a one in the $i$-th row and $j$-th column of the matrix.
constraints: $\sum_{j=1}^{c} x_{i,j} = \text{row\_sums}_i$ for $i \in \{1, \ldots, r\}$ and $\sum_{i=1}^{r} x_{i,j} = \text{col\_sums}_j$ for $j \in \{1, \ldots, c\}$. These constraints ensure that the reconstructed matrix has the same row and column sums as the original matrix.
objective: The objective of the problem is to find a feasible solution that satisfies all the constraints, as there is no explicit objective function to minimize or maximize. The goal is to reconstruct the original matrix, and any solution that satisfies the constraints is considered valid.
