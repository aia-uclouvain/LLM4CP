name: Assignment Problem
description: The assignment problem is a fundamental combinatorial optimization problem that involves assigning a set of agents to a set of tasks in such a way that the total cost of the assignment is minimized. Each agent can be assigned to perform any task, incurring some cost that may vary depending on the agent-task assignment. The goal is to perform all tasks by assigning exactly one agent to each task.
variables: $x_{i,j}, z, \text{assigned}_i, \text{costs}_i$ where $x_{i,j}$ is a binary variable that equals 1 if task $j$ is assigned to agent $i$, $z$ is the total cost of the assignment, $\text{assigned}_i$ is the task assigned to agent $i$, and $\text{costs}_i$ is the cost of assigning the task to agent $i$. These variables represent the assignment of tasks to agents and the associated costs.
constraints: $\forall i \in I, j \in J: c_{i,j} \geq 0, x_{i,j} \geq 0, \sum_{j \in J} x_{i,j} \leq 1, \sum_{i \in I} x_{i,j} = 1, z = \sum_{i \in I, j \in J} c_{i,j} \cdot x_{i,j}, \text{assigned}_i = \sum_{j \in J} j \cdot x_{i,j}, \text{costs}_i = \sum_{j \in J} c_{i,j} \cdot x_{i,j}$ where these constraints ensure that each agent is assigned to at most one task, each task is assigned to exactly one agent, and the total cost is calculated correctly.
objective: minimize $z$
