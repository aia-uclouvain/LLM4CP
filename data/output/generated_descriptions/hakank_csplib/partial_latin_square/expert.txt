name: Partial Latin Square
description: A partial Latin square is a mathematical problem where a square matrix is filled with symbols from a set of possible symbols, such that each row and column contains each symbol at most once. However, unlike a traditional Latin square, each cell in the matrix has its own set of possible symbols, which can vary from cell to cell.
variables: $x_{i,j}$, where $i$ and $j$ are the row and column indices, respectively. These variables represent the symbol assigned to each cell in the matrix, and are chosen from a set of possible symbols defined by the sets $s_{i,j}$.
constraints: $x_{i,j} \in s_{i,j}$, $\forall i, j$, and $\text{latin\_square}(x)$, which ensures that each row and column contains each symbol at most once. The $\text{latin\_square}(x)$ constraint is defined as $\forall i ( \text{all\_different}([x_{i,j}]_{j=1}^n) \wedge \text{all\_different}([x_{j,i}]_{j=1}^n) )$.
objective: The objective of the problem is to find a satisfying assignment of symbols to the cells in the matrix, such that all constraints are met. The problem is a satisfaction problem, meaning that there is no objective function to minimize or maximize, but rather a set of feasible solutions that satisfy all constraints.
