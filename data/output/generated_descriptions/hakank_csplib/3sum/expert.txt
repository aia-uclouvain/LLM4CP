name: 3SUM (Three Elements That Sum To Zero)
description: The 3SUM problem is a well-known problem in computer science and mathematics, which involves finding three elements in an array of integers that sum to zero. The problem statement is to return the indices of any three elements in the array that add up to zero, with the condition that no index can be used more than once. If no such combination exists, the problem is considered unsatisfiable.
variables: $x = [x_1, x_2, ..., x_n]$, where $x_i$ is a binary decision variable that represents whether the $i^{th}$ element in the array is included in the sum. In other words, $x_i = 1$ if the $i^{th}$ element is included, and $x_i = 0$ otherwise.
constraints: $\sum_{i=1}^{n} nums_i \cdot x_i = 0 \wedge \sum_{i=1}^{n} x_i = m$, where the first constraint ensures that the sum of the selected elements is zero, and the second constraint ensures that exactly $m$ elements are selected.
objective: The objective of the problem is to satisfy the given constraints, i.e., to find a feasible solution that meets the conditions of summing to zero and selecting exactly $m$ elements. The problem does not have a specific objective function to minimize or maximize.
