Bin Packing Problem
The Bin Packing Problem is about packing a set of items of different weights into bins of limited capacity in a way that minimizes the number of bins used.

The main decision variables are x[i,j], which represent whether item i is packed in bin j (x[i,j] = 1) or not (x[i,j] = 0), and used[j], which indicate whether bin j is used (used[j] = 1) or not (used[j] = 0). The variable bin_total[j] represents the total weight of items packed in bin j.

The key constraints are: 
- Each item must be packed in exactly one bin.
- The total weight of items in a bin must not exceed the bin's capacity.
- A bin is considered used if it contains at least one item.

The objective of the problem is to minimize the number of bins used, which is represented by the variable obj, calculated as the sum of used[j] for all bins j. The model uses a constraint programming approach to find the optimal packing of items into bins.
