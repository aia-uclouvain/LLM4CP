name: The Sixth Trial (Smullyan) puzzle
description: The problem is a classic puzzle where there are two doors, behind one of which is a lady and behind the other is a tiger. The doors have statements on them, and the goal is to figure out which door hides which prize. The statement on door 1 is "The lady is behind the other door" and the statement on door 2 is "The lady is behind door 1". The puzzle is to determine which door hides the lady and which door hides the tiger.
variables: $x_{i,j}, t_i, d_i$ where $x_{i,j}$ represents whether door $i$ hides prize $j$ (1 for yes, 0 for no), $t_i$ represents whether the statement on door $i$ is true (1 for yes, 0 for no), and $d_i$ are dummy variables used to model the logical constraints.
constraints: $\forall i \in D: \sum_{j \in P} x_{i,j} = 1, \forall i \in D: x_{i,i} = t_i, x_{1,1} - x_{2,1} - 2d_1 \leq -1, x_{1,1} - x_{2,1} - d_1 \geq -1, x_{1,1} - x_{2,1} + 2d_2 \geq 1, x_{1,1} - x_{2,1} + 2d_2 \leq 2, d_1 + d_2 - t_1 \leq 1, d_1 + d_2 - 2t_1 \geq 0, x_{1,1} = t_2$ which represent the following: each door hides one prize, the statement on each door is true if and only if the door hides the corresponding prize, the statement on door 1 is true if and only if the lady is behind door 2, the statement on door 2 is true if and only if the lady is behind door 1.
