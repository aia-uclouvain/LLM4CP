name: Costas Array Problem
description: The Costas Array Problem is a combinatorial problem that involves finding a permutation of numbers from 1 to n, such that the differences between the elements in each row of the triangular difference table are distinct. This problem has applications in various fields, including mathematics, computer science, and engineering.
variables: $n, costas = (costas_1, costas_2, ..., costas_n), differences = (differences_{i,j})_{i,j=1}^n$ 
These variables represent the following: $n$ is the size of the Costas array, $costas$ is a permutation of numbers from 1 to $n$ representing the Costas array, and $differences$ is a matrix where $differences_{i,j}$ represents the difference between the $j^{th}$ and $(j-i)^{th}$ elements of the Costas array.
constraints: 
$\forall i \in \{1,2,...,n\}: costas_i \in \{1,2,...,n\}$, 
$\forall i,j \in \{1,2,...,n\}: differences_{i,j} \in \{-n+1,-n+2,...,n-1\}$, 
$\forall i \in \{1,2,...,n\}: \forall j \in \{1,2,...,i\}: differences_{i,j} = -n+1$, 
$all\_different(costas)$, 
$\forall i,j \in \{1,2,...,n\} \text{ where } i < j: differences_{i,j} = costas_j - costas_{j-i}$, 
$\forall i \in \{1,2,...,n-1\}: all\_different(\{differences_{i,j} | j \in \{i+1,i+2,...,n\}\})$, 
$\forall i,j \in \{1,2,...,n\} \text{ where } i < j: differences_{i,j} \neq 0$, 
$\forall k,l \in \{3,4,...,n\} \text{ where } k < l: differences_{k-2,l-1} + differences_{k,l} = differences_{k-1,l-1} + differences_{k-1,l}$
These constraints ensure that the Costas array is a permutation of numbers from 1 to $n$, the differences matrix is correctly calculated, all rows of the differences matrix have distinct elements, and some redundant constraints are added to speed up the search.
objective: The objective of the problem is to find all possible Costas arrays of size $n$, which is equivalent to finding all satisfying assignments of the decision variables. The problem does not have a clear minimize or maximize objective, but rather a satisfy objective, where the goal is to find all possible solutions that satisfy the given constraints.
