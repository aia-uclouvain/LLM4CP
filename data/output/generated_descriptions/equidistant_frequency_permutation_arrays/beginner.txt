Equidistant Frequency Permutation Arrays
The problem is about creating a set of special words, called code words, that follow certain rules. 
The main variables are: 
- the number of different symbols (like letters) that can be used, 
- how many times each symbol should appear in each word, 
- the total number of code words we want to create, and 
- the minimum number of differences between any two code words.
These rules are: 
- each code word must be made up of the same symbols repeated a certain number of times, 
- each pair of code words must have a specific number of differences.
The goal is to find a set of code words that meet these rules, with the specific number of code words given.
