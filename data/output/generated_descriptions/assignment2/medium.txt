Assignment Problem
The Assignment Problem is about finding the optimal way to assign a set of agents to a set of tasks, where each agent can only be assigned to one task, and each task can only be assigned to one agent. 

The main decision variables in this problem are represented by the 2D array `x`, where `x[i, j]` is a binary variable that equals 1 if agent `i` is assigned to task `j`, and 0 otherwise. 

The key constraints in this problem are: 
- Each agent can only be assigned to one task, which means that for each agent `i`, the sum of `x[i, j]` over all tasks `j` must equal 1. 
- If the number of agents is equal to the number of tasks, then each task can only be assigned to one agent, which means that for each task `j`, the sum of `x[i, j]` over all agents `i` must equal 1. 
- If the number of agents is not equal to the number of tasks, then each task can either be assigned to one agent or not assigned to any agent, which means that for each task `j`, the sum of `x[i, j]` over all agents `i` must be less than or equal to 1. 

The objective of the Assignment Problem is to minimize the total cost of the assignments, which is calculated by summing the product of the assignment variables `x[i, j]` and the corresponding costs `cost[i, j]` over all agents `i` and tasks `j`. In other words, the goal is to find the assignment of agents to tasks that results in the lowest total cost.
