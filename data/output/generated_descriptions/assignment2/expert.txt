name: Assignment Problem
description: The assignment problem is a classic problem in operations research and computer science that involves finding the optimal way to assign a set of agents to a set of tasks. In this problem, we have a set of rows (agents) and a set of columns (tasks), and each cell in the matrix represents the cost of assigning a particular agent to a particular task. The goal is to find the optimal assignment of agents to tasks that minimizes the total cost.
variables: $x_{i,j}$, where $x_{i,j}$ is a binary decision variable that represents whether agent $i$ is assigned to task $j$. These variables are used to determine the optimal assignment of agents to tasks.
constraints: $\sum_{j=1}^{cols} x_{i,j} = 1$ for each row $i$, $\sum_{i=1}^{rows} x_{i,j} = 1$ for each column $j$ if $rows = cols$, and $\sum_{i=1}^{rows} x_{i,j} \leq 1$ for each column $j$ if $rows \neq cols$, $total\_sum = \sum_{i=1}^{rows} \sum_{j=1}^{cols} x_{i,j} \* cost_{i,j}$. These constraints ensure that each agent is assigned to exactly one task, and each task is assigned to at most one agent if there are more tasks than agents. The total cost is calculated as the sum of the costs of each assignment.
objective: minimize $total\_sum$
