name: Four Number Problem
description: The Four Number Problem is a constraint satisfaction problem where we are given an array of 4 random numbers between 1 and 10, and we need to find 3 numbers between 1 and 10 such that all the 4 numbers in the array can be generated by adding the 3 numbers in different combinations.
variables: $x = [x_1, x_2, x_3]$, where $x_1, x_2, x_3$ are decision variables representing the 3 numbers to be found, and $r = [r_1, r_2, r_3, r_4]$ is an input array of 4 random numbers.
constraints: $\forall i \in \{1, 2, 3, 4\}, \exists j_1, j_2, j_3 \in \{0, 1\}$ such that $j_1*x_1 + j_2*x_2 + j_3*x_3 = r_i$ and $x_1 \leq x_2 \leq x_3$, where the first constraint ensures that each number in the array can be generated by adding the 3 numbers, and the second constraint is a symmetry breaking constraint to ensure that the solution is unique.
objective: The objective of the problem is to find a feasible solution, i.e., to satisfy all the constraints, as it is a constraint satisfaction problem and not an optimization problem.
