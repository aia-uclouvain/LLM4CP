Heterosquare problem
The Heterosquare problem is about finding a square matrix of a given size where each cell contains a distinct integer from 1 to the square of the size, and the sums of the rows, columns, and diagonals are all different.
The main decision variables are the elements of the matrix `mat`, which represent the integers in the square, and the variables `row_sums`, `col_sums`, `diag1`, and `diag2`, which represent the sums of the rows, columns, and diagonals, respectively.
The key constraints are that all the entries in the matrix should be different, all the sums should be different, and the sums of the rows, columns, and diagonals should be calculated correctly. In plain language, this means that each number from 1 to the square of the size should appear only once in the matrix, and the sums of the numbers in each row, column, and diagonal should be unique.
The objective of the problem is to find a solution that satisfies all the constraints, which means finding a heterosquare of the given size.
