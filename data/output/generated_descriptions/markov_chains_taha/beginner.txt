Markov Chains Fertilizer Problem
The problem is about a company that produces fertilizer and wants to determine the best way to manage its production process. 
The main variables are: 
- p, which represents the probability of being in a certain state of production, 
- mean_first_return_time, which represents the average time it takes to return to a certain state, 
- tot_cost, which represents the total cost of production, 
- mat, which is a table showing the probability of moving from one state to another, 
- cost, which is a list of costs associated with each state.
The basic restrictions are that the probabilities must add up to 1, and the total cost is calculated by multiplying the cost of each state by the probability of being in that state. 
The goal is to find the steady state probabilities and the total cost, given the transition matrix and the costs associated with each state.
