name: Markov Chain Steady State Problem with Cost Optimization
description: This problem involves finding the steady state probabilities of a Markov chain, which represents the long-term behavior of a system that can be in one of several states, and then using these probabilities to calculate the total cost of being in each state. The system is defined by a transition matrix, where each entry represents the probability of moving from one state to another. The goal is to find the steady state probabilities and the total cost, given the transition matrix and the cost of being in each state.
variables: $p = [p_1, p_2, ..., p_n], mean\_first\_return\_time = [mfrt_1, mfrt_2, ..., mfrt_n], tot\_cost$ 
These are the decision variables, where $p_i$ represents the steady state probability of being in state $i$, $mfrt_i$ represents the mean first return time to state $i$, and $tot\_cost$ represents the total cost of being in each state.
constraints: $p_i = \sum_{j=1}^{n} p_j \* mat_{j,i} \forall i \in \{1, 2, ..., n\}, \sum_{i=1}^{n} p_i = 1.0, mfrt_i = \frac{1}{p_i} \forall i \in \{1, 2, ..., n\}, tot\_cost = \sum_{i=1}^{n} cost_i \* p_i$ 
These constraints ensure that the steady state probabilities satisfy the Markov chain equations, the probabilities sum to 1, the mean first return times are calculated correctly, and the total cost is calculated correctly.
objective: The objective of this problem is not explicitly stated as a minimization or maximization problem, but rather as a satisfaction problem, where the goal is to find the steady state probabilities and the total cost that satisfy the given constraints. However, the total cost $tot\_cost$ can be considered as the value of interest, and the problem can be seen as finding the total cost given the transition matrix and the cost of being in each state.
