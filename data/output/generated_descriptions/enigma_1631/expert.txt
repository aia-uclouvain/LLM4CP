name: Joe's Pyramid
description: The problem is about a pyramid with 21 stones, each marked with a different one or two-digit positive number. The stones are arranged in a pyramid shape, and each stone rests on two others. The number on each stone is the sum of the numbers on the two stones it rests on. The goal is to find the number on the top stone, denoted as X.
variables: $x_1, x_2, ..., x_{21}$, where $x_i$ represents the number marked on the $i^{th}$ stone, and is a decision variable that can take any value between 0 and 99.
constraints: $x_1 = x_2 + x_3, x_2 = x_4 + x_5, x_3 = x_5 + x_6, x_4 = x_7 + x_8, x_5 = x_8 + x_9, x_6 = x_9 + x_{10}, x_7 = x_{11} + x_{12}, x_8 = x_{12} + x_{13}, x_9 = x_{13} + x_{14}, x_{10} = x_{14} + x_{15}, x_{11} = x_{16} + x_{17}, x_{12} = x_{17} + x_{18}, x_{13} = x_{18} + x_{19}, x_{14} = x_{19} + x_{20}, x_{15} = x_{20} + x_{21}, \forall i \neq j, x_i \neq x_j$, which represent the relationships between the numbers on the stones and ensure that all numbers are distinct.
objective: The objective of the problem is to satisfy all the constraints, and the solution is the value of $x_1$, which represents the number on the top stone, X. There is no explicit objective function to minimize or maximize, as the goal is to find a feasible solution that satisfies all the constraints.
