name: Message Sending Puzzle
description: The problem describes a situation where a teacher wants to send a message to Robert through a network of students. The teacher asks Craig to tell Robert, and Craig will either tell Robert directly or send the message to everyone he meets, who will then pass it on. The goal is to determine if the teacher will wait forever for Robert and to find all possible paths from Craig to Robert.
variables: $x = [x_1, x_2, ..., x_{10}]$, where $x_i$ represents the $i^{th}$ student in the path from Craig to Robert, with $x_1 = Craig$ and $x_n = Robert$ if a path exists, and $x_i = 0$ otherwise. The $graph$ array represents the connections between students.
constraints: $all\_different\_except\_0(x)$: all students in the path must be different except for the zeros at the end, $x_1 = Craig$: the path starts with Craig, $last\_not\_0(x, Robert)$: the last non-zero student in the path is Robert, $all\_paths(x, graph)$: each student in the path must be connected to the previous one.
objective: The problem is a satisfaction problem, where the goal is to find all possible paths from Craig to Robert, rather than minimizing or maximizing a specific value.
