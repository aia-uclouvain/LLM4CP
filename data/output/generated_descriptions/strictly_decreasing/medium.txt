**Problem Name:** Constraint Satisfaction Problem with Global Constraints

**Description:** This problem is about finding a solution that satisfies a set of global constraints, specifically strictly decreasing, strictly increasing, and decreasing constraints, on a given array of variables.

**Decision Variables:** The main decision variable is an array `x` of length `n`, where each element `x[i]` is a variable that can take a value between 1 and 8. These variables represent the values to be assigned to the array.

**Key Constraints:**

* **Strictly Decreasing Constraint:** Each element in the array must be strictly less than the previous element.
* **Strictly Increasing Constraint:** Each element in the array must be strictly greater than the previous element.
* **Decreasing Constraint:** Each element in the array must be less than or equal to the previous element.
* **Assignment Constraint:** The array `x` must be assigned a specific value, which is defined by the `cp1d` predicate.

**Objective:** The objective of this problem is to find a solution that satisfies all the given constraints, specifically the strictly decreasing constraint in this case. The problem does not have a specific objective function to minimize or maximize, but rather aims to find a feasible solution that meets all the constraints.
