Weight Distribution Problem
The problem is about distributing different weights into a few containers so that the difference in weight between the containers is as small as possible.
The main variables are: 
- masses: the different weights to be distributed
- x: which container each weight goes into
- bins: the total weight in each container
- diffs: the difference in weight between each pair of containers
- total_diffs: the total of all the differences in weight between containers
The basic restrictions are: 
- each weight can only go into one container
- the total weight of all the weights must be distributed among the containers
- the difference in weight between each pair of containers must be calculated
The goal is to minimize the total difference in weight between all the containers, which means finding a way to distribute the weights so that the containers have as close to the same weight as possible.
