name: Weight Distribution Problem

description: The Weight Distribution Problem is a classic problem in constraint programming that involves distributing a set of masses of different values into a fixed number of buckets, such that the difference in mass between the filled buckets is minimized. The problem can be seen as a variation of the bin packing problem, where the goal is to pack the masses into the buckets in a way that minimizes the maximum difference in mass between any two buckets.

variables: $x_j \in \{1, 2, ..., num\_buckets\}$ for $j = 1, 2, ..., num\_masses$, $bins_b \in \{0, 1, ..., c \cdot num\_buckets\}$ for $b = 1, 2, ..., num\_buckets$, $diffs_{i,j} \in \{0, 1, ..., c\}$ for $i, j = 1, 2, ..., num\_buckets$. 
These variables represent the assignment of each mass to a bucket ($x_j$), the total mass in each bucket ($bins_b$), and the absolute difference in mass between each pair of buckets ($diffs_{i,j}$).

constraints: $bins_b = \sum_{j=1}^{num\_masses} masses_j \cdot \delta(b = x_j)$ for $b = 1, 2, ..., num\_buckets$, $\sum_{b=1}^{num\_buckets} bins_b = \sum_{j=1}^{num\_masses} masses_j$, $diffs_{i,j} = |bins_i - bins_j|$ for $i, j = 1, 2, ..., num\_buckets$, $bins_1 \geq bins_2 \geq ... \geq bins_{num\_buckets}$. 
These constraints ensure that the total mass in each bucket is calculated correctly, the total mass in all buckets is equal to the total mass of all masses, the absolute difference in mass between each pair of buckets is calculated correctly, and the masses in the buckets are ordered in non-increasing order to break symmetry.

objective: minimize $\sum_{i,j=1}^{num\_buckets} diffs_{i,j}$, which represents the total difference in mass between all pairs of buckets.
