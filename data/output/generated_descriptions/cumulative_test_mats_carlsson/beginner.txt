Resource Allocation Problem
The problem is about allocating resources to tasks without exceeding a certain limit. 
The main variables are: 
- Start time (when a task begins)
- Duration (how long a task takes)
- Height (the amount of resources a task requires)
The basic restrictions are: 
- Each task has a specific start time, duration, and resource requirement
- The total resources used by all tasks at any given time cannot exceed a certain limit
The goal is to find a schedule for the tasks that satisfies these restrictions, without trying to minimize or maximize anything, just finding a valid solution.
