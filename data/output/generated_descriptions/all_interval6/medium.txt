All Interval Series Problem
The All Interval Series problem is about finding a sequence of musical notes, represented by integers, such that each note appears exactly once and the differences between consecutive notes are all unique.

The main decision variables are `series` and `differences`. The `series` variable represents the sequence of notes, where each note is an integer between 0 and n-1. The `differences` variable represents the intervals between consecutive notes in the sequence.

The key constraints are: 
- each note appears exactly once in the sequence, 
- the differences between consecutive notes are all unique, and 
- the first note is less than the last note, which helps to reduce symmetry in the solution space.

The problem's objective is to find a sequence that satisfies these constraints. There is no explicit objective function to minimize or maximize; the goal is to find a feasible solution that meets all the constraints. In other words, the problem is a satisfaction problem, where the aim is to find a sequence of notes that satisfies the given conditions, rather than optimizing a specific criterion.
