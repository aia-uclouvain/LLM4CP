All Interval Problem
The problem is about creating a sequence of musical notes where each note is used exactly once and the differences between consecutive notes are all unique.
The main variables are: 
- series: the sequence of musical notes
- differences: the differences between consecutive notes in the series
The basic restrictions are: 
- each note can only be used once
- the differences between consecutive notes must all be different
- the first note must be less than the last note
The goal of the problem is to find a sequence of notes that satisfies these rules, with no specific objective to minimize or maximize, the goal is to simply find a valid sequence.
