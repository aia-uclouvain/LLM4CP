Generalized Assignment Problem
The Generalized Assignment Problem is about assigning a set of jobs to a set of agents, where each job must be assigned to exactly one agent, and the total resources consumed by all jobs assigned to an agent must not exceed the agent's capacity.

The main decision variables in this problem are represented by the array `x`, where `x[i,j]` is a binary variable that equals 1 if job `j` is assigned to agent `i`, and 0 otherwise. The `a` array represents the resources consumed by allocating job `j` to agent `i`, and the `b` array represents the resource capacity of each agent. The `c` array represents the cost of allocating job `j` to agent `i`.

The key constraints in this problem are: 
- Each job must be assigned to exactly one agent, which is ensured by the constraint that the sum of `x[i,j]` over all agents `i` equals 1 for each job `j`. 
- The total resources consumed by all jobs assigned to an agent must not exceed the agent's capacity, which is ensured by the constraint that the sum of `a[i,j] * x[i,j]` over all jobs `j` is less than or equal to `b[i]` for each agent `i`.

The objective of the problem is to find the cheapest assignment of jobs to agents, which is achieved by minimizing the total cost `obj` that is calculated as the sum of `c[i,j] * x[i,j]` over all agents `i` and jobs `j`. The problem can also be formulated as a maximization problem, where the goal is to maximize the total cost.
