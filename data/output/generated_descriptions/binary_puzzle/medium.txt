Binary Puzzle
The Binary Puzzle is a classic constraint programming problem where the goal is to fill a square grid with binary values (0s and 1s) such that certain rules are satisfied.
The main decision variables in this problem are the elements of the 2D array `x`, where `x[i, j]` represents the value of the cell at row `i` and column `j`. Each `x[i, j]` can take on a value of either 0 or 1.
The key constraints in the Binary Puzzle are: 
- Each cell must contain either a 0 or a 1.
- No three consecutive cells in a row or column can have the same value (i.e., no three 0s or 1s in a row).
- Each row and column must have an equal number of 0s and 1s.
The objective of the Binary Puzzle is to find a solution that satisfies all the constraints, rather than minimizing or maximizing a specific function. In other words, the goal is to find a valid configuration of the grid that meets all the rules.
