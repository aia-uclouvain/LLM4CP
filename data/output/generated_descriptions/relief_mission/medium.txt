Relief Mission
The Relief Mission problem is about coordinating relief efforts after a disaster by dropping two identical relief packages from the air to minimize the total distance that villagers must travel to pick up the supplies.

The main decision variables are the locations where the two relief packages will be dropped, represented by the 2D array `x`. Each element `x[i, j]` represents the row and column of the cell where a package will be dropped.

The key constraints are:
- Each villager must travel to the nearest package drop location.
- The distance between each villager and the nearest package drop location is calculated using the squared Euclidean distance.
- The total distance traveled by all villagers is the sum of the individual distances.
- The model also includes a symmetry-breaking constraint to avoid duplicate solutions.

The objective of the problem is to minimize the total distance that all villagers must travel to pick up the relief supplies, which is represented by the variable `total_dist`. The model uses a minimization algorithm to find the optimal locations for the two relief packages that result in the minimum total distance.
