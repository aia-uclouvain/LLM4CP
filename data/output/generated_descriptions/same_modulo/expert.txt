name: Same Modulo Problem
description: The Same Modulo problem is a constraint satisfaction problem where we have two collections of variables and a modulus M. The goal is to find values for the variables such that the number of variables in each collection that have the same remainder when divided by M is the same.
variables: $v1_1, v1_2, ..., v1_n, v2_1, v2_2, ..., v2_n$ - These are the decision variables representing the values of the two collections, where $n$ is the number of variables in each collection, and $v1_i$ and $v2_i$ are the values of the $i^{th}$ variable in the first and second collections, respectively.
constraints: $m1_i = v1_i \mod M$, $m2_i = v2_i \mod M$, $\forall i \in \{1, 2, ..., n\}$, $global\_cardinality(m1, array1d(0..M-1, set2array(index\_set(gcc))), gcc)$, $global\_cardinality(m2, array1d(0..M-1, set2array(index\_set(gcc))), gcc)$ - These constraints ensure that the number of variables in each collection that have the same remainder when divided by M is the same. The $global\_cardinality$ constraints count the number of variables in each collection that have each possible remainder.
objective: The objective of the problem is to satisfy all the constraints, i.e., to find values for the variables such that the Same Modulo constraint holds. There is no explicit objective function to minimize or maximize.
