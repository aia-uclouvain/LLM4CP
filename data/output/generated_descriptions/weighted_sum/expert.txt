name: Weighted Sum Problem
description: The Weighted Sum Problem is a constraint satisfaction problem where the goal is to find values for a set of variables such that their weighted sum satisfies a certain condition. The problem involves two arrays of variables, coeffs and x, where coeffs represents the weights and x represents the values to be summed. The weighted sum is then compared to a limit using a comparison operator.
variables: $s, coeffs = [c_1, c_2, ..., c_n], x = [x_1, x_2, ..., x_n]$ where $s$ is the weighted sum, $coeffs$ is an array of coefficients or weights, and $x$ is an array of variables to be summed. The coefficients and variables are decision variables that are assigned values to satisfy the constraints.
constraints: $s = \sum_{i=1}^{n} c_i \cdot x_i, \quad c = [1, 2, -3]$ where the first constraint represents the weighted sum, and the second constraint fixes the values of the coefficients.
objective: The objective of the problem is to satisfy the constraints, there is no explicit objective function to minimize or maximize. The goal is to find values for the decision variables that make the weighted sum equal to a certain value, in this case, the value is not specified, but the model is looking for a feasible solution.
