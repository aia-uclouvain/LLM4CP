name: Number Cross Puzzle
description: The Number Cross Puzzle is a problem where you have to place numbers from 1 to N in a cross-shaped grid such that no two adjacent cells (horizontally, vertically, or diagonally) contain adjacent numbers. The goal is to find all possible solutions for a given grid size.
variables: $x = (x_1, x_2, ..., x_n)$, where $x_i$ represents the number assigned to the $i^{th}$ cell in the grid, and $n$ is the total number of cells in the grid. These variables are used to determine the placement of numbers in the grid.
constraints: $\forall i \in \{1, 2, ..., n\}, x_i \in \{1, 2, ..., n\}$, $all\_different(x)$, and $\forall (i, j) \in adj, |x_i - x_j| > 1$, where $adj$ represents the set of adjacent cell pairs in the grid. These constraints ensure that each cell is assigned a unique number, and that no two adjacent cells contain adjacent numbers.
objective: The objective of the problem is to find all possible solutions, i.e., all possible assignments of numbers to cells that satisfy the given constraints. There is no explicit objective function to minimize or maximize, but rather a satisfaction problem where the goal is to find all feasible solutions.
