Enigma problem 1570 (Set of cubes)
The problem is about finding a set of perfect cubes (numbers that can be expressed as the cube of an integer, like 1, 8, 27, etc.) that use each of the digits 0 to 9 exactly once, and the sum of these cubes is as small as possible.

The main variables are: 
- x, an array of digits from 0 to 9, 
- num1, num2, num3, num4, which are the perfect cubes, 
- len1, len2, len3, which represent the number of digits in each cube, 
- total, which is the sum of the cubes.

The basic restrictions or rules are: 
- Each digit from 0 to 9 must be used exactly once, 
- Each number (num1, num2, num3, num4) must be a perfect cube, 
- The numbers must be in increasing order (num1 < num2 < num3 < num4).

The goal of the problem is to minimize the total sum of the cubes.
