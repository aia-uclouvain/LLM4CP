**Vessel Loading Problem**
The Vessel Loading problem is a classic constraint programming problem that involves loading containers of different sizes and classes onto a ship's deck while satisfying various constraints.

**Decision Variables**
The main decision variables in this problem are:
- The x and y coordinates of each container, representing its position on the deck.
- The width and height of each container, which can be rotated by 90 degrees.
- The orientation of each container (0 or 1), indicating whether it is rotated or not.

**Constraints**
The key constraints in this problem are:
- Each container must be placed within the boundaries of the deck.
- Containers cannot overlap with each other.
- Containers of certain classes must be separated by a minimum distance in either the x or y direction.
- The width and height of each container must be consistent with its orientation.

**Objective**
The objective of the Vessel Loading problem is to find a feasible placement of all containers on the deck that satisfies the given constraints. There is no explicit objective function to minimize or maximize, but rather a requirement to find a valid solution that meets all the constraints.
