name: Balance Interval Problem
description: The Balance Interval Problem is a constraint satisfaction problem that involves assigning values to a set of variables such that the difference between the maximum and minimum number of occurrences of values within certain intervals is minimized. The problem is defined by a set of variables, an interval size, and a balance value.
variables: $x_1, x_2, ..., x_n, bal$ - These are the decision variables where $x_i$ represents the value assigned to the $i^{th}$ variable and $bal$ represents the balance value, which is the difference between the maximum and minimum number of occurrences of values within certain intervals.
constraints: $bal = c_max - c_min, c_max = max(counts), min_except_0(c_min, counts), counts_j = \sum_{i=1}^{n} (x_i \geq intervals_{j,1} \wedge x_i \leq intervals_{j,2}), intervals_{j,1} = interval_size \cdot (j-1), intervals_{j,2} = interval_size \cdot j - 1$ - These constraints define the balance value, the maximum and minimum number of occurrences of values within certain intervals, and the intervals themselves.
objective: The objective of the problem is to satisfy the given constraints, specifically to find values for the variables $x_i$ such that the balance value $bal$ is equal to the given value, in this case $bal = 3$.
