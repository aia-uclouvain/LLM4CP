name: Letter Square Problem

description: The Letter Square Problem is a classic constraint programming problem where the goal is to create a square matrix of size n x n, filled with letters (represented by numbers) and blanks, such that each row and column contains each letter at most once, and the first non-blank letter seen from each side (top, bottom, left, and right) matches the given hints.

variables: $x_{i,j}$, where $i$ and $j$ are indices ranging from 1 to $n$, representing the value (letter or blank) in the $i$-th row and $j$-th column of the matrix. These variables are decision variables that determine the arrangement of letters and blanks in the matrix.

constraints: 
$\forall i \in [1,n]: \forall j \in [1,n]: (x_{i,j} \neq 0 \wedge x_{i,k} \neq 0) \rightarrow x_{i,j} \neq x_{i,k}$, 
$\forall i \in [1,n]: \forall j \in [1,n]: (x_{j,i} \neq 0 \wedge x_{k,i} \neq 0) \rightarrow x_{j,i} \neq x_{k,i}$, 
$\forall i \in [1,n]: \sum_{j=1}^{n} (x_{i,j} = 0) = num\_zeros$, 
$\forall i \in [1,n]: \sum_{j=1}^{n} (x_{j,i} = 0) = num\_zeros$, 
$\forall i \in [1,n]: row\_upper[i] > 0 \rightarrow \exists j \in [1,n]: (\forall k \in [1,j-1]: x_{k,i} = 0) \wedge x_{j,i} = row\_upper[i]$, 
$\forall i \in [1,n]: row\_lower[i] > 0 \rightarrow \exists j \in [1,n]: (\forall k \in [j+1,n]: x_{k,i} = 0) \wedge x_{j,i} = row\_lower[i]$, 
$\forall i \in [1,n]: col\_left[i] > 0 \rightarrow \exists j \in [1,n]: (\forall k \in [1,j-1]: x_{i,k} = 0) \wedge x_{i,j} = col\_left[i]$, 
$\forall i \in [1,n]: col\_right[i] > 0 \rightarrow \exists j \in [1,n]: (\forall k \in [j+1,n]: x_{i,k} = 0) \wedge x_{i,j} = col\_right[i]$. 
These constraints ensure that each row and column contains each letter at most once, the number of blanks in each row and column is correct, and the first non-blank letter seen from each side matches the given hints.

objective: The objective of the problem is to find a feasible solution that satisfies all the constraints, as there is no explicit objective function to minimize or maximize. The problem is a satisfaction problem, where the goal is to find an arrangement of letters and blanks that meets all the given conditions.
