name: Employee Salary Statistics

These variables represent the maximum salary, minimum salary, average salary, the set of high-paid employees, and the number of high-paid employees, respectively. They are used to store the computed statistics for the employees.
constraints: true. This constraint is a trivial constraint that is always satisfied, indicating that there are no specific constraints that need to be enforced in this problem. The calculations for the statistics are performed using the given data and the variables are assigned values based on these calculations.
objective: The objective of the problem is not to minimize or maximize a specific value, but rather to compute various statistics for the employees.
