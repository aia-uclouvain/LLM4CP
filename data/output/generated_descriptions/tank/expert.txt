name: Tank Attack Puzzle
description: The Tank Attack Puzzle is a classic problem in constraint programming where we have a grid of tanks, and each tank has a number on it. The number on each tank represents the total number of tanks that are "attacking" it, where a tank is considered to be attacking another tank if they are in the same row or column and the absolute difference between their positions is equal to the number on the attacking tank. The goal is to find an assignment of numbers to the tanks such that each tank is assigned a number that is equal to the total number of tanks attacking it.
variables: $x_{i,j,k}, res_{i,j}$, where $x_{i,j,k}$ is a binary variable that is 1 if the number on tank $(i,j)$ is $k$, and $res_{i,j}$ is the output result, which is the number assigned to tank $(i,j)$.
constraints: $\sum_{p \in N, p \neq j} x_{i,p,|j-p|} + \sum_{q \in N, q \neq i} x_{q,j,|i-q|} = \sum_{k \in M} kx_{i,j,k}$, $\sum_{k \in M} x_{i,j,k} = 1$, $x_{i,j,k} = x_{n+1-i,n+1-j,k}$, $x_{i,j,k} = x_{j,n+1-i,k}$, $res_{i,j} = \sum_{k \in M} kx_{i,j,k}$, which represent the constraints that the number on each tank is equal to the total number of tanks attacking it, each tank has a unique number, symmetry constraints to reduce the search space, and the output result is the number assigned to each tank.
objective: The objective of the problem is to satisfy all the constraints, there is no explicit objective function to minimize or maximize. The goal is to find a feasible assignment of numbers to the tanks that satisfies all the constraints.
