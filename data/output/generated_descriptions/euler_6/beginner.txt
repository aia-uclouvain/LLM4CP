Problem name: Project Euler problem 6
This problem is about comparing two ways of calculating a number using the first 100 natural numbers (1, 2, 3, ..., 100). 
The main variables are: 
- ss (sum of the squares), which means adding up the squares of each number (1^2 + 2^2 + 3^2 + ... + 100^2)
- s (the sum), which means adding up all the numbers (1 + 2 + 3 + ... + 100)
- s2 (square of the sum), which means taking the sum of all the numbers and then squaring it
The basic restrictions or rules are: 
- we are only using the first 100 natural numbers
- we need to calculate the sum of the squares and the square of the sum
The goal of the problem is to find the difference between the square of the sum and the sum of the squares, which is calculated as s2 - ss.
