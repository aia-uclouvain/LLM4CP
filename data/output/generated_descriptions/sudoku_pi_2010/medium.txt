Sudoku
Sudoku is a classic constraint programming problem that involves filling in a 9x9 grid with numbers from 1 to 9, such that each row, column, and 3x3 sub-grid contains each number exactly once.
The main decision variables in this problem are the elements of the 2D array `x`, where `x[i,j]` represents the number that should be placed in the cell at row `i` and column `j`.
The key constraints in this problem are: 
- each cell that has a given number (or "hint") must be filled with that number, 
- each row must contain each number from 1 to 9 exactly once, 
- each column must contain each number from 1 to 9 exactly once, and 
- each 3x3 sub-grid must contain each number from 1 to 9 exactly once.
The objective of this problem is not to minimize or maximize a specific function, but rather to find a feasible solution that satisfies all the constraints, which is achieved by using the `satisfy` keyword in the MiniZinc model.
