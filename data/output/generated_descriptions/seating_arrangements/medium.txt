Seating Arrangement Problem
The Seating Arrangement Problem is about organizing a party with multiple tables, each having a limited number of chairs. The goal is to assign each guest to a table while satisfying certain conditions.

The main decision variables are `x` and `tables`. The variable `x` is an array that assigns each guest to a table, and `tables` is an array of sets where each set represents the guests assigned to a particular table.

The key constraints are:
- Each table has a limited number of chairs, and the number of guests assigned to a table cannot exceed this limit.
- Guests who like each other must be assigned to the same table.
- Guests who dislike each other must be assigned to different tables.
- Symmetry breaking constraints are used to reduce the search space by fixing the table assignment for the first guest and ordering the tables by the guests assigned to them.

The problem's objective is to find a seating arrangement that satisfies all the given constraints. If no such arrangement exists, the problem is considered unsolvable. The model uses a satisfiability solver to find a solution that meets all the conditions, and if no solution is found, it indicates that the problem instance has no feasible solution.
