Coin Changing Problem
The problem is about finding the minimum number of coins needed to make any amount from 1 to 99 cents using different coin denominations.
The main variables are: 
- variables: the different coin denominations (e.g., 1, 2, 5, 10, 20, 50 cents)
- x: the number of coins of each denomination needed
- num_coins: the total number of coins needed
The basic restrictions or rules are: 
- each coin denomination must be unique
- the coin denominations must be in increasing order
- the total value of all coins must not exceed 99 cents
- it must be possible to make any amount from 1 to 99 cents using the selected coins
The goal is to minimize the total number of coins (num_coins) needed to make any amount from 1 to 99 cents.
