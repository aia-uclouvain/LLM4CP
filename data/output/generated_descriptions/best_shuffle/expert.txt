name: Best Shuffle Problem
description: The Best Shuffle Problem is a constraint programming problem that involves shuffling the characters of a given string in such a way that as many character values are in a different position as possible. The goal is to find a shuffled string that minimizes the number of positions where the character value remains the same.
variables: $shuffle_1, shuffle_2, ..., shuffle_{len}, num\_same$ - These are the decision variables where $shuffle_i$ represents the index of the character in the original string that should be placed at position $i$ in the shuffled string, and $num\_same$ represents the number of positions where the character value did not change.
constraints: $alldifferent(shuffle) \wedge num\_same = \sum_{i=1}^{len} (word[shuffle_i] = word_i) \wedge num\_same = 0$ - These constraints ensure that each character in the original string is placed at a unique position in the shuffled string ($alldifferent(shuffle)$), calculate the number of positions where the character value did not change ($num\_same$), and set $num\_same$ to 0 to find the best shuffle.
objective: Minimize $num\_same$ - The objective is to minimize the number of positions where the character value did not change, which means finding a shuffled string where as many character values are in a different position as possible.
