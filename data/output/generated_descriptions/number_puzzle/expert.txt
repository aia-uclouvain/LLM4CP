name: Alphametic Problem
description: This problem is an instance of an alphametic puzzle, where letters are replaced by numbers to form valid mathematical equations. The goal is to find an assignment of numbers to letters such that the resulting equations are true and all numbers are distinct.
variables: $a, b, c, d, e, f, g, h, i, j, k, l, m, n, o, p, q, r, s, t, u, v, w, x, y, z, rad01, rad02, ..., rad40, a1, a2, ..., a5, b1, b2, ..., b5, ..., z1, z2, ..., z5$ - These variables represent the numbers to be assigned to the letters, with $a, b, ..., z$ being the final 5-digit numbers, $rad01, ..., rad40$ being intermediate results, and $a1, ..., a5, ..., z1, ..., z5$ being the individual digits of the numbers.
constraints: $a = a1*10000+a2*1000+a3*100+a4*10+a5, ..., z = z1*10000+z2*1000+z3*100+z4*10+z5, rad01 = h5*1000+i4*1000+q2*100+u4*10+y1, ..., rad40 = h4*1000+i3*1000+q1*100+u3*10+z5, all\_different([a, ..., z, rad01, ..., rad40]), a >= 10000, ..., z >= 10000, rad01 >= 10000, ..., rad40 >= 10000$ - These constraints define the relationships between the numbers, ensure that all numbers are distinct, and enforce the range of the numbers to be between 10000 and 99999.
objective: The objective of this problem is to satisfy all the constraints, which means finding an assignment of numbers to letters such that the resulting equations are true and all numbers are distinct. There is no explicit objective function to minimize or maximize.
