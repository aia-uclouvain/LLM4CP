The Ormat Game problem.
The Ormat Game is a classic constraint programming problem that involves finding a solution to a puzzle where a set of rules must be satisfied. 
The main decision variables in the Ormat Game problem are the values assigned to each cell in the grid, which represent whether a cell is filled or empty.
The key constraints in the Ormat Game problem include ensuring that each row and column contains a specific number of filled cells, and that certain cells are either filled or empty based on the given rules. Additionally, there may be constraints that restrict the placement of filled cells in relation to each other.
The objective of the Ormat Game problem is to find a valid solution that satisfies all the given constraints, which can be viewed as a feasibility problem where the goal is to determine whether a solution exists, rather than optimizing a specific objective function.
