Bobsledders Puzzle
The Bobsledders puzzle is a classic constraint programming problem that involves determining the partners, countries, and sled colors of ten bobsledders, given certain hints and constraints.

The main decision variables in this problem are:
- `pairs`: an array that assigns each person to a pair
- `countries`: an array that assigns each pair to a country
- `sleds`: an array that assigns each pair to a sled color
- `partners`: an array that assigns each person to their partner

The key constraints in this problem can be described as follows:
- Each pair consists of one male and one female
- Each country, sled color, and partner is unique
- The given hints, such as "Betsy and Phil are the Green sled duo" and "Lulu is from USA", must be satisfied

The objective of this problem is to find a solution that satisfies all the given constraints, effectively determining the partners, countries, and sled colors of the ten bobsledders. The problem does not involve minimizing or maximizing a specific objective function, but rather finding a feasible solution that meets all the constraints.
