name: 3-Partition Problem
description: The 3-partition problem is a classic problem in computer science where we are given a multiset of integers and need to decide whether it can be partitioned into triples that all have the same sum. The problem requires finding a partition of the multiset into disjoint subsets, each containing exactly three elements, such that the sum of the numbers in each subset is equal.
variables: $a = (a_1, a_2, ..., a_n)$ and $a\_sum = (a\_sum_1, a\_sum_2, ..., a\_sum_n)$, where $a_i$ is a set of numbers representing the $i^{th}$ subset and $a\_sum_i$ is the sum of the numbers in the $i^{th}$ subset. These decision variables represent the partition of the multiset into subsets and the sum of each subset.
constraints: 
- $|nums| = n \* m$
- $a\_sum_i \geq 0$ for all $i \in 1..n$
- $a$ is a partition of $nums$
- $|a_i| = m$ for all $i \in 1..n$
- $a\_sum_i = \sum_{j \in a_i} j$ for all $i \in 1..n$
- $a\_sum_i = a\_sum_{i-1}$ for all $i \in 2..n$
These constraints ensure that the multiset is partitioned into subsets of size $m$, the sum of each subset is calculated correctly, and all subsets have the same sum.
objective: The objective of the problem is to find a feasible solution that satisfies all the constraints, i.e., to determine whether the given multiset can be partitioned into triples with the same sum. The problem does not have a specific objective function to minimize or maximize.
