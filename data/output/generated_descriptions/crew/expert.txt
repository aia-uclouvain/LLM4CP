name: Crew Allocation Problem
description: The Crew Allocation Problem is a classic problem in constraint programming that involves assigning a set of crew members to a set of flights, subject to certain constraints such as the number of crew members required for each flight, the languages spoken by the crew members, and the requirement that each crew member must have two flights off after attending a flight.
variables: $z, crew_{f,p}$ where $f \in 1..numFlights, p \in 1..numPersons$. These variables represent the number of persons working ($z$) and the crew schedule ($crew_{f,p}$), where $crew_{f,p}$ is a binary variable that is 1 if person $p$ is assigned to flight $f$ and 0 otherwise.
constraints: $\sum_{i=1}^{numPersons} crew_{f,i} = requiredCrew_{f,1}$, $\sum_{i=1}^{numPersons} attributes_{i,j} \cdot crew_{f,i} \geq requiredCrew_{f,j+1}$ for $j \in 1..5$, $crew_{f,p} + crew_{f+1,p} + crew_{f+2,p} \leq 1$ for $f \in 1..numFlights-2, p \in 1..numPersons$. These constraints ensure that each flight has the required number of crew members, that each flight has the required number of crew members with specific attributes (such as language skills), and that each crew member has two flights off after attending a flight.
objective: minimize $z$, which represents the number of persons working. The goal is to find a crew schedule that minimizes the number of crew members required to cover all flights while satisfying the constraints.
