name: Relative Sizes Puzzle (Enigma 1515)
description: The problem is a classic puzzle where whole numbers are expressed in words written in capital letters, with different letters standing for different digits, and the same letter consistently standing for the same digit. The goal is to find the value of "BEETLES" given two conditions: the total of four "BEES" and a "SNAIL" is greater than the total of eight "FLEAS" and an "ANT", and the total of 18 "ANTS" and a "GNAT" is greater than the total of 18 "FLIES" and a "FLEA".

constraints: $bees = 1000b + 100e + 10e + s, snail = 10000s + 1000n + 100a + 10i + l, fleas = 10000f + 1000l + 100e + 10a + s, ant = 100a + 10n + t, 4bees + snail > 8fleas + ant, ants = 1000a + 100n + 10t + s, gnat = 1000g + 100n + 10a + t, flies = 10000f + 1000l + 100i + 10e + s, flea = 1000f + 100l + 10e + a, 18ants + gnat > 18flies + flea, beetles = 1000000b + 100000e + 10000e + 1000t + 100l + 10e + s$ - These constraints define the relationships between the digits and the numerical values of the words, as well as the two conditions given in the problem.
objective: The objective of the problem is to satisfy all the constraints, with the ultimate goal of finding the value of "BEETLES".
