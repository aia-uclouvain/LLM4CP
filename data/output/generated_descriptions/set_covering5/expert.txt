name: Set Covering Problem with Preferences
description: The problem involves assigning tasks to employees in a security company, where each task has different qualifications and each employee has preferences for certain tasks. The goal is to assign one combination of tasks to each employee such that all tasks are covered exactly once and the total preference points are maximized.
variables: $x_1, x_2, ..., x_{19}$, where $x_i$ is a binary decision variable indicating whether combination $i$ is chosen or not. $preference\_points_p$ is a variable representing the total preference points for employee $p$.
constraints: $\sum_{j \in 1..num\_combinations \: where \: combinations\_person[j] = p} x_j = 1 \: \forall p \in 1..num\_persons$, which ensures that each employee is assigned exactly one combination. $\sum_{i \in 1..num\_combinations} x_i \* bool2int(j \in c_i) = 1 \: \forall j \in 1..num\_tasks$, which ensures that each task is covered exactly once. $preference\_points_p = \sum_{i \in 1..num\_combinations, j \in 1..num\_tasks \: where \: combinations\_person[i] = p} x_i \* bool2int(j \in c_i) \* preferences_{p,j}$, which calculates the total preference points for each employee.
objective: Maximize $sum\_pref\_points = \sum_{p \in 1..num\_persons} preference\_points_p$, which represents the total preference points for all employees.
