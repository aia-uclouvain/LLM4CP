Set Covering Problem
The problem is about creating a work schedule for a security company with three employees: Anders, Lotta, and Per. Each employee has different qualifications and preferences for various tasks, and the goal is to assign tasks to each employee so that all tasks are covered exactly once and the total preference points are maximized.

The main variables are:
- x: a decision variable that determines which combination of tasks to assign to each employee
- preference_points: the total preference points for each employee
- sum_pref_points: the total preference points for all employees

The basic restrictions or rules of the problem are:
- Each employee must work 7 or 8 hours per night
- Each task must be covered exactly once
- Each employee can only be assigned one combination of tasks

The goal of the problem is to maximize the total preference points, which means assigning tasks to each employee in a way that maximizes their individual preference points while still covering all tasks exactly once.
