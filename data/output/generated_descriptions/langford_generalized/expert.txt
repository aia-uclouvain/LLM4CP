name: Langford's Number Problem

description: The problem is to arrange k sets of numbers from 1 to n into a single sequence such that each appearance of the number m is m numbers apart from the last appearance of m. This means that the two 1's appear one number apart, the two 2's appear two numbers apart, and so on, up to the two n's which appear n numbers apart.

variables: $a = (a_1, a_2, ..., a_{kn})$, $pos = (pos_1, pos_2, ..., pos_{kn})$ 
These are the decision variables where $a_i$ represents the number at position i in the sequence and $pos_i$ represents the position of the i-th number in the sequence.

constraints: $\forall i \in \{1, 2, ..., n\}: \exists j \in \{1, 2, ..., kn - (k-1)i\}: \forall c \in \{0, 1, ..., k-1\}: a_{j + ic + c} = i \land pos_{(i-1)k + c + 1} = j + ic + c$, 
$\forall i \in \{1, 2, ..., n\}: \sum_{j=1}^{kn} [a_j = i] = k$, 
$pos_1 < pos_2 < ... < pos_{kn}$, 
$a_1 < a_{kn}$ 
These constraints ensure that each number appears the correct number of times, that each appearance of a number is the correct distance from the previous appearance, that all positions are distinct, and that the sequence is symmetric.

objective: satisfy 
The objective of the problem is to find a sequence that satisfies all the constraints, there is no explicit objective function to minimize or maximize.
