Quasigroup Existence Problem
The Quasigroup existence problem is about finding an $m \times m$ multiplication table of integers $1..m$ where each element occurs exactly once in each row and column, and certain multiplication axioms hold.

The main decision variables are `quasiGroup` and `quasiGroupColumns`, which are two-dimensional arrays representing the quasigroup table and its column-reflected version, respectively. Each element in these arrays represents the result of multiplying two elements in the quasigroup.

The key constraints are:
- Each row and column in the quasigroup table must contain each element exactly once.
- The column-reflected quasigroup table must have the same properties as the original table.
- A specific multiplication axiom must hold: $(a*b)*b = a*(a*b)$, which is represented by the constraint `quasiGroup[i, quasiGroup[i,j]] = quasiGroupColumns[j,quasiGroup[i,j]]`.
- Idempotency is also implied, meaning that $a*a = a$ for all elements $a$.
- Symmetry-breaking constraints are added to reduce the search space.

The problem's objective is to find a satisfying assignment for the quasigroup table, which means finding values for the `quasiGroup` array that satisfy all the constraints. The problem is a satisfaction problem, meaning that the goal is to find a solution that satisfies all the constraints, rather than optimizing a specific objective function.
