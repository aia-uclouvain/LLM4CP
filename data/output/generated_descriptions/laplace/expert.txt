name: Laplace Problem
description: The Laplace problem is a classic problem in mathematics and physics that involves finding a function that satisfies Laplace's equation, a partial differential equation that describes the behavior of gravitational, electric, and fluid potentials. In the context of this MiniZinc model, the problem is discretized using Leibman's five-point finite-difference approximation, which transforms the partial differential equation into a system of linear equations. The goal is to find the values of the function at a set of discrete points, given some boundary conditions.
variables: $x_{i,j}$, where $i \in [0,r]$ and $j \in [0,c]$. These variables represent the values of the function at each point in the discrete grid, where $r$ and $c$ are the number of rows and columns in the grid, respectively.
constraints: $\forall i \in [1,r-1], j \in [1,c-1]: 4x_{i,j} = x_{i-1,j} + x_{i,j-1} + x_{i+1,j} + x_{i,j+1}$. These constraints represent the discretized Laplace equation, which states that the value of the function at each point is equal to the average of its four neighboring points. The boundary conditions are also implicitly defined by the given values of $x_{i,j}$ for $i = 0$, $i = r$, $j = 0$, and $j = c$.
objective: minimize $x_{1,1}$. The objective of the problem is to minimize the value of the function at the point $(1,1)$, although this is not a fundamental aspect of the Laplace problem and is likely included to guide the solver towards a specific solution.
