name: Max Flow Problem
description: The Max Flow Problem is a classical problem in operations research and computer science that involves finding the maximum amount of flow that can be sent through a network of nodes and arcs, subject to certain capacity constraints. The problem has numerous applications in fields such as logistics, transportation, and network optimization.
variables: $flow_{i,j}, z$ - The decision variables in this problem are the flow variables $flow_{i,j}$, which represent the amount of flow sent from node $i$ to node $j$, and the variable $z$, which represents the maximum flow that can be sent through the network. The flow variables are used to determine the optimal flow pattern in the network.
constraints: $flow_{i,j} \geq 0, flow_{i,j} \leq cap_i, \sum_{k|arc_k = (i,j)} flow_{i,j} = \sum_{k|arc_k = (j,i)} flow_{j,i}$ - The constraints in this problem ensure that the flow through each arc does not exceed its capacity ($flow_{i,j} \leq cap_i$), that the flow is non-negative ($flow_{i,j} \geq 0$), and that the flow into each node is equal to the flow out of the node ($\sum_{k|arc_k = (i,j)} flow_{i,j} = \sum_{k|arc_k = (j,i)} flow_{j,i}$).
objective: Maximize $z = flow_{5,1}$ - The objective of the problem is to maximize the flow sent from node 5 to node 1, which represents the maximum flow that can be sent through the network.
