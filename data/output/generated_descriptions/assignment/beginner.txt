Assignment Problem
The problem is about assigning tasks to workers in the best way possible, so that the total cost is as low as possible. 
The main variables are: 
- rows, which means the number of workers
- cols, which means the number of tasks
- x, which is like a table that shows which worker is assigned to which task (it's either 0, meaning not assigned, or 1, meaning assigned)
- cost, which is another table that shows how much it costs to assign a worker to a task
The basic restrictions are: 
- each worker can only be assigned to one task
- each task can only be assigned to one worker, but some tasks might not be assigned to anyone
The goal is to minimize the total cost, which means finding the best way to assign workers to tasks so that the cost is as low as possible.
