Assignment Problem
The Assignment Problem is about finding the optimal way to assign a set of agents to a set of tasks, where each agent can only be assigned to one task, and each task can only be assigned to one agent, with the goal of minimizing the total cost of the assignments.

The main decision variables in this problem are represented by the 2D array `x`, where `x[i, j]` is a binary variable that equals 1 if agent `i` is assigned to task `j`, and 0 otherwise. The `cost` array represents the cost of assigning each agent to each task.

The key constraints in this problem are: 
- each agent must be assigned to exactly one task, 
- each task can be assigned to at most one agent. 
These constraints ensure that the assignments are valid and efficient.

The objective of the Assignment Problem is to minimize the total cost of the assignments, which is calculated by summing the costs of each assignment. In other words, the goal is to find the set of assignments that results in the lowest total cost, while satisfying the constraints that each agent is assigned to exactly one task and each task is assigned to at most one agent.
