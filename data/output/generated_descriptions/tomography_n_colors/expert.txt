name: Discrete Tomography Problem
description: The Discrete Tomography Problem is a classic problem in computer science and operations research that involves reconstructing a binary or multi-colored image from its row and column sums. The problem is defined by a set of row and column sums, and the goal is to find a binary or multi-colored matrix that satisfies these sums. The problem has applications in various fields, including image reconstruction, data analysis, and computer vision.
variables: $x_{i,j}$, where $i \in \{1, \ldots, r\}$ and $j \in \{1, \ldots, c\}$. These variables represent the color of the cell at position $(i, j)$ in the matrix, where $x_{i,j} \in \{1, \ldots, num\_colors\}$.
constraints: $\sum_{j=1}^{c} (x_{i,j} = k) = row\_sums_{k,i}$ for all $i \in \{1, \ldots, r\}$ and $k \in \{1, \ldots, num\_colors\}$, and $\sum_{i=1}^{r} (x_{i,j} = k) = col\_sums_{k,j}$ for all $j \in \{1, \ldots, c\}$ and $k \in \{1, \ldots, num\_colors\}$. These constraints ensure that the row and column sums of the reconstructed matrix match the given sums.
objective: The objective of the problem is to find a feasible solution that satisfies all the constraints, i.e., to reconstruct a matrix that matches the given row and column sums.
