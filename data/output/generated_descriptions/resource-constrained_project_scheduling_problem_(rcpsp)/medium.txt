Resource-Constrained Project Scheduling Problem (RCPSP)
The problem is about scheduling a set of jobs with given durations and resource requirements on a limited number of resources, while respecting precedence constraints between jobs.

The main decision variables are the starting times `s[i]` of each job `i`, which represent the time at which the job starts being executed.

The key constraints are:
- Precedence constraints: a job cannot start before its predecessors have finished.
- Resource constraints: the total amount of resources used by all jobs at any given time cannot exceed the available capacity of each resource.

The objective of the problem is to minimize the completion time of the last job, i.e., the makespan, which is the maximum completion time among all jobs. In other words, the goal is to find a schedule that allows all jobs to be completed as early as possible, while respecting the given constraints.
