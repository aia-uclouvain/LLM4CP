name: Alien Tiles Puzzle
description: The Alien Tiles Puzzle is a classic problem in constraint programming where the goal is to find a configuration of tiles that satisfies certain constraints, given a start and target position. The puzzle consists of a grid where each cell can have one of four values (0, 1, 2, or 3) and the constraints are based on the sum of values in each row and column.
variables: $x_{i,j}, d_{i,j}$ where $i,j \in N$, $x_{i,j}$ represents the value of the tile in the $i^{th}$ row and $j^{th}$ column, and $d_{i,j}$ represents a derived value based on the tile values.
constraints: $\forall i,j \in N: \sum_{l \in N} x_{i,l} + \sum_{k \in N, k \neq i} x_{k,j} = 4d_{i,j} + t_{i,j}$, $\sum_{i \in 1..fhr, j \in N} x_{i,j} \leq \sum_{i \in shr..n, j \in N} x_{i,j}$, $\sum_{i \in N, j \in 1..fhr} x_{i,j} \leq \sum_{i \in N, j \in shr..n} x_{i,j}$, where $t_{i,j}$ is the target value for the $i^{th}$ row and $j^{th}$ column, and $fhr$ and $shr$ are values used to eliminate symmetries.
objective: satisfy, the problem is a satisfaction problem, where the goal is to find a configuration of tiles that satisfies all the constraints, there is no objective function to minimize or maximize.
