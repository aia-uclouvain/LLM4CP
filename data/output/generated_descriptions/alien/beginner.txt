Alien Tiles Puzzle
The Alien Tiles Puzzle is about placing tiles in a grid to match a target pattern. 
The main variables are: 
- x, which represents the tiles in the grid, with each tile having a value from 0 to 3, 
- d, which represents the number of times a tile is used, with a value from 0 to 10, 
- s, which is the starting position of the tiles, 
- t, which is the target position of the tiles.
The basic restrictions are: 
- each row and column can have a certain number of tiles, 
- the tiles must be placed in a way that matches the target pattern, 
- there are symmetry constraints to reduce the number of possible solutions.
The goal of the problem is to find a solution that satisfies all the constraints, there is no specific objective to minimize or maximize.
