name: Minimum Feedback Arc Set Problem (MFASP)
description: The Minimum Feedback Arc Set Problem is a classic problem in graph theory and computer science. Given a directed graph G = (V, E), where V is a set of vertices and E is a set of arcs, the goal is to find a minimal subset of arcs that, when removed from the graph, make it acyclic. In other words, the problem seeks to identify the smallest set of arcs that, when deleted, would result in a directed acyclic graph (DAG).
variables: $x = (x_1, x_2, ..., x_{num\_edges})$, $k = (k_1, k_2, ..., k_n)$, $obj$. $x_i$ is a binary decision variable that indicates whether the $i^{th}$ arc is a feedback arc (i.e., it is part of the minimal subset of arcs to be removed), $k_i$ is an integer decision variable that represents the number assigned to vertex $i$ in the topological sorting of the graph, and $obj$ is an integer variable that represents the objective function value.
constraints: $\forall i \in \{1, 2, ..., num\_edges\}: k_{E[i,2]} - k_{E[i,1]} \geq 1 - num\_edges \cdot x_i$. This constraint ensures that if an arc $(i, j)$ is not removed (i.e., $x_i = 0$), then the number assigned to vertex $j$ must be greater than the number assigned to vertex $i$.
objective: minimize $obj = \sum_{i=1}^{num\_edges} x_i$, which represents the cardinality of the minimal subset of feedback arcs to be removed.
