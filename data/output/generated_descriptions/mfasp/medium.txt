**Minimum Feedback Arc Set Problem**

The Minimum Feedback Arc Set Problem is about finding the smallest set of arcs (directed edges) in a directed graph that, when removed, make the graph acyclic (i.e., no cycles remain). This problem is relevant in various fields, such as network analysis, scheduling, and computer science.

**Decision Variables**

The main decision variables are:
- `x[i]`: a binary variable indicating whether the `i-th` arc is a feedback arc (i.e., part of the set of arcs to be removed).
- `k[i]`: an integer variable representing the number assigned to the `i-th` vertex, used to ensure the graph becomes acyclic when the selected arcs are removed.

**Constraints**

The key constraints are:
- For each arc, if it is not removed (i.e., `x[i]` is 0), the number assigned to the destination vertex must be greater than the number assigned to the source vertex. If the arc is removed (i.e., `x[i]` is 1), this constraint is relaxed.
- The objective is to minimize the total number of arcs removed, which is represented by the sum of all `x[i]` values.

**Objective**

The objective of the Minimum Feedback Arc Set Problem is to minimize the number of arcs that need to be removed from the graph to make it acyclic. This is achieved by minimizing the sum of all `x[i]` values, which represents the cardinality of the subset of feedback arcs.
