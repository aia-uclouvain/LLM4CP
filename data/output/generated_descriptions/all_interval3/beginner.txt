All Interval Problem
The problem is about creating a sequence of musical notes where each note is used exactly once and the differences between consecutive notes are all unique.
The main variables are: 
- series: an array representing the sequence of musical notes, where each note is a number from 0 to n-1.
- differences: an array representing the differences between consecutive notes in the sequence.
The basic restrictions are: 
- Each note can only be used once in the sequence.
- The difference between each pair of consecutive notes must be unique.
The goal of the problem is to find a sequence of notes that satisfies these restrictions.
