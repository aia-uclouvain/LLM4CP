description: The All Interval Problem is a well-known problem in music theory and combinatorial optimization. It involves finding a sequence of musical notes, represented by integers from 0 to n-1, such that each note appears exactly once and the absolute differences between consecutive notes are all distinct. This problem has applications in music composition, as it can be used to generate musical sequences with unique interval patterns.
variables: $series = [s_0, s_1, ..., s_{n-1}], differences = [d_1, d_2, ..., d_{n-1}]$, where $series$ is an array of decision variables representing the sequence of notes, and $differences$ is an array of decision variables representing the absolute differences between consecutive notes.
constraints: $s_i \neq s_j \forall i \neq j, d_i = |s_{i+1} - s_i| \forall i \in 1..n-2, d_i \neq d_j \forall i \neq j$, which ensure that each note appears exactly once, the differences between consecutive notes are calculated correctly, and all differences are distinct.
objective: The objective of the problem is to find a feasible solution that satisfies all the constraints, as there is no explicit objective function to minimize or maximize. The goal is to generate a sequence of notes that meets the specified conditions.
