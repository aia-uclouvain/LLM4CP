De Bruijn sequence with no direct repetition
The problem is about generating a sequence of numbers, known as a De Bruijn sequence, where each number is represented by a sequence of digits (or bits) from a given alphabet, and no digit is directly followed by the same digit.
The main decision variables are `x`, `binary`, and `bin_code`. `x` represents the sequence of numbers in decimal form, `binary` represents the binary (or base-representation) of each number, and `bin_code` represents the sequence in base-representation.
The key constraints are: all numbers in the sequence must be different, each number must be represented uniquely in binary form, the first digit of each number is the same as the last digit of the previous number (the De Bruijn condition), and no digit can be directly followed by the same digit. Additionally, the sequence must wrap around, meaning the last digit of the last number is the same as the first digit of the first number.
The objective of the problem is to find a sequence that satisfies all the constraints, which is essentially a feasibility problem, where the goal is to find a solution that meets all the conditions, rather than optimizing a specific objective function. In this case, the problem is solved using a satisfaction-based approach, where the model is solved to find a feasible solution that meets all the constraints.
