name: Gray Code Problem
description: The Gray Code problem is a classical problem in computer science and mathematics that involves generating a sequence of binary numbers such that only one bit changes between each pair of consecutive numbers. This sequence is known as a Gray code. The problem has applications in various fields, including coding theory, data transmission, and combinatorial design.

variables: $x = (x_1, x_2, ..., x_m)$, $binary = (binary_{i,j})_{m \times n}$, $num\_bits = (num\_bits_1, num\_bits_2, ..., num\_bits_m)$, where $x$ is an array of variables representing the numbers in the Gray code sequence, $binary$ is a 2D array of variables representing the binary representation of each number, and $num\_bits$ is an array of variables representing the number of bits set to 1 in each binary representation.

constraints: 
$x_1 = 0$, 
$\sum_{j=1}^{n} binary_{m,j} = 1$, 
$all\_different(x)$, 
$toNum(t, x_i, 2.0)$, 
$binary_{i,j} = t_j$, 
$\sum_{j=1}^{n} (binary_{i,j} \neq binary_{i-1,j}) = 1$, 
$num\_bits_i = \sum_{j=1}^{n} binary_{i,j}$, 
where these constraints ensure that the first number is 0, the last number has exactly one bit set to 1, all numbers are distinct, each number is represented correctly in binary, only one bit changes between consecutive numbers, and the number of bits set to 1 is calculated correctly for each number.

objective: The objective of the problem is to find a satisfying assignment of values to the decision variables, i.e., to generate a valid Gray code sequence. The problem does not have a specific objective function to minimize or maximize, but rather aims to find a feasible solution that satisfies all the constraints.
