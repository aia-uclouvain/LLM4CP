name: Schuh's Queen Placement Problem

description: The problem is to place the maximum number of queens on an 8x8 chessboard such that no queen attacks any other queen. A queen attacks another queen if they are in the same row, column, or diagonal.

variables: $x_{i,j}, a_{i,j}, numq$ 
These variables represent the following: $x_{i,j}$ is a binary variable that equals 1 if a queen is placed at position $(i,j)$ on the chessboard and 0 otherwise, $a_{i,j}$ is a binary variable that equals 1 if the position $(i,j)$ is attacked by a queen and 0 otherwise, and $numq$ is an integer variable that represents the total number of queens placed on the chessboard.

constraints: $\sum_{i,j} x_{i,j} \leq size$, $\sum_{m \neq i, m-i+j \geq 1, m-i+j \leq size} x_{m,m-i+j} + \sum_{m \neq i, i+j-m \geq 1, i+j-m \leq size} x_{m,i+j-m} + \sum_{m \neq i} x_{m,j} + \sum_{n \neq j} x_{i,n} \leq 99a_{i,j}$, $a_{i,j} + x_{i,j} = 1$
These constraints ensure that the total number of queens placed does not exceed the size of the chessboard, that a position is attacked if a queen is placed in the same row, column, or diagonal, and that each position is either occupied by a queen or attacked by a queen.

objective: maximize $numq$ 
The objective is to maximize the total number of queens placed on the chessboard.
