name: Sum First and Last Problem
description: The Sum First and Last Problem is a constraint satisfaction problem where we have a square grid with some numbers on top, below, left, and right of each column and row. The goal is to fill in the grid such that each number from 0 to grid size - 1 appears exactly once in each row and column. The numbers on the top and right are the sum of the first and last non-zero digits in the column and row respectively, while the numbers on the bottom and left are the sum of the second and second last non-zero digits in the column and row respectively. Zeros are used to represent blank spaces and do not contribute to the sums.
variables: $x_{i,j}$ for $i,j = 1 \ldots n$, TopConstraint, RightConstraint, BottomConstraint, LeftConstraint. These are the decision variables representing the values in the grid, where $x_{i,j}$ is the value in the $i^{th}$ row and $j^{th}$ column, and TopConstraint, RightConstraint, BottomConstraint, LeftConstraint are arrays representing the given constraints for each row and column.
constraints: $\forall i \in 1 \ldots n: \text{all\_different}(x_{i,1 \ldots n}) \land \text{all\_different}(x_{1 \ldots n,i}) \land (TopConstraint_i > 0 \rightarrow TopConstraint_i = x_{1,i} + x_{n,i}) \land (RightConstraint_i > 0 \rightarrow RightConstraint_i = x_{i,1} + x_{i,n}) \land (LeftConstraint_i > 0 \rightarrow LeftConstraint_i = x_{i,2} + x_{i,n-1}) \land (BottomConstraint_i > 0 \rightarrow BottomConstraint_i = x_{2,i} + x_{n-1,i})$. These constraints ensure that each row and column contains each number exactly once, and that the sums of the first and last non-zero digits in each row and column match the given constraints.
objective: The objective of this problem is to find a satisfying assignment, i.e., to fill in the grid such that all constraints are satisfied. There is no explicit objective function to minimize or maximize.
