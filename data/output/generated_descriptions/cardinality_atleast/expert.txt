name: Cardinality Atleast Problem
description: The Cardinality Atleast Problem is a constraint satisfaction problem that involves finding an assignment of values to a set of variables such that at least a certain number of variables take on values from a specified set. The problem is defined by a set of variables, a set of values, and a minimum cardinality constraint that specifies the minimum number of times each value should appear in the variables.
variables: $nvar \in \{0, \ldots, 8\}, variables = (variables_1, \ldots, variables_n) \in \{1, \ldots, 8\}^n, values = (values_1, \ldots, values_m) \in \{1, \ldots, 8\}^m$ - $nvar$ is the minimum number of times each value should appear, $variables$ is the set of variables to be assigned, and $values$ is the set of values that should appear in the variables.
constraints: $nvar \geq \sum_{j=1}^{n} (values_i = variables_j) \forall i \in \{1, \ldots, m\}, variables = [3, 3, 8], values = [3, 8], nvar = 1$ - The constraints specify that the minimum number of times each value appears is at least $nvar$, the variables should take on the values $[3, 3, 8]$, the values of interest are $[3, 8]$, and the minimum cardinality is $1$.
objective: satisfy - The objective of the problem is to find an assignment of values to the variables that satisfies all the constraints.
