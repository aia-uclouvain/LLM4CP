**Minimum Feedback Vertex Set Problem**
The Minimum Feedback Vertex Set Problem is about finding the smallest set of vertices in a directed graph that, when removed, makes the graph acyclic. 

**Decision Variables**
The main decision variables are `x` and `k`. `x[i]` represents whether an arc is a feedback arc (1) or not (0), and `k[i]` represents the order or number assigned to vertex `i` in the graph.

**Constraints**
The key constraints are that for every arc in the graph, if the arc is not a feedback arc, then the order of the destination vertex must be greater than the order of the source vertex. This ensures that the graph is acyclic. 

**Objective**
The objective of the problem is to minimize the number of feedback vertices, which is represented by the variable `obj`. This is calculated as the sum of all `x[i]` values, where `x[i]` is 1 if the corresponding arc is a feedback arc and 0 otherwise. The goal is to find the smallest set of vertices that, when removed, makes the graph acyclic.
