name: Stochastic General Assignment Problem

description: The Stochastic General Assignment Problem is a scheduling problem where a set of tasks need to be assigned to a set of machines. Each task has a set of optional tasks with different processing times, and the goal is to minimize the expected makespan. The problem is stochastic because the processing times of the tasks are uncertain and are modeled using scenarios. The problem involves two stages: first, tasks are assigned to machines, and then, based on the observed processing times, the tasks are scheduled on their respective machines.

variables: $start_{s,t}, dur_{s,t}, b_o$ 
These are the decision variables of the problem. $start_{s,t}$ represents the start time of task $t$ in scenario $s$, $dur_{s,t}$ represents the duration of task $t$ in scenario $s$, and $b_o$ is a binary variable that indicates whether optional task $o$ is executed or not.

constraints: $\forall s \in SCENARIOS, \forall j \in Jobs, \forall i \in tasks[j] \text{ where } i < last\_task[j]: start_{s,i} + dur_{s,i} \leq start_{s,i+1}$, 
$\forall o \in OptTs, \forall s \in SCENARIOS: b_o \rightarrow dur_{s,t} = optt\_dur_{s,o}$, 
$\forall t \in Tasks \text{ where } card(optts[t]) > 1: \sum_{o \in optts[t]} b_o \leq 1 \land \exists o \in optts[t]: b_o$, 
$\forall m \in Mach, \forall s \in SCENARIOS: cumulative([start_{s,optt\_task[o]} | o \in MTasks], [optt\_dur_{s,o} | o \in MTasks], [bool2int(b_o) | o \in MTasks], 1)$, 
$\forall s \in SCENARIOS, \forall j \in Jobs: start_{s,last\_task[j]} + dur_{s,last\_task[j]} \leq de\_objective_s$ 
These constraints model the precedence relations between tasks, the duration of tasks, the execution of optional tasks, the resource constraints, and the objective constraint.

objective: minimize $\sum_{s \in SCENARIOS} weights_s \cdot de\_objective_s$ 
The objective of the problem is to minimize the expected makespan, which is calculated as the weighted sum of the makespan in each scenario.
