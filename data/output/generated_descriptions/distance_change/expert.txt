name: Distance Change Problem
description: The Distance Change Problem is a constraint satisfaction problem that involves finding the number of times a certain condition is true between two sequences of variables. The condition is based on a comparison operator and checks if the relationship between consecutive variables in one sequence is different from the relationship between consecutive variables in the other sequence.
variables: $dist, v1_1, v1_2, ..., v1_n, v2_1, v2_2, ..., v2_n$ where $dist$ is the number of times the condition is true, and $v1_i$ and $v2_i$ are variables representing the values in the two sequences, which are integers between 1 and 3 for $v1_i$ and between 1 and 4 for $v2_i$.
constraints: $cp1d(v1, [3, 3, 1, 2, 2]), cp1d(v2, [4, 4, 3, 3, 3]), distance\_change(dist, v1, v2, ctr)$ where $cp1d$ is a constraint that checks if two arrays are equal, and $distance\_change$ is the constraint that checks the number of times the condition is true based on the comparison operator $ctr$.
objective: satisfy, the problem is a constraint satisfaction problem, so the objective is to find a solution that satisfies all the constraints, and the value of $dist$ is output as part of the solution.
