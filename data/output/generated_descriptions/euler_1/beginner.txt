Problem name: Project Euler problem 1
This problem is about finding the sum of all numbers below 1000 that are multiples of 3 or 5. 
The main variables are: n, which is the largest number below 1000, x, which is an array that shows whether each number is a multiple of 3 or 5, and s, which is the sum of these multiples.
The basic restrictions are: a number is considered a multiple of 3 or 5 if it can be divided by 3 or 5 without leaving a remainder.
The goal of the problem is to find the sum of all these multiples, which is stored in the variable s.
