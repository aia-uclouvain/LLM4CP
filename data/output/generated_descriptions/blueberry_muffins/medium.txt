Blueberry Muffins problem
The Blueberry Muffins problem is a classic constraint programming problem that involves determining the full name, profession, and number of muffins snatched by each of four housemates. 
The main decision variables in this problem are the last names, professions, and number of muffins snatched by each housemate, which are represented by the variables `lastname`, `job`, and `muffins` respectively. 
The key constraints in this problem can be described as follows: each housemate has a unique last name and profession, each housemate snatched a different number of muffins, and there are specific relationships between the number of muffins snatched by each housemate based on their profession and last name. For example, Bill and Ellis snatched a total of six muffins, the secretary snatched more muffins than the dogcatcher, and the flautist snatched twice as many muffins as Ms. Oakley. 
The objective of this problem is to find a solution that satisfies all the given constraints, which will allow us to determine the full name, profession, and number of muffins snatched by each housemate. The problem does not involve minimizing or maximizing a specific objective function, but rather finding a feasible solution that meets all the constraints.
