name: Global Constraint Count
description: The Global Constraint Count problem is a constraint programming problem that involves enforcing a condition on the number of variables in a collection that are assigned to a specific value. The condition is specified by a relational operator and a threshold value. The problem is to find an assignment of values to the variables that satisfies the given condition.

variables: $x = (x_1, x_2, ..., x_n)$, $nvar$, $val$
$x$ is an array of decision variables representing the collection of variables, $nvar$ is a decision variable representing the threshold value, and $val$ is a decision variable representing the specific value.

constraints: $x = (4, 5, 5, 4, 5)$, $val = 5$, $count\_ctr(val, x, 1, nvar)$
The constraints specify that the collection of variables $x$ is fixed to $(4, 5, 5, 4, 5)$, the specific value $val$ is fixed to $5$, and the $count\_ctr$ constraint enforces the condition that the number of variables in $x$ assigned to $val$ is greater than or equal to $nvar$.

objective: satisfy
The objective of the problem is to find a satisfying assignment of values to the variables, i.e., to find values for $nvar$ such that the $count\_ctr$ constraint is satisfied.
