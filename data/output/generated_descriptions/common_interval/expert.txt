name: Common Interval Problem
description: The Common Interval Problem is a constraint satisfaction problem that involves finding the number of common intervals between two sets of variables. Each variable in the sets is associated with an interval, and the goal is to determine the number of variables in each set that fall within an interval associated with a variable in the other set.
variables: $a, b, x_1, x_2, ..., x_4, y_1, y_2, ..., y_6$ - These are the decision variables, where $a$ and $b$ represent the number of common intervals between the two sets, and $x_i$ and $y_j$ are the variables in the two sets, which are assigned integer values between 0 and 9.
constraints: $a = \sum_{i=1}^{int\_size\_x} (x\_counts[i] * bool2int(y\_counts[i] > 0))$, $b = \sum_{j=1}^{int\_size\_y} (y\_counts[j] * bool2int(x\_counts[j] > 0))$, $make\_interval(x\_intervals, interval\_size, int\_size\_x)$, $make\_interval(y\_intervals, interval\_size, int\_size\_y)$, $count\_interval(x, x\_intervals, x\_counts, int\_size\_x)$, $count\_interval(y, y\_intervals, y\_counts, int\_size\_y)$ - These constraints define the relationships between the variables and ensure that the intervals are properly created and counted.
objective: The objective of the problem is to satisfy the constraints, which means finding the values of $a$ and $b$ that represent the number of common intervals between the two sets of variables.
