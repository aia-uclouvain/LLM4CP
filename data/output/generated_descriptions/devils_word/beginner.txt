Devil's Word
The problem is about finding a way to add or subtract a list of numbers to get a specific total. 
The main variables are: 
- total: the target number we want to reach
- arr: the list of numbers we can use
- plus and minus: these show whether each number in the list should be added or subtracted to get the total
The basic restrictions are: 
- each number in the list can only be used once
- each number can only be either added or subtracted, not both
The goal is to find a combination of additions and subtractions that adds up to the target total, in this case 666.
