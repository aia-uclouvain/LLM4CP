name: Minesweeper
description: Minesweeper is a classic puzzle game where the player is presented with a grid of cells, some of which contain hidden mines. The player's goal is to clear all non-mine cells without detonating a mine. The game provides clues in the form of numbers in some cells, which indicate how many mines are adjacent to that cell. The puzzle is solved when all non-mine cells have been cleared, and the location of all mines has been determined.
variables: $x_{i,j} \in \{0,1\}$ for $i \in \{1, \ldots, r\}$ and $j \in \{1, \ldots, c\}$, where $x_{i,j} = 1$ if there is a mine in cell $(i,j)$ and $x_{i,j} = 0$ otherwise. These variables represent the presence or absence of a mine in each cell of the grid.
constraints: $\sum_{(i',j') \in N(i,j)} x_{i',j'} = g_{i,j}$ for all $(i,j)$ such that $g_{i,j} \neq X$, where $N(i,j)$ is the set of neighboring cells of $(i,j)$ and $g_{i,j}$ is the given clue in cell $(i,j)$. These constraints ensure that the number of mines adjacent to each cell with a given clue matches the clue.
objective: The objective of the problem is to find a feasible solution, i.e., an assignment of mines to cells that satisfies all the constraints, as there is no explicit objective function to minimize or maximize.
