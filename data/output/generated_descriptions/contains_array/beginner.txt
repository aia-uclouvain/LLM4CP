Name of the problem: Contains Problem

This problem is about finding a sequence of numbers that represents a specific large number, with a given digit appearing in the sequence. 

The main variables are: 
- x, which is an array of numbers from 0 to 9, representing the digits of the large number.
- el, which is a single digit that must appear in the sequence x.
- num, which is the large number that the sequence x represents.

The basic restrictions or rules of the problem are: 
- The sequence x must be 9 digits long.
- The digit el must appear in the sequence x.
- The sequence x represents a number in base 10, with each digit in the correct position (e.g., the first digit is the hundreds millions place, the second digit is the tens millions place, and so on).
- The number represented by the sequence x must be exactly 123402345.

The goal of the problem is to find a sequence x that satisfies all these rules, with the digit el appearing in the sequence, and the sequence representing the number 123402345.
