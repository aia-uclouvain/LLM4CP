The problem is known as the Cycle Problem or the Permutation Cycle Problem. 
It is about finding a permutation of numbers from 1 to n, such that the permutation can be divided into a specific number of cycles. 
A cycle is a sequence of numbers where each number is mapped to the next number in the sequence, and the last number is mapped back to the first number.

The main decision variables are the array x, which represents the permutation of numbers from 1 to n, and the variable num_cycles, which represents the number of cycles in the permutation.

- all_different(x), which ensures that each number from 1 to n appears exactly once in the permutation, 
- cycle(num_cycles, x), which ensures that the permutation can be divided into num_cycles cycles.
