name: Sudoku
description: The Sudoku problem is a classic puzzle where the goal is to fill a 9x9 grid with numbers from 1 to 9, such that each row, column, and 3x3 sub-grid contains each number exactly once.
variables: $x_{i,j}$, where $i$ and $j$ are indices from 1 to 9, representing the value of the cell at row $i$ and column $j$ in the Sudoku grid. These variables are decision variables that need to be assigned a value from 1 to 9.
constraints: 
$\forall i \in 1..9: global\_cardinality\_old([x_{i,j} | j \in 1..9], occ)$, 
$\forall j \in 1..9: global\_cardinality\_old([x_{i,j} | i \in 1..9], occ)$, 
$\forall i \in 0..2, j \in 0..2: global\_cardinality\_old([x_{r,c} | r \in i*3+1..i*3+3, c \in j*3+1..j*3+3], occ)$, 
$cp2d(x, initial\_grid)$, 
where $occ$ is an array of occurrences of each number, and $initial\_grid$ is the initial Sudoku grid with some given values. These constraints ensure that each row, column, and 3x3 sub-grid contains each number exactly once, and that the assigned values match the initial grid.
objective: The objective of the Sudoku problem is to find a satisfying assignment of values to the decision variables $x_{i,j}$, such that all constraints are satisfied. There is no explicit objective function to minimize or maximize, as the goal is to find a feasible solution that satisfies all constraints.
