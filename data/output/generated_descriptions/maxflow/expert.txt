name: Maximum Flow Problem
description: The Maximum Flow Problem is a classical problem in operations research and computer science that involves finding the maximum amount of flow that can be sent from a source node to a sink node in a flow network, subject to capacity constraints on the arcs and conservation of flow constraints at each node.
variables: $x_i, flow$, where $x_i$ represents the elementary flow through arc $i$ and $flow$ represents the total flow from the source node to the sink node. These variables are decision variables that need to be determined in order to maximize the total flow.
constraints: $0 \leq x_i \leq a_i, 0 \leq flow \leq 100, \sum_{k|E_k^2=i} x_k + flow \cdot \mathbb{1}_{i=s} = \sum_{k|E_k^1=i} x_k + flow \cdot \mathbb{1}_{i=t}$, where the first constraint ensures that the flow through each arc does not exceed its capacity, the second constraint ensures that the total flow is non-negative and bounded, and the third constraint ensures that the flow is conserved at each node.
objective: Maximize $flow$, which represents the total flow from the source node to the sink node. The goal is to find the maximum possible value of $flow$ that satisfies all the constraints.
