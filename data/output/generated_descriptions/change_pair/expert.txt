name: Change Pairs Problem
description: The Change Pairs Problem is a constraint satisfaction problem that involves finding the number of times a certain condition is met between consecutive pairs of variables. The condition is defined by two comparison operators applied to two arrays of variables. The goal is to find an assignment of values to the variables that satisfies the given constraints.
variables: $x_1, x_2, ..., x_n, y_1, y_2, ..., y_n, nchange$, where $x_i$ and $y_i$ are decision variables representing the values of the arrays at index $i$, and $nchange$ is a decision variable representing the total number of times the condition is met.
constraints: $nchange = \sum_{i=2}^{n} (x_{i-1} \neq x_i \lor y_{i-1} \neq y_i)$, $x_i \in [1,3]$, $y_i \in [1,8]$, where the first constraint defines the change pairs condition, and the last two constraints define the domains of the variables.
objective: The objective of the problem is to find a satisfying assignment, i.e., to solve the problem as a satisfaction problem, there is no explicit objective function to minimize or maximize.
