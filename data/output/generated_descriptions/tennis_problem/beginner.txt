Tennis problem
The problem is about a tennis match between two players, Frank and George, where Frank wins 6 games and George wins 3 games. 
The main variables are: 
- fs (Frank serves): whether Frank serves in each game
- gs (George serves): whether George serves in each game
- fw (Frank wins): whether Frank wins each game
- gw (George wins): whether George wins each game
The basic restrictions are: 
- In each game, either Frank or George serves, but not both
- The server alternates between Frank and George for each game
- Frank wins 6 games in total
- In 4 games, the server loses
The goal is to find out who served the first game, given these conditions.
