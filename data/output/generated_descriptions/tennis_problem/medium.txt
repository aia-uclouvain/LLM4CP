Tennis Problem
The problem is about determining the sequence of serves and wins in a tennis match between Frank and George, given certain conditions.
The main decision variables are: 
- `fs`: an array representing whether Frank serves in each game (1 if Frank serves, 0 otherwise)
- `gs`: an array representing whether George serves in each game (1 if George serves, 0 otherwise)
- `fw`: an array representing whether Frank wins in each game (1 if Frank wins, 0 otherwise)
- `gw`: an array representing whether George wins in each game (1 if George wins, 0 otherwise)
The key constraints are: 
- In each game, either Frank or George serves, but not both.
- The server alternates between Frank and George.
- Frank wins exactly 6 games.
- In exactly 4 games, the server loses.
The objective is to find a sequence of serves and wins that satisfies all the given conditions, with no specific objective function to minimize or maximize, as the problem is a satisfaction problem.
