name: Quasigroup Completion Problem
description: The Quasigroup Completion Problem is a classic problem in constraint programming that involves completing a partially filled Latin square, which represents a quasigroup operation table. A quasigroup is a mathematical structure that satisfies certain properties, including the existence of a unique solution for every pair of elements. The problem requires finding a completion of the given partial Latin square such that the resulting table represents a quasigroup.
variables: $x_{i,j}$ for $i = 1..n, j = 1..n$, where $x_{i,j}$ represents the value in the $i$-th row and $j$-th column of the quasigroup operation table, which is a decision variable that takes a value from the set $\{1, 2, ..., n\}$.
constraints: $\forall i, j, k \in \{1, 2, ..., n\}: \forall x_{i,j}, x_{j,k}, x_{i,k} \in \{1, 2, ..., n\}: x_{i,j} \neq x_{i,k} \wedge x_{j,k} \neq x_{i,k} \wedge x_{i,j} \cdot x_{j,k} = x_{i,k}$, which ensures that each row and column contains each value exactly once and that the quasigroup properties are satisfied, as well as the given partial assignments.
objective: The objective of the problem is to find a feasible solution, i.e., to complete the partial Latin square such that the resulting table represents a quasigroup, rather than minimizing or maximizing a specific value.
