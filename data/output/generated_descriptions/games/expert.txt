name: Two-Person Zero-Sum Game
description: The Two-Person Zero-Sum Game is a classical problem in game theory where two players, often referred to as the row player and the column player, engage in a competitive game. The game is zero-sum, meaning that one player's gain is equal to the other player's loss. The row player has a set of strategies, and the column player has a set of strategies. The outcome of the game is determined by a payoff matrix, where each entry represents the payoff to the row player when the row player chooses a particular strategy and the column player chooses a particular strategy. The goal of each player is to maximize their expected payoff.
variables: $X = (X_1, X_2, ..., X_n), w$ where $X$ is a vector of decision variables representing the probability of each strategy for the column player, and $w$ is a decision variable representing the expected payoff to the row player.
constraints: $\sum_{j=1}^{n} A_{ij}X_j - w \geq 0, \forall i \in \{1, 2, ..., n\}$ and $\sum_{j=1}^{n} X_j = 1$ where the first set of constraints ensures that the expected payoff to the row player is greater than or equal to $w$ for all possible strategies, and the second constraint ensures that the probabilities of the column player's strategies sum to 1.
objective: maximize $w$
