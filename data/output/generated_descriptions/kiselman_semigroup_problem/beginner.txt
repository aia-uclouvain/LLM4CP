Kiselman Semigroup Problem
The problem is about finding a special sequence of numbers from 1 to a given number n, such that between every two occurrences of the same number, there is a larger and a smaller number.
The main variables are: 
- n, the maximum number in the sequence
- m, the maximum length of the sequence
- x, an array of numbers representing the sequence, where each number is between 0 and n
The basic restrictions are: 
- the sequence must contain numbers between 1 and n
- between every two occurrences of the same number, there must be a larger and a smaller number
- zeros are used as dummy values and must be placed at the end of the sequence
The goal is to find a sequence that satisfies these restrictions, and the problem is usually interested in counting the number of such sequences for a given n.
