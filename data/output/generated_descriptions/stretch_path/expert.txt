name: Stretch Path Problem
description: The Stretch Path Problem is a constraint satisfaction problem that involves finding a sequence of values that satisfy certain stretch constraints. A stretch is a sequence of consecutive variables that take the same value, and the span of the stretch is the number of variables in the sequence. The problem requires that the span of each stretch is within a specified interval, and that the sequence satisfies certain conditions such as having different values at the start and end of each stretch.
variables: $x_1, x_2, ..., x_n, val_1, val_2, ..., val_m, lmin_1, lmin_2, ..., lmin_m, lmax_1, lmax_2, ..., lmax_m$. These variables represent the sequence of values ($x_i$), the set of possible values for each stretch ($val_j$), and the minimum and maximum span of each stretch ($lmin_j$ and $lmax_j$).
constraints: $lmin_j \leq lmax_j, first\_pos_i = 1 \leftrightarrow x_i \neq x_{i-1}, val_{s\_ix} = x_i, j-i+1 \geq lmin_{s\_ix} \wedge j-i+1 \leq lmax_{s\_ix} \wedge x_k = x_i \wedge x_{i-1} \neq x_i \wedge x_{j+1} \neq x_j$. These constraints ensure that the minimum span is less than or equal to the maximum span, that the first position of each stretch is correctly identified, that the value of each stretch is correctly assigned, and that the span of each stretch is within the specified interval.
objective: The objective of the problem is to satisfy all the constraints, i.e., to find a sequence of values that satisfies all the stretch constraints. The problem does not have a specific objective function to minimize or maximize.
