The Stretch Constraint Problem
The Stretch Constraint problem is about finding a sequence of values that satisfies certain constraints on consecutive stretches of the same value. 

The main decision variables are: 
- `x`: an array of variables representing the sequence of values, 
- `val`: an array of variables representing the values that each stretch can take, 
- `lmin` and `lmax`: arrays of variables representing the minimum and maximum length of each stretch.

The key constraints are: 
- Each stretch must have a length within the specified minimum and maximum limits for its value, 
- Each stretch must start and end with a value that is different from the preceding and following values, 
- The sequence must satisfy these constraints for all stretches.

The objective of the problem is to find a sequence `x` that satisfies the stretch constraints defined by `val`, `lmin`, and `lmax`. The problem does not have a specific minimization or maximization objective, but rather aims to find a feasible solution that meets all the constraints.
