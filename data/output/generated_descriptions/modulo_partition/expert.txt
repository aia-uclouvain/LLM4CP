name: Function Partitions Problem
description: The Function Partitions Problem is a constraint programming problem where the goal is to partition the integers from 1 to n into sets based on a given function, such as the modulus or division of the integer by another number, and then calculate the sum of the integers in each partition.
variables: $x: \{x_0, x_1, ..., x_n\}$ where $x_i$ is a set of integers, $sums: \{sum_0, sum_1, ..., sum_n\}$ where $sum_i$ is the sum of the integers in $x_i$, $x\_bool: \{x\_bool_{i,j}\}$ where $x\_bool_{i,j}$ is a binary variable indicating whether the integer j is in the set $x_i$. These variables represent the partitions of the integers, the sums of the integers in each partition, and a binary matrix indicating which integers are in which partitions.
constraints: $\forall i \in \{1, ..., n\}: i \in x_{n \mod i}$, $\forall i \in \{0, ..., n\}: set\_sum(x_i, sums_i)$, $\forall i \in \{0, ..., n\}: link\_set\_to\_booleans(x_i, [x\_bool_{i,j} = 1 | j \in \{0, ..., n\}])$, $partition\_set(x, \{1, ..., n\})$. These constraints ensure that each integer is assigned to the correct partition based on the given function, calculate the sum of the integers in each partition, link the sets to the binary matrix, and ensure that each integer is only in one partition.
objective: The objective of the problem is not to minimize or maximize a specific value, but rather to find a feasible solution that satisfies all the constraints, i.e., to find a valid partitioning of the integers based on the given function.
