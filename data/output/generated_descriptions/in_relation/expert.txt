name: In Relation Problem
description: The problem is to find a tuple of values that exists in a given set of tuples, where the tuple of values is defined by a set of decision variables. The problem is modeled using the global constraint "in_relation" which enforces the tuple of variables to take its value out of a set of tuples of values.
variables: $v1 = (v1_1, v1_2, v1_3), arr = (a_{ij})_{i=1..n, j=1..m}$, where $v1$ is a tuple of decision variables representing the values to be found in the set of tuples, and $arr$ is a 2D array of decision variables representing the set of tuples.
constraints: $v1 = (5, 3, 3), arr = \begin{bmatrix} 5 & 2 & 3 \\ 5 & 2 & 6 \\ 5 & 3 & 3 \\ 1 & 3 & 2 \end{bmatrix}, in\_relation(v1, arr)$, where the first two constraints define the values of $v1$ and $arr$, and the third constraint enforces the tuple $v1$ to be in the set of tuples defined by $arr$.
objective: satisfy, the objective is to find a solution that satisfies all the constraints, there is no explicit objective function to minimize or maximize.
