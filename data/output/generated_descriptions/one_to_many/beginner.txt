Name of the problem: One-To-Many Relationship Extraction

This problem is about figuring out the relationships between mothers and their children based on the projects they are involved in. Imagine a school where kids participate in different projects, and for each project, there is a list of mothers and a list of children. The goal is to determine which mother is related to which child.

The main variables are:
- Mothers: a list of mothers' names (e.g., Jane, Claire, Sophia)
- Children: a list of children's names (e.g., Brian, Stephen, Emma)
- x: a decision variable that represents the mother of each child

The basic restrictions or rules of the problem are:
- Each child belongs to exactly one mother
- Each project has a list of mothers and a list of children
- The lists of mothers and children for each project are not necessarily in the same order

The goal of the problem is to assign each mother to her children based on the projects they are involved in, using logical inference. The problem aims to find a solution that satisfies all the constraints, which means finding the correct mother-child relationships.
