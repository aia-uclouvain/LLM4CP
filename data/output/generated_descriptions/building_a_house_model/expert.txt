name: Resource-Constrained Project Scheduling Problem (RCPSp)
description: The Resource-Constrained Project Scheduling Problem is a classic problem in operations research and management science that involves scheduling a set of tasks with given durations and resource requirements, subject to precedence constraints and limited resource availability, with the goal of minimizing the makespan, which is the maximum completion time of all tasks.
variables: $start_t, end_t, limitx, makespan$ where $t = 1, \ldots, num\_tasks$. These variables represent the start time, end time, and height limit of each task, as well as the makespan, which is the maximum completion time of all tasks. The start and end times are decision variables that determine the schedule of each task, while the height limit is a decision variable that determines the maximum height of the tasks at any given time.
constraints: $\forall t: end_t = start_t + duration_t$, $\forall p: start_{p_1} + duration_{p_1} \leq start_{p_2}$, $cumulative(start, duration, height, limitx)$. These constraints ensure that the end time of each task is equal to its start time plus its duration, that the precedence constraints are satisfied, and that the cumulative height of the tasks at any given time does not exceed the height limit.
objective: minimize $makespan$
