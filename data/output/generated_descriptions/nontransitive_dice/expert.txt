name: Nontransitive Dice Problem
description: The Nontransitive Dice Problem is a classic problem in mathematics and game theory where a set of dice is created such that the relation "is more likely to roll a higher number" is not transitive, meaning that die A can beat die B, die B can beat die C, but die C can beat die A, similar to the game Rock, Paper, Scissors.
variables: $m, n, dice_{i,j}, comp_{i,j}, max\_val, max\_win, gap_i, gap\_sum$ 
These variables represent the following: $m$ is the number of dice, $n$ is the number of sides on each die, $dice_{i,j}$ represents the value on the $j^{th}$ side of the $i^{th}$ die, $comp_{i,j}$ represents the number of wins for die $i$ against die $i+1$ (with $i+1$ wrapping around to $1$ when $i=m$) in the $j^{th}$ comparison, $max\_val$ is the maximum value on any die, $max\_win$ is the maximum number of wins for any die, $gap_i$ is the difference in wins between die $i$ and die $i+1$, and $gap\_sum$ is the sum of all $gap_i$ values.
constraints: $\forall d \in 1..m: increasing([dice_{d,i} | i \in 1..n])$, $max\_val = max([dice_{i,j} | i \in 1..m, j \in 1..n])$, $max\_win = max([comp_{i,j} | i \in 0..m-1, j \in 1..2])$, $comp_{d,1} = \sum_{r1,r2 \in 1..n} (bool2int(dice_{1+(d mod m), r1} > dice_{1+((d + 1) mod m), r2}))$, $comp_{d,2} = \sum_{r1,r2 \in 1..n} (bool2int(dice_{1+((d+1) mod m), r1} > dice_{1+((d) mod m), r2}))$, $gap_d = comp_{d,1} - comp_{d,2}$, $\forall d \in 0..m-1: comp_{d,1} > comp_{d,2}$ 
These constraints ensure that the values on each die are in increasing order, the maximum value and maximum number of wins are calculated correctly, the number of wins for each die against its successor is calculated correctly, the difference in wins between each die and its successor is calculated correctly, and each die beats its successor.
objective: minimize $max\_val$
