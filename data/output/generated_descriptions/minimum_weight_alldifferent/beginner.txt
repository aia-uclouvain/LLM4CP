Minimum Weight All Different problem
The problem is about assigning different values to a set of variables in a way that minimizes the total cost. 
The main variables are: 
- variables: a set of values that must all be different from each other
- matrix: a table of costs where each cell represents the cost of assigning a particular value to a particular variable
- cost: the total cost of the assignments
The basic restrictions are: 
- each variable must have a unique value
- the cost of each assignment is determined by the matrix
The goal is to minimize the total cost by finding the optimal assignments of values to variables.
