name: Longest Change Problem
description: The Longest Change Problem is a constraint satisfaction problem where we are given a sequence of variables and a comparison operator, and we need to find the longest subsequence of consecutive variables where the comparison operator holds, and also find a sequence of variables that satisfies this condition.
variables: $x = (x_1, x_2, ..., x_n)$, $c$, where $x$ is an array of decision variables representing the sequence of values, and $c$ is a decision variable representing the length of the longest subsequence of consecutive variables where the comparison operator holds.
constraints: $x_i \in [1, 8]$ for $i = 1, 2, ..., n$, $c \in \mathbb{Z}$, $cp1d(x, [8, 8, 3, 4, 1, 1, 5, 5, 2])$, $longest\_change(c, x, 10)$, where the first constraint defines the domain of the variables, the second constraint defines the domain of $c$, the third constraint ensures that $x$ is equal to a given sequence, and the fourth constraint ensures that $c$ is equal to the length of the longest subsequence of consecutive variables where the comparison operator holds.
objective: The objective of the problem is to satisfy all the constraints, there is no explicit objective function to minimize or maximize.
