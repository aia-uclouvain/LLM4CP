name: Calculs d'Enfer Puzzle

description: The Calculs d'Enfer puzzle is a classic constraint programming problem where the goal is to assign integer values to a set of variables, representing letters from 'a' to 'z', such that a set of given equations are satisfied and the maximum absolute value of the assigned values is minimized.

variables: $a, b, c, d, e, f, g, h, i, j, k, l, m, n, o, p, q, r, s, t, u, v, w, x, y, z$ - These variables represent the letters from 'a' to 'z' and are assigned integer values from the set $R = \{-100, \ldots, 100\}$.

constraints: 
$A_{abs}[i] = |A[i]| \forall i \in \{1, \ldots, 26\}$, 
$\max(a_{max}, A_{abs})$, 
$all\_different(A)$, 
$z + e + r + o = 0$, 
$o + n + e = 1$, 
$t + w + o = 2$, 
$t + h + r + e + e = 3$, 
$f + o + u + r = 4$, 
$f + i + v + e = 5$, 
$s + i + x = 6$, 
$s + e + v + e + n = 7$, 
$e + i + g + h + t = 8$, 
$n + i + n + e = 9$, 
$t + e + n = 10$, 
$e + l + e + v + e + n = 11$, 
$t + w + e + l + f = 12$ 
- These constraints ensure that each variable is assigned a unique value, and the given equations are satisfied.

objective: Minimize $a_{max}$, which represents the maximum absolute value of the assigned values.
