name: Young Tableaux Problem
description: The Young Tableaux problem is a combinatorial problem that involves finding a standard Young tableau of a given size, which is a matrix filled with numbers from 1 to n in such a way that each row and column is non-decreasing, and each number appears exactly once. The problem also involves finding the partition structure of the tableau, which represents the number of elements in each row.
variables: $x_{i,j}, p_i$ where $i \in \{1, \ldots, n\}$ and $j \in \{1, \ldots, n\}$. The variables $x_{i,j}$ represent the elements of the Young tableau, where $x_{i,j}$ is the element in the $i$-th row and $j$-th column, and $p_i$ represents the number of elements in the $i$-th row of the tableau.
constraints: $\forall i \in \{1, \ldots, n\}: \sum_{a,b \in \{1, \ldots, n\}} (x_{a,b} = i) = 1$, $x_{1,1} = 1$, $\forall i \in \{1, \ldots, n\}, j \in \{2, \ldots, n\}: x_{i,j} \geq x_{i,j-1}$, $\forall j \in \{1, \ldots, n\}, i \in \{2, \ldots, n\}: x_{i,j} \geq x_{i-1,j}$, $\forall i \in \{1, \ldots, n\}: p_i = \sum_{j \in \{1, \ldots, n\}} (x_{i,j} \leq n)$, $\sum_{i \in \{1, \ldots, n\}} p_i = n$, $\forall i \in \{2, \ldots, n\}: p_{i-1} \geq p_i$. These constraints ensure that each number appears exactly once, the rows and columns are non-decreasing, and the partition structure is correctly calculated.
objective: The objective of the problem is to find a feasible solution that satisfies all the constraints, as indicated by the "solve satisfy" statement in the model. There is no explicit objective function to minimize or maximize.
