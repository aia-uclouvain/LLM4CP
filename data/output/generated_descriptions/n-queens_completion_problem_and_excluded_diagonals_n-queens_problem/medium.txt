N-Queens Problem
The N-Queens problem is a classic constraint programming problem where the goal is to place N queens on an NxN chessboard such that no two queens attack each other.

The main decision variables in this problem are the positions of the queens on the board, which can be represented as a matrix or a set of coordinates (x, y) for each queen.

The key constraints in this problem are:
- Each row and column can contain at most one queen.
- Each diagonal can contain at most one queen.
- No two queens can attack each other, meaning that no two queens can be in the same row, column, or diagonal.

The objective of the problem is to find a placement of N queens on the board that satisfies all the constraints, or to determine that no such placement exists.

In the context of the provided Minizinc models, the problem is often formulated as a satisfaction problem, where the goal is to find a solution that satisfies all the constraints, rather than optimizing a specific objective function.
