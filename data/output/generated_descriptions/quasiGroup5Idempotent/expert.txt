name: Quasigroup Existence Problem
description: The Quasigroup existence problem is a classic problem in constraint programming that involves finding an m-order quasigroup, which is an mxm multiplication table of integers 1..m. The goal is to fill in the table such that each element occurs exactly once in each row and column, and certain multiplication axioms hold.
variables: $quasiGroup_{i,j}$ where $i,j \in \{0,1,...,n-1\}$, which represent the decision variables that denote the value at row $i$ and column $j$ in the quasigroup table. These variables are used to construct the quasigroup table.
constraints: 
$\forall i \in \{0,1,...,n-1\}: \forall j \in \{0,1,...,n-1\}: quasiGroup_{i,j} \in \{0,1,...,n-1\}$, 
$\forall i \in \{0,1,...,n-1\}: \text{all\_different}([quasiGroup_{i,j}]_{j \in \{0,1,...,n-1\}})$, 
$\forall j \in \{0,1,...,n-1\}: \text{all\_different}([quasiGroup_{i,j}]_{i \in \{0,1,...,n-1\}})$, 
$\forall i \in \{0,1,...,n-1\}: \forall j \in \{0,1,...,n-1\}: quasiGroup_{quasiGroup_{quasiGroup_{i,j},j},j} = i$, 
$\forall i \in \{0,1,...,n-1\}: quasiGroup_{i,i} = i$, 
$\forall i \in \{0,1,...,n-1\}: \forall j \in \{0,1,...,n-1\}: (quasiGroup_{i,j} = i) \leftrightarrow (quasiGroup_{j,i} = i)$, 
$\forall i \in \{0,1,...,n-1\}: quasiGroup_{i,n-1} + 2 \geq i$, 
which represent the constraints that the quasigroup table must satisfy, including row and column uniqueness, multiplication axioms, idempotency, and symmetry-breaking constraints.
objective: The objective of the problem is to find a satisfying assignment of values to the decision variables, i.e., to construct a valid quasigroup table. There is no explicit objective function to minimize or maximize, as the goal is to find a feasible solution that satisfies all the constraints.
