Table Filling Problem
The problem is about filling a 4x6 table with a certain number of ticks (or marks) in such a way that each row and column has an even number of ticks, and no row or column is left empty.

The main variables are: 
- ticks: the total number of ticks to be placed in the table
- x: a 4x6 array where each cell can either have a tick (represented by 1) or not (represented by 0)

The basic restrictions or rules are: 
- the total number of ticks is fixed (in this case, 18)
- each row and column must have at least one tick
- the number of ticks in each row and column must be even
- no row or column can be empty

The goal of the problem is to find a way to fill the table with the given number of ticks while satisfying all the rules, and to do so in a way that minimizes the number of ticks used (although in this specific case, the number of ticks is fixed at 18).
