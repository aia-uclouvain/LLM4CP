name: Lehmer Code from Permutation
description: The problem involves finding the Lehmer code corresponding to a given permutation and restoring the original permutation from the Lehmer code. The Lehmer code is created by counting how many subsequent values each number in the permutation exceeds. The problem is to find the Lehmer code and restore the original permutation.
variables: $x = (x_1, x_2, ..., x_n)$, $y = (y_1, y_2, ..., y_n)$, where $x$ represents the Lehmer code and $y$ represents the restored original permutation, with $x_i$ being the number of subsequent values that $a_i$ exceeds and $y_i$ being the $i^{th}$ element in the restored permutation.
constraints: $\forall i \in \{1, 2, ..., n\}: x_i = \sum_{j=i}^{n} (a_j < a_i)$, $\forall i \in \{1, 2, ..., n\}: x_i = \sum_{j=i}^{n} (y_j < y_i)$, $\text{all\_different}(y)$, which ensure that the Lehmer code is correctly calculated, the original permutation is restored, and all elements in the restored permutation are distinct.
objective: satisfy, as the problem is a constraint satisfaction problem, where the goal is to find a solution that satisfies all the constraints, rather than minimizing or maximizing a specific value.
