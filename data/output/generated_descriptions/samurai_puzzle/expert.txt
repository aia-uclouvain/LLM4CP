name: Samuraï Puzzle
description: The Samuraï puzzle is a constraint satisfaction problem where the goal is to place a single 1 in each sqrt(n) by sqrt(n) sub-square of an n by n grid, such that each row, column, and sub-square contains exactly one 1.
variables: x = (x[1], x[2], ..., x[n]), y = (y[i, j]) for i, j in 1..n. 
These are decision variables where x[i] represents the column in which the 1 is placed in the i-th row, and y[i, j] is a 0/1 variable representing whether the cell at row i and column j contains a 1.
constraints: 
alldifferent(x), 
y[i, j] = 1 <-> x[i] = j for i, j in 1..n, 
sum(y[r, c] | r in i*m+1..i*m+m, c in j*m+1..j*m+m) = 1 for i, j in 0..m-1. 
These constraints ensure that each row contains exactly one 1, that the 0/1 variables y are consistent with the x variables, and that each sub-square contains exactly one 1.
objective: satisfy
