name: Primality Test Problem
description: The problem is a simple number theory problem that determines whether a given number is prime or not by calculating its divisors. A prime number is a natural number greater than 1 that has no positive divisors other than 1 and itself. The problem uses constraint programming to find the divisors of a given number and then checks if the number of divisors is greater than 0 to determine if the number is prime.
variables: $n, num\_divisors, divisor, is\_prime$, where $n$ is the input number to be checked for primality, $num\_divisors$ is the number of divisors of $n$, $divisor$ is an array of variables representing the divisors of $n$, and $is\_prime$ is a variable indicating whether $n$ is prime or not.
constraints: $divisors(n, num\_divisors, divisor)$, $is\_prime = 1 \leftrightarrow num\_divisors = 2$, $divisor[i] = 1 \leftrightarrow n \mod i = 0$, where the first constraint calculates the number of divisors of $n$, the second constraint checks if $n$ is prime based on the number of divisors, and the third constraint checks if each number $i$ is a divisor of $n$.
objective: The problem does not have a specific objective function to minimize or maximize, but rather it aims to satisfy the constraints and determine whether the given number is prime or not. The objective is to find a solution that satisfies all the constraints, which in this case is to determine the primality of the given number.
