Number Theory Problem: Primality Test
The problem is about determining whether a given number is prime or not by calculating its divisors.
The main decision variables are: 
- `num_divisors`: the total number of divisors of the given number `n`, 
- `divisor`: an array representing whether each number from 2 to `t` (where `t` is `(n div 2)+1`) is a divisor of `n`, 
- `is_prime`: a variable indicating whether the number `n` is prime or not.
The key constraints are: 
- each number from 2 to `t` is checked to see if it is a divisor of `n` (i.e., `n` modulo `i` equals 0), 
- the total number of divisors is calculated by summing up the divisors found, 
- a number is considered prime if it has no divisors other than 1 and itself, which is represented by `num_divisors` being equal to 0 (although this is not entirely accurate as 1 is not considered and the model does not account for the divisor 1, this seems to be an oversimplification).
The problem's objective is to satisfy the constraints, specifically to determine whether the given number `n` is prime or not, by calculating its divisors and checking the condition for primality.
