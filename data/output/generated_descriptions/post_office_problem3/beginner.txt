Post Office Problem
The problem is about a post office that needs to hire full-time employees to work on different days of the week. 
The main variables are: 
- the number of workers that start on each day 
- the number of workers that work on each day 
- the number of extra workers on each day 
- the total number of workers 
- the number of days each person works.
The basic restrictions are: 
- each full-time employee must work for a certain number of consecutive days 
- after working for those days, the employee gets two days off 
- the post office must meet its daily worker requirements using only full-time employees.
The goal is to minimize the total number of workers that must be hired to meet the post office's daily requirements.
