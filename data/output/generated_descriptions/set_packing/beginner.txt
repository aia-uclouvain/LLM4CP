Set Packing Problem
The problem is about finding the largest number of groups that do not overlap with each other from a given set of groups.
The main variables are: 
- x, which represents whether a group is chosen or not
- numNeeded, which represents the total number of chosen groups
- belongs, which represents the elements in each group.
The basic restrictions or rules of the problem are: 
- each element can only be in one chosen group
- the goal is to find the largest number of groups that do not overlap with each other.
The goal of the problem is to maximize the number of chosen groups, which is represented by numNeeded.
