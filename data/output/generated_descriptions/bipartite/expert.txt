name: Bipartite Graph Problem
description: The Bipartite Graph Problem is a classic problem in graph theory and constraint programming. It involves finding a partition of the nodes in a graph into two disjoint sets such that there are no edges between nodes in the same set. The problem can be used to model various real-world scenarios, such as scheduling, resource allocation, and network optimization.
variables: $s = (s_1, s_2, ..., s_n)$ and $b = (b_1, b_2)$, where $s_i$ represents the set of nodes connected to node $i$ and $b_j$ represents the set of nodes in the $j^{th}$ partition. The variables $s_i$ and $b_j$ are decision variables that need to be determined.
constraints: 
- $i \notin s_i$ for all $i \in 1..n$ 
- $|s_i| \geq 2$ for all $i \in 1..n$ 
- $j \in s_i \leftrightarrow i \in s_j$ for all $i, j \in 1..n$ 
- $b_1 \cup b_2 = 1..n$ 
- $b_1 \cap b_2 = \emptyset$ 
- $i \in b_1 \leftrightarrow \forall j \in 1..n (j \in s_i \rightarrow j \in b_2)$ for all $i \in 1..n$ 
- $i \in b_2 \leftrightarrow \forall j \in 1..n (j \in s_i \rightarrow j \in b_1)$ for all $i \in 1..n$, 
which ensure that the graph is symmetric, the nodes are partitioned into two disjoint sets, and there are no edges between nodes in the same set.
objective: The objective of the problem is to satisfy all the constraints, i.e., to find a feasible solution that partitions the nodes into two disjoint sets such that there are no edges between nodes in the same set. There is no explicit objective function to minimize or maximize.
