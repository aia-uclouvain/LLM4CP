name: Strimko
description: Strimko is a puzzle where the goal is to fill an nxn grid with numbers from 1 to n, such that each row, column, and "stream" (a connected path in the grid) contains each number exactly once. The puzzle is a variant of Latin squares with the added constraint of streams.
variables: $x_{i,j}, streams_{i,j}, num\_placed, placed_{i,j}$, where $x_{i,j}$ represents the number in the cell at row i and column j, $streams_{i,j}$ represents the stream that the cell at row i and column j belongs to, $num\_placed$ is the number of already placed numbers, and $placed_{i,j}$ represents the row, column, and value of a pre-placed number.
constraints: $\forall i \in 1..n: \forall j \in 1..n: x_{i,j} \in 1..n \land all\_different([x_{i,j} | j \in 1..n]) \land all\_different([x_{j,i} | j \in 1..n]) \land all\_different([x_{i,j} | i,j \in 1..n \land streams_{i,j} = s]) \land x_{placed_{i,1}, placed_{i,2}} = placed_{i,3}$, which ensure that each row and column contains each number exactly once, each stream contains each number exactly once, and the pre-placed numbers are respected.
objective: satisfy, which means the goal is to find a feasible solution that satisfies all the constraints, there is no objective function to minimize or maximize.
