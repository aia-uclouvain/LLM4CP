Color Sequence Optimization Problem
The problem involves optimizing an array of colors to minimize the number of color changes, while also considering the interchangeability of colors.
The main decision variables are:
- `colors`: the original array of colors
- `perm`: an array of permutation indices, where each index represents the position of the corresponding color in the optimal sequence
- `optimal`: the optimized array of colors, where the goal is to minimize the number of color changes
The key constraints are:
- The `permutation3` constraint ensures that the `perm` array represents a valid permutation of the `colors` array to produce the `optimal` array
- The symmetry-breaking constraint (commented out in the provided model) ensures that if two colors are the same in the `optimal` array, their corresponding permutation indices in the `perm` array are ordered, reducing the number of symmetric solutions
The objective of the problem is to find an optimal sequence of colors that minimizes the number of color changes, while also considering the interchangeability of colors, and to output the original colors array, the permutation indices array, and the optimized colors array.
