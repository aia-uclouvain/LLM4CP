name: Set Partition Problem
description: The Set Partition Problem is a classic problem in computer science and operations research that involves partitioning a set of integers into two or more subsets such that the sum of the elements in each subset is equal, and the sum of the squares of the elements in each subset is also equal.
variables: $a_{ij}$, where $a_{ij} = 1$ if element $j$ is in set $i$, and $0$ otherwise. These variables represent the assignment of each element to a set.
constraints: $\sum_{j=1}^{n} a_{ij} = \frac{n}{2}$ for each set $i$, $\sum_{j=1}^{n} j \cdot a_{ij} = \sum_{j=1}^{n} j \cdot a_{i'j}$ for each pair of sets $i$ and $i'$, $\sum_{j=1}^{n} j^2 \cdot a_{ij} = \sum_{j=1}^{n} j^2 \cdot a_{i'j}$ for each pair of sets $i$ and $i'$, and $\sum_{i=1}^{2} a_{ij} = 1$ for each element $j$. These constraints ensure that each set has the same number of elements, the sum of the elements in each set is equal, the sum of the squares of the elements in each set is equal, and each element is assigned to exactly one set.
objective: The objective of the problem is to find a feasible solution that satisfies all the constraints, as there is no explicit objective function to minimize or maximize.
