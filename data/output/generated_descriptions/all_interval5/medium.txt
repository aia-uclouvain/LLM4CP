All Interval Problem
The All Interval Problem is about finding a sequence of musical notes, represented by pitch classes, such that each note appears exactly once and the intervals between consecutive notes are all different.
The main decision variables are `series` and `differences`. The `series` variable represents the sequence of pitch classes, where each element in the array corresponds to a note in the sequence. The `differences` variable represents the intervals between consecutive notes, calculated as the absolute difference between the pitch classes of two adjacent notes.
The key constraints are: each pitch class occurs exactly once in the sequence, the intervals between consecutive notes are all different, and the first note is less than the last note. In plain language, this means that the sequence must contain all the notes exactly once, the distance between each pair of consecutive notes must be unique, and the sequence must start with a note that has a lower pitch than the last note.
The objective of the problem is to find a sequence that satisfies all the constraints, which is a satisfaction problem rather than an optimization problem. In other words, the goal is to find a valid sequence, not to minimize or maximize a particular value. The problem is solved using a constraint programming approach, where the constraints are used to guide the search for a valid solution.
