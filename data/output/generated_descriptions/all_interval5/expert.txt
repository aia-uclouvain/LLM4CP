name: All Interval Problem
description: The All Interval Problem is a well-known problem in music theory and constraint programming. It involves finding a sequence of musical notes, represented by integers from 0 to n-1, such that each note occurs exactly once and the intervals between consecutive notes are all different. This problem has applications in music composition and is a classic example of a constraint satisfaction problem.
variables: $series = [s_0, s_1, ..., s_{n-1}], differences = [d_0, d_1, ..., d_{n-2}]$, where $series$ is a permutation of the integers from 0 to n-1, representing the sequence of notes, and $differences$ is an array of variables representing the intervals between consecutive notes.
constraints: $\forall i, j \in classes, i \neq j: s_i \neq s_j, \forall i \in 0..n-2: d_i = |s_{i+1} - s_i|, \forall i, j \in 0..n-2, i \neq j: d_i \neq d_j, s_0 < s_{n-1}$, which ensure that each note occurs exactly once, the intervals between consecutive notes are all different, and the sequence is symmetrically broken by ordering the first note to be less than the last one.
objective: The objective of the problem is to find a satisfying assignment, i.e., a sequence of notes that satisfies all the constraints, rather than minimizing or maximizing a specific value. The problem is a constraint satisfaction problem, and the goal is to find a solution that meets all the constraints.
