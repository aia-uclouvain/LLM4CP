Students and Languages problem
The problem is about five students from different countries - Poland, Hungary, Finland, Sweden, and Germany - who are on vacation together. Each student speaks their native language and possibly some other languages. The goal is to figure out which languages each student can speak, given some rules about how they communicate with each other.
The main variables are the students and the languages they speak. There are five students (Polish, Hungarian, Finnish, Swedish, and German) and five languages (the same as the students' nationalities). The variable "Students" is a table that shows which languages each student can speak.
The basic restrictions are: each student speaks at least one foreign language, there is no language that all students speak, each student can communicate with every other student in some language, and some specific rules about which students speak which languages. For example, the Pole and the Hungarian speak three foreign languages, and the Swede has to leave the group for the others to have a common language.
The goal of the problem is to find a solution that satisfies all the given rules and restrictions, so we can determine which languages each student can speak.
