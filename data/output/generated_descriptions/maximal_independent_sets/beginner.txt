Maximal Independent Set problem
The problem is about finding a group of nodes in a graph that do not connect to each other, and this group is as large as possible.
The main variables are: 
- n, the number of nodes in the graph
- v, a list of neighboring nodes for each node
- x, a list of decision variables where x[i] is 1 if node i is in the maximal independent set and 0 otherwise
- k, the size of the maximal independent set
The basic restrictions or rules of the problem are: 
- No two nodes in the maximal independent set can be connected by an edge
- The maximal independent set must be as large as possible
The goal of the problem is to maximize k, the size of the maximal independent set.
