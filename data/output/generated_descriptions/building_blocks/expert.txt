name: Building Blocks Puzzle
description: The problem is to determine the arrangement of letters on four alphabet blocks, each with six sides, such that various words can be spelled by arranging the blocks in different ways. The blocks contain every letter of the alphabet except Q and Z, and the goal is to figure out how the letters are arranged on the blocks to spell a given set of words.
variables: $dice[j]$ for $j = 1..24$, where $dice[j]$ represents the block number (1-4) that the $j^{th}$ letter is on. These decision variables determine the arrangement of letters on the blocks.
constraints: $\forall i \in 1..12 ( \text{alldifferent}([dice[words[i,j]] | j \in 1..4]) ) \wedge global\_cardinality(dice, [i | i \in 1..4], [6 | i \in 1..4]) \wedge dice[1] \leq dice[7] \leq dice[13]$, which ensure that each word can be spelled using letters from different blocks, each block has exactly 6 letters, and symmetry is broken to reduce the search space.
objective: The objective of the problem is to find a satisfying assignment of letters to blocks, such that all the given words can be spelled. The problem is a satisfaction problem, meaning that there is no objective function to minimize or maximize, but rather a set of constraints that must be satisfied.
