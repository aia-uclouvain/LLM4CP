name: Shortest Path Problem (SPP)
description: The Shortest Path Problem is a classic problem in graph theory and operations research that involves finding the path with the minimum total weight or length between two nodes in a weighted directed graph. The problem is defined by a set of nodes and edges, where each edge has a weight or length associated with it. The goal is to find the path from a source node to a target node that minimizes the total weight or length.

variables: $x = (x_1, x_2, ..., x_{num\_edges})$, $z$ 
These are decision variables where $x_i$ represents whether the $i^{th}$ edge is included in the shortest path (1 if included, 0 otherwise), and $z$ represents the total length of the shortest path.

constraints: $\forall i \in \{1, 2, ..., n\}: \sum_{k \in \{1, 2, ..., num\_edges\} | E[k,2] = i} x_k + (i = s) = \sum_{k \in \{1, 2, ..., num\_edges\} | E[k,1] = i} x_k + (i = t)$
These constraints are conservation conditions for unity flow from the source node $s$ to the target node $t$, ensuring that every feasible solution is a path from $s$ to $t$.

objective: minimize $z = \sum_{i \in \{1, 2, ..., num\_edges\}} c_i \cdot x_i$
The objective is to minimize the total length of the path, which is calculated by summing the weights of all edges included in the path.
