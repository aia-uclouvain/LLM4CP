Furniture Offer Optimization
The problem is about buying furniture from a store that offers different packages of items, such as armchairs, sidechairs, and benches, at various prices. The goal is to find the best combination of packages to buy in order to get at least a certain number of each item while spending the least amount of money.

The main variables are:
- to_buy: the number of each package to buy
- prefs: the minimum number of each item (armchairs, sidechairs, benches) that we want to buy
- offer_price: the price of each package
- mat: a table that shows how many of each item are included in each package

The basic restrictions or rules are:
- We must buy at least the minimum number of each item specified in prefs
- We can only buy a whole number of packages (we can't buy a fraction of a package)

The goal of the problem is to minimize the total cost (Total) by finding the best combination of packages to buy.
