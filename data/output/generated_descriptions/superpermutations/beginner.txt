Superpermutation problem
The problem is about finding the shortest string that contains all possible orderings of a set of symbols. 
The main variables are: 
- x, which represents the string of symbols, 
- ps, which represents the starting position of each permutation in the string, 
- n, which is the number of symbols, 
- z, which is the length of the string.
The basic restrictions are: 
- the string must contain all possible orderings of the symbols, 
- each permutation must appear in the string, 
- the string should be as short as possible.
The goal is to minimize the length of the string, which is represented by z.
