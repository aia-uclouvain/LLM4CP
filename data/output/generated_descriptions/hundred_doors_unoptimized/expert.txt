name: 100 Doors Problem
description: The problem involves 100 doors in a row, initially closed. The task is to determine the state of the doors after 100 passes, where in each pass, every nth door is toggled, and n increases from 1 to 100.
variables: $x_{i,j}, s$ - $x_{i,j}$ is a 100x100 matrix of decision variables where $x_{i,j}$ represents the state of the jth door after the ith pass (0 for closed, 1 for open), and $s$ is a set of door numbers that are open after the last pass.
constraints: $x_{1,j} = 1, \forall j \in r$; $x_{pass-1,j} = 1 \leftrightarrow x_{pass,j} = 0 \land x_{pass-1,j} = 0 \leftrightarrow x_{pass,j} = 1, \forall pass \in 2..n, \forall j \in r$ such that $j \mod pass = 0$; $x_{pass,j} = x_{pass-1, j}, \forall pass \in 2..n, \forall j \in r$ such that $j \mod pass \neq 0$; $x_{n, j} == 1 \leftrightarrow j \in s, \forall j \in r$ - These constraints initialize the doors to open in the first pass, toggle the doors in subsequent passes, and relate the final state of the doors to the set s.
objective: The problem does not have a specific objective to minimize or maximize, but rather to find the state of the doors after the last pass, which can be determined by the set $s$ of door numbers that are open.
