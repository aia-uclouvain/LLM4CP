name: Fizz Buzz Problem
description: The Fizz Buzz problem is a classic problem in computer science and programming that involves printing numbers from 1 to a given number, replacing multiples of 3 with "fizz", multiples of 5 with "buzz", and numbers that are multiples of both 3 and 5 with "fizzbuzz". This version of the problem also considers numbers that contain the digits 3 or 5 as "fizz" or "buzz" respectively.
variables: $x = (x_1, x_2, ..., x_n)$, where $x_i$ is a decision variable that represents the output for the number $i$, with $x_i = 0$ indicating the number itself, $x_i = 1$ indicating "fizz", $x_i = 2$ indicating "buzz", and $x_i = 3$ indicating "fizzbuzz".
constraints: 
$\forall i \in \{1, 2, ..., n\}:$
- $(i \mod 3 = 0 \land i \mod 5 = 0) \lor (contains(3, a) \land contains(5, a)) \lor (i \mod 3 = 0 \land contains(5, a)) \lor (i \mod 5 = 0 \land contains(3, a)) \leftrightarrow x_i = 3$
- $(i \mod 3 = 0 \lor contains(3, a)) \land (i \mod 5 > 0 \land \neg contains(5, a)) \leftrightarrow x_i = 1$
- $(i \mod 5 = 0 \lor contains(5, a)) \land (i \mod 3 > 0 \land \neg contains(3, a)) \leftrightarrow x_i = 2$
- $(i \mod 3 > 0 \land i \mod 5 > 0 \land \neg contains(3, a) \land \neg contains(5, a)) \leftrightarrow x_i = 0$
These constraints ensure that the output for each number $i$ is correctly determined based on its multiples and digits.
objective: The objective of this problem is not to minimize or maximize a specific value, but rather to find a solution that satisfies all the given constraints, effectively generating the correct Fizz Buzz sequence.
