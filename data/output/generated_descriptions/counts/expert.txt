name: Global Constraint Counts Problem
description: The problem is a constraint programming model that represents the global constraint counts, which enforces a condition on the number of variables assigned to a specific value. The model is used to demonstrate the counts constraint, where the goal is to find an assignment of values to variables such that the number of variables assigned to a specific value satisfies a certain condition.
variables: $m, n, variables_{1..n}, values_{1..m}, limitx$ - $m$ and $n$ are parameters representing the number of values and variables, respectively. $variables_{1..n}$ is an array of decision variables representing the assignment of values to variables, $values_{1..m}$ is an array of decision variables representing the specific values, and $limitx$ is a decision variable representing the limit on the number of variables assigned to a specific value.
constraints: $cp1d([1,3,4,9], values) \wedge cp1d([4,5,5,4,1,5], variables) \wedge limitx = 3 \wedge counts(values, variables, 0, limitx)$ - The constraints ensure that the values and variables are assigned as specified, the limit on the number of variables assigned to a specific value is set to 3, and the counts constraint is enforced, which checks that the number of variables assigned to a specific value is equal to the limit.
objective: satisfy - The objective of the problem is to find a satisfying assignment of values to variables that satisfies all the constraints, rather than minimizing or maximizing a specific value.
