name: Quasigroup Existence Problem
description: The Quasigroup existence problem is a classic problem in constraint programming that involves finding an m x m multiplication table of integers 1..m, where each element occurs exactly once in each row and column, and certain multiplication axioms hold. In this specific case, the problem is looking for an order m quasigroup for which the axiom (a*b)*(b*a) = a holds.
variables: $quasiGroup_{i,j}, qgDiagonal_i$ - These are the decision variables of the problem, where $quasiGroup_{i,j}$ represents the value at the i-th row and j-th column of the quasigroup table, and $qgDiagonal_i$ represents the value at the i-th position of the diagonal of the quasigroup table. The $quasiGroup_{i,j}$ variables are used to construct the quasigroup table, while the $qgDiagonal_i$ variables are used to access the diagonal elements of the table.
constraints: 
$\forall i \in nDomain: qgDiagonal_i = quasiGroup_{i,i}$, 
$\forall row \in nDomain: all\_different([quasiGroup_{row,col} | col \in nDomain])$, 
$\forall col \in nDomain: all\_different([quasiGroup_{row,col} | row \in nDomain])$, 
$\forall i \in nDomain: \forall j \in nDomain: quasiGroup_{quasiGroup_{i,j},quasiGroup_{j,i}} = i$ 
- These constraints ensure that the quasigroup table is constructed correctly. The first constraint links the diagonal variables to the corresponding elements in the quasigroup table. The second and third constraints ensure that each row and column of the quasigroup table contains all different elements. The fourth constraint enforces the axiom (a*b)*(b*a) = a.
objective: The objective of the problem is to find a satisfying assignment of values to the decision variables, i.e., to find a quasigroup table that satisfies all the constraints. The problem does not have a specific objective function to minimize or maximize, but rather seeks to find a feasible solution that satisfies all the constraints.
