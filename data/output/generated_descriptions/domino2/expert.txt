name: Domino Problem
description: The Domino Problem is a classic problem in constraint programming where the goal is to place domino pieces on a board such that each piece covers two fields and has two numbers, and each field on the board matches the number of the domino piece placed on that field.
variables: $x_{i,j}$, where $i \in 1..r$ and $j \in 1..c$, which represent the decision variables denoting the piece to place in each cell $(i, j)$ on the board.
constraints: $\forall i \in 1..r, \forall j \in 1..c, \exists a, b \in \{-1, 0, 1\}$ such that $i+a \geq 1, j+b \geq 1, i+a \leq r, j+b \leq c, |a| + |b| = 1$, and $(pieces[x_{i,j},1] = board[i,j] \land pieces[x_{i+a,j+b},2] = board[i+a,j+b]) \lor (pieces[x_{i+a,j+b},1] = board[i+a,j+b] \land pieces[x_{i,j},2] = board[i,j])$ and $x_{i,j} = x_{i+a,j+b}$, and $\forall p \in 1..m, \exists i \in 1..r, \exists j \in 1..c, \exists a, b \in \{-1, 0, 1\}$ such that $i+a \geq 1, j+b \geq 1, i+a \leq r, j+b \leq c, |a| + |b| = 1$, and $(pieces[p,1] = board[i,j] \land pieces[p,2] = board[i+a,j+b]) \lor (pieces[p,1] = board[i+a,j+b] \land pieces[p,2] = board[i,j])$ and $x_{i,j} = p$ and $x_{i+a,j+b} = p$, which ensure that each piece is placed correctly and that all pieces are used.
objective: The objective of the problem is to satisfy all the constraints, i.e., to find a feasible solution that places all domino pieces on the board correctly, and there is no explicit objective function to minimize or maximize.
