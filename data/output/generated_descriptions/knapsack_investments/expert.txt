name: Knapsack (Investment) Problem
description: The problem is about a company that needs to invest in some building projects with certain limits on budget, personnel, and number of projects. Each project has a value, budget, and personnel requirement, and there are also constraints on which projects can be selected together.
variables: $x_1, x_2, ..., x_{15}$, where $x_i$ is a decision variable representing whether project $i$ is selected (1) or not (0). These variables determine which projects to invest in.
constraints: $total\_budget \leq max\_budget$, $total\_persons \leq max\_persons$, $total\_projects \leq max\_projects$, $x_{requires[i, 1]} - x_{requires[i, 2]} \leq 0$, $x_{not\_with[i, 1]} + x_{not\_with[i, 2]} \leq 1$, where $total\_budget = \sum_{i=1}^{15} x_i \* budgets_i$, $total\_persons = \sum_{i=1}^{15} x_i \* personell_i$, $total\_projects = \sum_{i=1}^{15} x_i$. These constraints represent the limits on budget, personnel, and number of projects, as well as the requirements that certain projects must be selected together and that certain projects cannot be selected together.
objective: Maximize $total\_values = \sum_{i=1}^{15} x_i \* values_i$, which represents the total value of the selected projects.
