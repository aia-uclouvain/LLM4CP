Knapsack Problem
The problem is about a company that wants to invest in some building projects, but it has limited resources such as budget, personnel, and a maximum number of projects it can select. 
The main variables are: 
- x: which projects to select 
- total_persons: total number of personnel used 
- total_budget: total budget used 
- total_projects: total number of projects selected 
- total_values: total value of the selected projects.
The basic restrictions or rules of the problem are: 
- the total budget used cannot exceed the available budget 
- the total number of personnel used cannot exceed the available personnel 
- the total number of projects selected cannot exceed the maximum allowed 
- some projects cannot be selected together 
- some projects require other projects to be selected.
The goal of the problem is to maximize the total value of the selected projects while satisfying all the restrictions.
