Resource Constrained Scheduling Problem
The problem is about scheduling a set of tasks with given durations and resource requirements, such that the total end time is minimized without exceeding a limited resource capacity.
The main decision variables are: 
- `End`: the maximum end time of all tasks, 
- `LO` (origin): the start time of each task, 
- `LE` (end times): the end time of each task.
The key constraints are: 
- each task's end time is the sum of its start time and duration, 
- the maximum end time `End` is the maximum of all task end times, 
- the cumulative resource usage of all tasks at any time does not exceed the resource limit.
The objective of the problem is to minimize the maximum end time `End`, which represents the earliest time when all tasks can be completed without violating the resource constraint.
