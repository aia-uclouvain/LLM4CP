Car Painting Problem
The problem is about finding the best order to paint cars in a factory to minimize the number of times the paint color needs to be changed, which is expensive and time-consuming.
The main variables are: 
- x, which represents the order in which the cars are painted
- num_purgings, which is the number of times the paint color needs to be changed
- distance, which is the total distance that cars are moved from their original position
The basic restrictions are: 
- each car can only be painted once
- a car must be painted within three positions of its original position in the production line
- the paint color can only be changed a certain number of times
The goal is to minimize the number of paint color changes, or purgings, while still following the rules about where each car can be painted.
