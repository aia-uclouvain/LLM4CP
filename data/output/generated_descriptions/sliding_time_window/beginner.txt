Sliding Time Window Problem
The problem is about scheduling tasks within a certain time frame. 
The main variables are: 
- tasks: a list of tasks, where each task has a start time and a duration
- window_size: the size of a time window
- limitx: the maximum amount of tasks that can be done at the same time
- sums: the total tasks done in each time window
The basic restrictions are: 
- each task must start and end within a certain time frame (max_time)
- the total tasks done in any time window cannot exceed the limit (limitx)
The goal of the problem is to find a schedule for the tasks such that the total tasks done in any time window do not exceed the limit, given the start time and duration of each task, and the size of the time window.
