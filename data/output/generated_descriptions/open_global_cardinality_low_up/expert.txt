name: Global Cardinality Problem with Lower and Upper Bounds
description: The Global Cardinality Problem with Lower and Upper Bounds is a constraint satisfaction problem where we have a set of variables, each assigned a value from a given set, and a set of values with their corresponding lower and upper bounds. The goal is to find an assignment of values to variables such that each value is assigned to at least its lower bound and at most its upper bound of variables.
variables: $s, variables, values$ - $s$ is a set of indices, $variables$ is an array of decision variables where each $variables_i$ represents the value assigned to the $i^{th}$ variable, and $values$ is a 2D array where each $values_{ij}$ represents the $j^{th}$ attribute of the $i^{th}$ value (with $values_{i1}$ being the value itself, $values_{i2}$ being the lower bound, and $values_{i3}$ being the upper bound).
constraints: $s = \{2,3,4\}, variables = [3,3,8,6], open\_global\_cardinality\_low\_up(s, variables, values)$ - The first two constraints fix the set $s$ and the variables array, while the $open\_global\_cardinality\_low\_up$ constraint ensures that each value in $values$ is assigned to at least its lower bound and at most its upper bound of variables in the set $s$.
objective: satisfy - The objective of this problem is to find a satisfying assignment of values to variables that meets all the given constraints.
