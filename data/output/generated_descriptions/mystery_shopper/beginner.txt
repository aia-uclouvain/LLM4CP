The problem is called Mistery Shopper. 
It's about assigning visitors to visitees in a way that meets certain rules, like making sure each visitor sees a different visitee every week and that each visitee is seen by a different visitor every week. 
The main variables are: 
- r, which represents the visitor for each visitee at each week, 
- e, which represents the visitee for each visitor at each week, 
- rg, which represents the visitor group for each visitee at each week, 
- eg, which represents the visitee group for each visitor at each week, 
- vr_sizes and ve_sizes, which represent the sizes of the visitor and visitee groups. 
The basic restrictions are: 
- each week, all visitors must be different for each visitee, 
- each week, all visitees must be different for each visitor, 
- the visitor groups must be different for each visitee, 
- the visitee groups must be different for each visitor. 
The goal of the problem is to find an assignment of visitors to visitees that satisfies these rules.
