name: Numberlink problem
description: The Numberlink problem is a classic puzzle where the goal is to connect pairs of identical numbers in a grid by drawing a path between them, such that each path does not intersect with any other path. The puzzle starts with a grid containing some given numbers, and the task is to find a way to connect all the pairs of numbers.
variables: $SubM_{i,j} \in \{1, 2, ..., NP\}$, where $SubM_{i,j}$ represents the number assigned to the cell at row $i$ and column $j$ in the grid, and $NP$ is the total number of pairs of numbers.
constraints: $\forall i \in \{1, 2, ..., NR\}, j \in \{1, 2, ..., NC\}: (InputM_{i,j} > 0 \rightarrow SubM_{i,j} = InputM_{i,j}) \wedge (InputM_{i,j} = 0 \rightarrow \sum_{(i',j') \in N(i,j)} (SubM_{i',j'} = SubM_{i,j}) = 2)$, where $N(i,j)$ is the set of neighboring cells of $(i,j)$, and $\forall i \in \{1, 2, ..., NR\}, j \in \{1, 2, ..., NC\}: InputM_{i,j} \neq 0 \rightarrow \sum_{(i',j') \in N(i,j)} (SubM_{i',j'} = SubM_{i,j}) = 1$, which ensure that each cell is assigned a number, and that each number is connected to exactly two neighboring cells, except for the given numbers which are connected to exactly one neighboring cell.
objective: satisfy, which means that the goal is to find a feasible solution that satisfies all the constraints, without any optimization objective.
