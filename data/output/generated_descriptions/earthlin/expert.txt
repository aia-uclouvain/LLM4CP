name: Earthlings Puzzle
description: The Earthlings puzzle is a classic problem in constraint programming. It involves three teams (Zaire, Uruguay, and Spain) and three places (1st, 2nd, and 3rd). Each team has a specific type (truth-teller, alternator, or liar) and makes a statement about the placement of the teams. The goal is to determine the placement of the teams and the type of each team based on the given statements.
variables: $x_{i,j}, y_{k,l}, d_k$ where $x_{i,j}$ represents whether team $i$ is in place $j$, $y_{k,l}$ represents whether statement $k$ is made by type $l$, and $d_k$ represents the number of truths in statement $k$. These variables are used to model the placement of the teams, the type of each team, and the truthfulness of each statement.
constraints: $\forall j \sum_{i} x_{i,j} = 1, \forall i \sum_{j} x_{i,j} = 1, \forall k \sum_{l} y_{k,l} = 1, \forall l \sum_{k} y_{k,l} = 1, x_{1,1}+x_{2,2}+x_{3,3} = d_1, x_{1,1}+x_{3,2}+x_{2,3} = d_2, x_{2,1}+x_{3,2}+x_{1,3} = d_3, d_k - 3y_{k,1} \geq 0, d_k - 3y_{k,1} \leq 2, d_k + 3y_{k,3} \leq 3, d_k + y_{k,3} \geq 1, x_{1,1}=x_{3,3}, x_{1,1}=x_{2,3}, x_{2,1}=x_{1,3}$. These constraints ensure that each place has one team, each team has one place, each type makes one statement, each statement is made by one type, and the number of truths in each statement is consistent with the placement of the teams and the type of each team.
