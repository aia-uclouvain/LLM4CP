The Earthlings puzzle 
is a classic constraint programming problem. 
The problem is about determining the placement of three teams (Zaire, Uruguay, and Spain) in a competition, where each team has a specific type (truth-teller, alternator, or liar), and each team makes a statement about the placement of the teams. 

The main decision variables are x(i,j), which represents whether team i is in place j, and y(k,l), which represents whether statement k is made by type l. 

The key constraints are: 
- each place can only be occupied by one team, 
- each team can only occupy one place, 
- each type makes one statement, 
- each statement is made by one type, 
- the number of truths in each statement is consistent with the placement of the teams, 
- the statements made by truth-tellers and liars are consistent with their types, 
- and the statements made by alternators are consistent with their type. 

The objective of the problem is to find a solution that satisfies all the constraints, which is achieved by using the "solve satisfy" statement in the Minizinc model. 
In other words, the problem is a feasibility problem, where the goal is to find a feasible solution that meets all the constraints, rather than optimizing a specific objective function.
