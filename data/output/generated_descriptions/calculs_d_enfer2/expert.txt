name: Calculs d'Enfer Puzzle
description: The Calculs d'Enfer puzzle is a classic constraint programming problem where the goal is to assign integer values to a set of variables representing letters, such that a set of constraints are satisfied. The puzzle consists of a set of equations, where each equation is represented by a word, and the sum of the letters in each word corresponds to a specific value. The objective is to find an assignment of values to the letters that minimizes the maximum absolute value of the letters.

variables: $a, b, c, d, e, f, g, h, i, j, k, l, m, n, o, p, q, r, s, t, u, v, w, x, y, z, A, A_{abs}, a_{max}$ 
These variables represent the letters in the puzzle, where $a$ to $z$ are the decision variables representing the letters, $A$ is an array of these variables, $A_{abs}$ is an array of the absolute values of $A$, and $a_{max}$ is the maximum absolute value of $A$. The $words$ array is also a decision variable, but it is defined by the problem and not a free variable.

constraints: $\forall I \in 1..26 ( A_{abs}[I] = |A[I]| ) \wedge maximum(a_{max}, A_{abs}) \wedge all\_different(A) \wedge \forall W \in 0..12 ( \sum_{I=1}^{6} words[W,I] = W )$
These constraints ensure that the absolute values of the letters are calculated correctly, the maximum absolute value is tracked, all letters have different values, and the sum of the letters in each word corresponds to the specified value.

objective: minimize $a_{max}$ 
The objective of the problem is to minimize the maximum absolute value of the letters, which means finding an assignment of values to the letters that minimizes the largest absolute value among all letters.
