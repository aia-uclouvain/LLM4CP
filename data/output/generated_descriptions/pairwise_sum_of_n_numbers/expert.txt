name: Pairwise Sum Problem

description: The Pairwise Sum Problem is a classic problem in constraint programming where we are given a set of pairwise sums of n numbers in non-decreasing order, and we need to identify the individual numbers. If the sum is corrupted, the problem should print -1.

variables: $x = [x_1, x_2, ..., x_m]$ where $x_i$ is a decision variable representing the i-th individual number. These variables are the numbers that we are trying to find, given their pairwise sums.

constraints: 
- $x_i + x_j = input_k$ for $1 \leq k \leq n$, $1 \leq i < j \leq m$ (each pairwise sum is equal to the corresponding input value)
- $x_1 \leq x_2 \leq ... \leq x_m$ (the individual numbers are in non-decreasing order, for symmetry breaking)

objective: The objective of the problem is to find the values of $x$ that satisfy all the constraints, i.e., to find the individual numbers given their pairwise sums. The problem does not have a specific objective function to minimize or maximize, but rather to find a feasible solution that satisfies all the constraints.
