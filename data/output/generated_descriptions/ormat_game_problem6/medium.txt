The Ormat Game problem
The Ormat Game is a classic constraint programming problem that involves finding a solution to a puzzle where a set of rules must be satisfied. 
The main decision variables in the Ormat Game problem are the values assigned to each cell in the grid, which represent whether a cell is filled (1) or empty (0).
The key constraints in the Ormat Game problem are that each row and column must have the same number of filled cells, and that the filled cells must be contiguous. 
The objective of the Ormat Game problem is to find a solution that satisfies all the constraints, with no specific objective function being minimized or maximized, as the goal is to simply find a feasible solution.
