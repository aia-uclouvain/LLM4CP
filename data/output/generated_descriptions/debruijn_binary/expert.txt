name: De Bruijn Sequence Problem
description: The De Bruijn sequence problem is a classical problem in combinatorics and constraint programming that involves finding a sequence of symbols, typically binary digits, such that every possible substring of a certain length appears exactly once in the sequence. This problem has applications in various fields, including coding theory, cryptography, and computer science.
variables: $x, binary, bin\_code, gcc$ 
These variables represent the decision variables in the problem: $x$ is an array of integers representing the sequence of numbers, $binary$ is a 2D array representing the binary representation of the numbers, $bin\_code$ is an array representing the sequence in base-representation, and $gcc$ is an array representing the number of occurrences of each value in $bin\_code$.
constraints: $all\_different(x)$, $minimum(x[1], x)$, $toNum(t, x[i], base)$, $binary[i-1, j] = binary[i, j-1]$, $binary[m, j] = binary[1, j-1]$, $global\_cardinality(bin\_code, array1d(0..base-1, [i | i in 0..base-1]), gcc)$, $gcc[i] = gcc[i-1]$, $gcc[i] = m div base$ 
These constraints ensure that: all numbers in the sequence are different, the minimum element is the first element, the binary representation of each number is correct, the De Bruijn condition is satisfied (i.e., the first elements in $binary[i]$ is the same as the last elements in $binary[i-1]$), the sequence wraps around correctly, the number of occurrences of each value in $bin\_code$ is counted correctly, and when $m$ is a multiple of $base$, the number of occurrences of each value is equal.
objective: The objective of the problem is to find a satisfying assignment of values to the decision variables, i.e., to find a De Bruijn sequence that satisfies all the constraints. There is no explicit objective function to minimize or maximize.
