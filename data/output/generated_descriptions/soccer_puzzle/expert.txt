name: Soccer Puzzle
description: The problem involves determining the number of points the fifth team scored in a soccer tournament where five teams played against each other once, with a win giving 3 points, a tie giving 1 point, and a loss giving 0 points, given that four of the teams scored 1, 2, 5, and 7 points.
variables: $x_1, x_2, ..., x_5, matches_{i,j}$ where $i, j = 1, 2, ..., 5$. These variables represent the total points scored by each team ($x_i$) and the outcome of each match between teams $i$ and $j$ ($matches_{i,j}$), where $matches_{i,j}$ is 3 if team $i$ wins, 1 if it's a tie, and 0 if team $i$ loses.
constraints: $x_i = res_i$ for $i = 1, 2, 3, 4$, $matches_{i,i} = 0$ for $i = 1, 2, ..., 5$, $(matches_{i,j} = 3 \leftrightarrow matches_{j,i} = 0)$ and $(matches_{i,j} = 1 \leftrightarrow matches_{j,i} = 1)$ for $i \neq j$, $x_i = \sum_{j=1}^{5} matches_{i,j}$ for $i = 1, 2, ..., 5$. These constraints ensure that the given teams' scores are respected, a team does not play against itself, the outcome of each match is consistent, and the total points for each team are calculated correctly.
objective: The objective of this problem is not to minimize or maximize a specific value, but rather to find a feasible solution that satisfies all the given constraints, specifically to determine the number of points the fifth team scored.
