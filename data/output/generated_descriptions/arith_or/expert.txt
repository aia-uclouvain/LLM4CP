name: Arithmetic Or Problem
description: The Arithmetic Or problem is a constraint satisfaction problem that involves finding values for two arrays of variables, x and y, such that for each pair of corresponding elements in x and y, at least one of the elements is equal to a given value z. The problem is modeled using the global constraint arith_or, which enforces this condition for all pairs of variables.
variables: $x = (x_1, x_2, ..., x_n), y = (y_1, y_2, ..., y_n), z$, where $x$ and $y$ are arrays of binary decision variables, and $z$ is an integer decision variable. These variables represent the values to be assigned to the elements of the arrays $x$ and $y$, and the value $z$ that is used in the comparison.
constraints: $x = (0,1,0,0,1), y = (0,0,0,1,0), \forall i \in \{1,2,...,n\}: x_i = 0 \lor y_i = 0, z = 0$, where the first two constraints assign fixed values to the elements of $x$ and $y$, the third constraint is the arith_or constraint that enforces the condition that for each pair of corresponding elements in $x$ and $y$, at least one of the elements is equal to $z$, and the last constraint assigns a fixed value to $z$.
objective: satisfy, which means that the objective is to find an assignment of values to the decision variables that satisfies all the constraints.
