Krypto Puzzle
The Krypto puzzle is a mathematical game where players are given a set of numbers and must use basic arithmetic operations to combine them and reach a target number.
The main decision variables in this problem are: 
- `cardsp`: an array representing the order in which the numbers should be used, 
- `x`: an array representing the intermediate results of the arithmetic operations, and 
- `ops`: an array representing the operations to be performed between the numbers.
The key constraints in this problem are: 
- The numbers must be used in a specific order (unless the `ordered` variable is set to `false`), 
- Each number can only be used once, 
- The operations must be basic arithmetic operations (addition, subtraction, multiplication, and division), and 
- The final result must equal the target number.
The objective of this problem is to find a sequence of arithmetic operations that combines the given numbers to reach the target number, while satisfying the constraints.
