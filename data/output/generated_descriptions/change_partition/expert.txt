name: Change Partition Problem

description: The Change Partition Problem is a constraint satisfaction problem that involves finding a partitioning of a set of variables into disjoint subsets, such that the number of changes between consecutive variables that do not belong to the same partition is equal to a given value.

variables: $nchange, variables = [x_1, x_2, ..., x_{11}], partitions = [P_1, P_2, P_3]$ 
These are the decision variables of the problem, where $nchange$ is an integer variable representing the number of changes between consecutive variables, $variables$ is an array of integer variables representing the values of the variables, and $partitions$ is an array of set variables representing the disjoint subsets.

constraints: $nchange = n - \sum_{i=2}^{11} \sum_{j=1}^{3} (x_{i-1} \in P_j \wedge x_i \in P_j) - 1$, 
$partition\_set2(partitions, \bigcup_{j=1}^{3} P_j)$, 
$\forall j \in \{1,2,3\} (|P_j| > 0)$, 
$variables = [6,6,2,1,3,3,1,6,2,2,2]$, 
$nchange = 2$ 
These constraints ensure that the number of changes between consecutive variables is equal to $nchange$, that the partitions are disjoint and cover all variables, that each partition is non-empty, that the variables are assigned the given values, and that the number of changes is equal to 2.

objective: satisfy 
The objective of the problem is to find a solution that satisfies all the constraints, without any optimization criterion.
