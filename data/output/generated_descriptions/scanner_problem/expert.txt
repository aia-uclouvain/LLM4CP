name: Scanner Problem
description: The Scanner Problem is a constraint satisfaction problem where the goal is to reveal the contents of a grid based on given information about the number of filled squares along each row, column, and diagonal. The grid can have any size, and each square can be either filled or non-filled. The problem is to find a configuration of the grid that satisfies all the given constraints.
variables: $x_{i,j}$ where $i \in 1..num\_rows$ and $j \in 1..num\_cols$, these are binary decision variables representing whether the square at row $i$ and column $j$ is filled (1) or not (0).
constraints: 
$\forall i \in 1..num\_rows: \sum_{j=1}^{num\_cols} x_{i,j} = rows_i$,
$\forall j \in 1..num\_cols: \sum_{i=1}^{num\_rows} x_{i,j} = columns_j$,
$\forall k \in 1..num\_diagonal1: \sum_{i=1}^{num\_rows} \sum_{j=1}^{num\_cols} x_{i,j} \cdot (i+j=k+1) = diagonal1_k$,
$\forall k \in 1..num\_diagonal2: \sum_{i=1}^{num\_rows} \sum_{j=1}^{num\_cols} x_{i,j} \cdot (i+(num\_cols-j+1)=k+1) = diagonal2_k$,
these constraints ensure that the number of filled squares in each row, column, and diagonal matches the given input.
objective: The objective of the problem is to find a satisfying assignment, i.e., to solve the problem such that all constraints are satisfied, there is no explicit objective function to minimize or maximize.
