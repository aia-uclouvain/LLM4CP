name: All Different Pairs Problem
description: The All Different Pairs Problem is a combinatorial problem where we need to find all possible pairs of distinct elements from a set of n elements, such that all pairs are different and the pairs are in increasing order. The problem also ensures that the first element of each pair is not equal to the second element.
variables: $x_{k,1}, x_{k,2}$ for $k = 1$ to $m$, where $m = \frac{n(n-1)}{2}$, $x_{k,1}$ and $x_{k,2}$ are decision variables representing the elements of each pair, with $x_{k,1}$ and $x_{k,2}$ being integers between 1 and $n$.
constraints: $\forall k \in 1..m, x_{k,1} \neq x_{k,2}$, $all\_different\_pairs(x, n)$, $increasing\_pairs(x, n)$, where $all\_different\_pairs(x, n)$ ensures that all pairs are different and $increasing\_pairs(x, n)$ ensures that the pairs are in increasing order.
objective: The objective of the problem is to find all possible assignments of values to the decision variables $x_{k,1}$ and $x_{k,2}$ that satisfy the given constraints, with no explicit objective function to minimize or maximize. The problem is a satisfaction problem, where the goal is to find all feasible solutions.
