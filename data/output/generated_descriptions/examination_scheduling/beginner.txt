Examination Scheduling Problem
The problem is about scheduling exams for students in a way that meets certain rules. 
The main variables are: 
- s: a table showing which exam is in which room at what time
- xrooms: the room where each exam will take place
- xslots: the time slot when each exam will take place
- xdays: the day of the week when each exam will take place
- z: the number of rooms used
The basic restrictions or rules of the problem are: 
- each exam must be assigned to a room and time slot
- students should not have more than two exams on the same day
- some exams need to be in larger rooms
The goal of the problem is to find a schedule that meets all the rules and uses the fewest number of rooms possible.
