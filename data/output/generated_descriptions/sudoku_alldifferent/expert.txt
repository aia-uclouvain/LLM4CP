name: Sudoku
description: Sudoku is a classic puzzle where the goal is to fill a 9x9 grid with numbers from 1 to 9, such that each row, column, and 3x3 sub-grid contains each number exactly once. The puzzle starts with some numbers already filled in, and the task is to fill in the remaining empty cells.
variables: $x_{i,j}$, where $i$ and $j$ range from 1 to $n$, representing the value of the cell at row $i$ and column $j$. These variables are decision variables that need to be assigned a value from 1 to $n$.
constraints: $\forall i \in \{1, \ldots, n\}: \text{alldifferent}(\{x_{i,j} | j \in \{1, \ldots, n\}\}) \land \forall j \in \{1, \ldots, n\}: \text{alldifferent}(\{x_{i,j} | i \in \{1, \ldots, n\}\}) \land \forall i, j \in \{0, \ldots, \text{reg}-1\}: \text{alldifferent}(\{x_{r,c} | r \in \{i\cdot\text{reg}+1, \ldots, i\cdot\text{reg}+\text{reg}\}, c \in \{j\cdot\text{reg}+1, \ldots, j\cdot\text{reg}+\text{reg}\}\})$, which ensure that each row, column, and 3x3 sub-grid contains each number exactly once.
objective: The objective of the problem is to find a feasible solution that satisfies all the constraints, as there is no explicit objective function to minimize or maximize. The goal is to find a valid Sudoku puzzle solution.
