name: Indexed Sum Problem
description: The Indexed Sum Problem is a constraint satisfaction problem where we have a set of items, each with a fixed index and a weight, and a table with summation variables. The goal is to assign each item to an entry in the table such that the sum of the weights of the items assigned to that entry equals the corresponding summation variable.
variables: $x_{table} = [x_1, x_2, ..., x_n]$ and $items = [item_{11}, item_{12}, ..., item_{nm}]$, where $x_{table}$ is an array of decision variables representing the summation variables in the table, and $items$ is a 2D array where each row represents an item with its index and weight.
constraints: $x_{table}[i] = \sum_{j \in index\_set(x_{table})} items[j, 2] \cdot (items[j, 1] = i)$ for all $i$ and $x_{table} = [6, 0, -3]$, where the first constraint ensures that the sum of the weights of the items assigned to each entry in the table equals the corresponding summation variable, and the second constraint specifies the target values for the summation variables.
objective: satisfy, The objective of the problem is to find an assignment of items to the table that satisfies all the constraints, rather than minimizing or maximizing a specific value.
