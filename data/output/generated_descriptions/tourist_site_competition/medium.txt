Tourist Site Competition Problem
The problem is about assigning judges to tourist sites in a way that satisfies certain constraints, such as each site being visited by a specific number of judges and each judge visiting a specific number of sites.

The main decision variables are represented by the 2D array `x`, where `x[s, j]` is a binary variable that equals 1 if judge `j` visits site `s`, and 0 otherwise. The auxiliary variable `judges_where` represents the set of sites visited by each judge.

The key constraints are: 
- Every tourist site is visited by exactly `r` judges.
- Every judge visits exactly `c` tourist sites.
- Every pair of sites is visited by exactly `lambda` common judge.
- Symmetry breaking constraints are also applied to reduce the search space, such as assigning specific judges to specific sites.

The objective of the problem is to find an assignment of judges to sites that satisfies all the given constraints, which is a satisfaction problem, not an optimization problem. In other words, the goal is to find a feasible solution that meets all the requirements, rather than minimizing or maximizing a specific objective function.
