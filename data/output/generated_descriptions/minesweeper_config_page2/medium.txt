Minesweeper Problem
The Minesweeper problem is about finding the location of mines in a grid-based game board, given some clues about the number of mines adjacent to each cell.
The main decision variables in this problem represent the state of each cell in the grid, typically a binary value indicating whether a cell contains a mine or not, and sometimes an additional variable to represent the number of mines adjacent to each cell.
The key constraints in the Minesweeper problem include: each cell can either contain a mine or be empty, the number of mines adjacent to each cell must match the given clues, and the total number of mines in the grid is fixed. Additionally, the constraints ensure that the mines are placed in a way that is consistent with the rules of the game, such as not placing a mine in a cell that is already known to be empty.
The objective of the Minesweeper problem is not to minimize or maximize a specific value, but rather to find a feasible solution that satisfies all the constraints, effectively "solving" the game by identifying the location of all mines. In other words, the goal is to reconstruct the original game board, given the partial information provided by the clues.
