name: Curve Fitting Problem
description: The Curve Fitting Problem is a classic problem in mathematical optimization and regression analysis. It involves finding the best-fitting linear line to a set of given data points, where the best fit is defined as the line that minimizes the maximum deviation between the observed and predicted values. The problem has applications in various fields, including statistics, engineering, and economics, where it is used to model relationships between variables and make predictions.

variables: $a, b, z, u_i, v_i$ 
These variables represent the parameters of the linear regression line ($a$ and $b$), the maximum deviation ($z$), and the deviations between the observed and predicted values for each data point ($u_i$ and $v_i$), where $i$ indexes the data points.

constraints: $u_i \geq 0, v_i \geq 0, b \cdot x_i + a + u_i - v_i = y_i, z - u_i \geq 0, z - v_i \geq 0$ 
These constraints ensure that the deviations are non-negative, the linear equation is satisfied for each data point, and the maximum deviation is greater than or equal to the individual deviations.

objective: minimize $z$ 
The objective of the problem is to minimize the maximum deviation ($z$) between the observed and predicted values, which corresponds to finding the best-fitting linear line to the given data points.
