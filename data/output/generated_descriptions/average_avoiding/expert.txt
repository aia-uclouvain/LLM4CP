name: Average Avoiding Problem

description: The Average Avoiding Problem is a constraint satisfaction problem where the goal is to arrange a given array of numbers in such a way that the average of any two numbers does not lie between them. In other words, for any two numbers in the array, their average should not be equal to any other number in the array that is between them.

variables: $x = [x_1, x_2, ..., x_n]$, where $x_i$ is a decision variable representing the $i^{th}$ element in the arranged array, and $x_i \in \{a_1, a_2, ..., a_n\}$. These variables represent the permutation of the input array $a$.

constraints: 
$\forall i, j \in \{1, 2, ..., n\}, i < j: \forall k \in \{i+1, i+2, ..., j-1\}: 2x_k \neq x_i + x_j$
$\min(x_1, x)$, 
$\forall v \in s: \text{count}(x, v) = \text{count}(a, v)$, 
where $s$ is the set of unique elements in the input array $a$. 
These constraints ensure that the average of any two numbers does not lie between them, the array is permuted, and the count of each number in the permuted array is the same as in the original array.

objective: The objective of the problem is to find a satisfying assignment, i.e., to find an arrangement of the input array that satisfies all the constraints. The problem does not have a specific objective function to minimize or maximize.
