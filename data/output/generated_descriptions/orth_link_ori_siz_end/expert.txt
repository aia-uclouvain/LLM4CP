name: Orthogonal Link Ori Siz End Problem
description: The Orthogonal Link Ori Siz End Problem is a constraint satisfaction problem that involves enforcing a specific relationship between the origin, size, and end points of a set of orthotopes. An orthotope is a rectangular box in n-dimensional space, and in this problem, we are dealing with 3-dimensional orthotopes. The problem requires that for each orthotope, the sum of its origin and size equals its end point.
variables: $orthotope_{i,1}, orthotope_{i,2}, orthotope_{i,3}$ for $i = 1, \ldots, n$, where $orthotope_{i,1}$, $orthotope_{i,2}$, and $orthotope_{i,3}$ represent the origin, size, and end point of the $i^{th}$ orthotope, respectively. These variables are decision variables that need to be assigned values such that the constraints of the problem are satisfied.
constraints: $orthotope_{i,3} = orthotope_{i,1} + orthotope_{i,2}$ for $i = 1, \ldots, n$, which ensures that the sum of the origin and size of each orthotope equals its end point. Additionally, there are constraints that enforce the values of the orthotopes to be equal to a given set of values.
objective: The objective of the problem is to satisfy all the constraints, i.e., to find an assignment of values to the decision variables such that all the constraints are satisfied. There is no explicit objective function to minimize or maximize.
