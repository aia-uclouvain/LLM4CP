name: Domain Constraint Problem
description: The Domain Constraint Problem is a constraint programming problem that involves linking a domain variable to a set of 0-1 variables, each associated with a potential value of the domain variable. The problem ensures that the 0-1 variable corresponding to the value taken by the domain variable is equal to 1, while the remaining 0-1 variables are all equal to 0.
variables: $var01, values, val$ - $var01$ is an array of 0-1 decision variables, $values$ is an array of decision variables representing potential values, and $val$ is a decision variable representing the value taken by the domain variable. These variables are used to model the relationship between the domain variable and its potential values.
constraints: $var01[i] = 1 \land val = values[i], \sum(var01) = 1, cp1d(var01, [0,1,0,0]), cp1d(values,[9,5,2,7]), val = 5$ - These constraints ensure that the 0-1 variable corresponding to the value taken by the domain variable is equal to 1, the sum of all 0-1 variables is equal to 1, and the given values and domain variable are consistent with the problem statement.
objective: satisfy - The objective of the problem is to find a satisfying assignment of values to the decision variables that satisfies all the given constraints.
