Anniversaries Problem
The Anniversaries problem is a classic constraint programming problem that involves determining the husband and wife that make up each couple and the number of years they have been married, given certain clues about their relationships and marriage lengths.

The main decision variables in this problem are the assignments of husbands, wives, and marriage lengths to each surname. Specifically, the variables represent the following:
- `husbands`: an array of variables representing the assignment of each husband's given name to a surname
- `wifes`: an array of variables representing the assignment of each wife's given name to a surname
- `married`: an array of variables representing the number of years each couple has been married

The key constraints in this problem can be described in plain language as follows:
- Each husband and wife can only be assigned to one surname
- No two couples can have been married for the same number of years
- The given clues about the relationships and marriage lengths must be satisfied, such as Joyce not being married as long as Charles or the Parkers, but longer than Douglas or the Johnstones

The objective of this problem is not to minimize or maximize a specific value, but rather to find a satisfying assignment of husbands, wives, and marriage lengths to each surname that satisfies all the given constraints. The problem is solved using a satisfaction-based approach, where the goal is to find a solution that meets all the constraints, rather than optimizing a specific objective function.
