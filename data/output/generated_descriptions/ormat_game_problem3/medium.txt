The Ormat Game problem.
The Ormat Game is a classic constraint programming problem that involves finding a solution to a puzzle where a set of rules must be satisfied. 
The main decision variables in the Ormat Game problem are the elements of a 2D array, where each element represents a cell in the puzzle and can take on a value of either 0 or 1.
The key constraints in the Ormat Game problem are that each row and column must have an equal number of 0s and 1s, and that no two adjacent cells (horizontally or vertically) can have the same value. 
The objective of the Ormat Game problem is to find a solution that satisfies all the constraints, with no specific minimization or maximization goal, as the problem is a satisfaction problem where the goal is to find a feasible solution.
