Planet Colonization Problem
The Planet Colonization problem is a classic constraint programming problem that involves finding the best landing site on a newly discovered planet to minimize the total distance to four essential resources: food, oxygen, water, and energy.

The main decision variables in this problem are the landing site and the selected resource sites. The landing site is a variable that represents the area where the colony will be established, and the selected resource sites are variables that represent the specific areas where the resources are located.

The key constraints in this problem are:
- The landing site must be one of the 20 areas on the planet.
- Each resource (food, oxygen, water, and energy) must be selected from its respective set of possible locations.
- The total distance from the landing site to each of the selected resource sites must be calculated.
- The landing site can be any area, but it is desirable to minimize the total distance to all resources.

The objective of the problem is to minimize the total distance from the landing site to all four resources. This is achieved by selecting the landing site and the resource sites that result in the shortest total distance. The problem is solved using a constraint programming approach, where the constraints are modeled and solved using a solver to find the optimal solution.
