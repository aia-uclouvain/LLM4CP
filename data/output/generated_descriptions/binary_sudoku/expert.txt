name: Binary Sudoku
description: The Binary Sudoku problem is a variant of the classic Sudoku puzzle, where the goal is to fill an NxN matrix with 0s and 1s, such that each row and column contains an equal number of 0s and 1s, no two rows or columns are identical, and there are no sequences of 3 or more consecutive 0s or 1s in rows or columns.
variables: $x_{i,j}$, where $i \in [1..n]$ and $j \in [1..n]$. These variables represent the value of the cell at row $i$ and column $j$ in the matrix, which can be either 0 or 1.
constraints: 
$\forall i \in [1..n], \sum_{j=1}^{n} x_{i,j} = k$,
$\forall i \in [1..n], \sum_{j=1}^{n} x_{j,i} = k$,
$\forall i \in [1..n], sliding\_sum(1, 2, 3, [x_{i,j} | j \in [1..n]])$,
$\forall i \in [1..n], sliding\_sum(1, 2, 3, [x_{j,i} | j \in [1..n]])$,
$\forall i \in [1..n], \forall j \in [i+1..n], lex\_ne([x_{i,c} | c \in [1..n]], [x_{j,c} | c \in [1..n]])$,
$\forall i \in [1..n], \forall j \in [i+1..n], lex\_ne([x_{c,i} | c \in [1..n]], [x_{c,j} | c \in [1..n]])$.
These constraints ensure that each row and column has an equal number of 0s and 1s, no sequences of 3 or more consecutive 0s or 1s, and that no two rows or columns are identical.
objective: The objective of the problem is to find a satisfying assignment of values to the decision variables $x_{i,j}$, such that all constraints are satisfied. The problem is a satisfaction problem, meaning that there is no objective function to minimize or maximize, but rather a set of constraints that must be satisfied.
