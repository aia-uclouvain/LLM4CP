name: Least Squares Curve Fitting
description: The problem is a curve fitting problem where we have a set of data points representing temperature and percentage gas, and we want to find the best fit curve to this data using a polynomial equation of degree 4. The goal is to minimize the sum of the squared differences between the observed data points and the predicted values from the curve.

variables: $a_0, a_1, a_2, a_3, a_4, z$ 
These are the decision variables where $a_0, a_1, a_2, a_3, a_4$ represent the coefficients of the polynomial equation $a_0 + a_1t + a_2t^2 + a_3t^3 + a_4t^4$ that best fits the data, and $z$ is the sum of the squared differences between the observed data points and the predicted values.

constraints: 
$a_0 + 20a_1 + 400a_2 + 8000a_3 + 160000a_4 = 0$
$a_0 + 700a_1 + 490000a_2 + 343000000a_3 + 240100000000a_4 = 100$
$a_1 + 2a_2t + 3a_3t^2 + 4a_4t^3 \geq 0 \quad \forall t$
These constraints represent the conditions that the curve must satisfy: the first two constraints are specific points that the curve must pass through, and the third constraint ensures that the derivative of the curve is non-negative for all temperatures.

objective: minimize $z$ 
The objective is to minimize the sum of the squared differences between the observed data points and the predicted values from the curve, which represents the best fit of the curve to the data.
