name: Conditional Lexicographic Cost Problem
description: The Conditional Lexicographic Cost Problem is a constraint satisfaction problem where the goal is to assign a vector to a specific position in a preference table based on certain conditions. The problem involves finding a vector that matches a specific item in the preference table, and the position of this item in the table is determined by a conditional lexicographic cost constraint.
variables: $n, m, arr, v1, pos$ - $n$ and $m$ are parameters representing the size of the arrays, $arr$ is a 2D array of variables representing the preference table, $v1$ is a 1D array of variables representing the vector to be assigned, and $pos$ is a variable representing the position of the assigned vector in the preference table.
constraints: $cp2d([|1,0,|0,1,|0,0,|1,1|], arr)$, $cond\_lex\_cost(v1, arr, pos)$ - The first constraint $cp2d$ ensures that the array $arr$ is equal to a given 2D array, and the second constraint $cond\_lex\_cost$ ensures that the vector $v1$ is assigned to the $pos^{th}$ item in the array $arr$.
objective: satisfy - The objective of the problem is to find an assignment of values to the decision variables that satisfies all the given constraints.
