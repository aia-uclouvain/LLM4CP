name: Job Shop Scheduling Problem
description: The problem is to schedule a set of tasks with given durations and deadlines on a single machine, such that all tasks are completed before their respective deadlines and the last task finishes as early as possible.
variables: $start\_times_i, end\_times_i, last\_time$ where $i = 1, \ldots, n$. These variables represent the start time, end time, and the last time a task is completed, respectively. $start\_times_i$ and $end\_times_i$ are arrays of decision variables representing the start and end times of each task, and $last\_time$ is a decision variable representing the time when the last task is completed.
constraints: $\forall i: end\_times_i = start\_times_i + duration_i$, $\forall i: end\_times_i \leq deadline_i$, $\forall i, j: end\_times_i \leq start\_times_j \lor start\_times_i \geq end\_times_j$, $last\_time \geq end\_times_i$. These constraints ensure that each task is completed within its duration, before its deadline, and that tasks do not overlap. The last constraint ensures that $last\_time$ is the maximum end time of all tasks.
objective: minimize $last\_time$
