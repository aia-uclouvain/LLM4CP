Task Scheduling Problem
The problem is about scheduling tasks in a way that they are completed before their deadlines. 
The main variables are: 
- start_times: when each task starts
- end_times: when each task ends
- duration: how long each task takes
- deadline: the latest time each task can be completed
The basic restrictions are: 
- each task must start after the previous one has finished
- each task must be completed before its deadline
The goal is to find a schedule that allows all tasks to be completed as soon as possible, minimizing the time it takes to complete all tasks.
