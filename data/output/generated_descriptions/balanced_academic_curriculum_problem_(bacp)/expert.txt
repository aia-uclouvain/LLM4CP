name: Balanced Academic Curriculum Problem
description: The Balanced Academic Curriculum Problem is a scheduling problem where a set of courses with prerequisites must be assigned to a limited number of periods. Each course has a certain load, and students have lower and upper bounds on the number of credits and courses they can take in a given period. The goal is to assign a period to every course satisfying these criteria, minimizing the load for all periods.
variables: $s_c, x_{p,c}, load_p, objective$ where $s_c$ represents the period assigned to course $c$, $x_{p,c}$ is a binary variable indicating whether course $c$ is assigned to period $p$, $load_p$ is the total load for period $p$, and $objective$ is the optimization target. These variables are used to model the assignment of courses to periods and the resulting loads.
constraints: $\forall p \in periods: \sum_{c \in courses} x_{p,c} \geq courses\_per\_period\_lb \wedge \sum_{c \in courses} x_{p,c} \leq courses\_per\_period\_ub$, $\forall p \in periods: load_p = \sum_{c \in courses} x_{p,c} \* course\_load_c$, $\forall p \in periods: load_p \geq load\_per\_period\_lb \wedge load_p \leq load\_per\_period\_ub$, $\forall c_1, c_2 \in courses: prerequisite(c_1, c_2) \rightarrow s_{c_1}  s_{c_2}$ where these constraints ensure that the number of courses per period is within the given bounds, the load for each period is calculated correctly, the load for each period is within the given bounds, and courses are assigned to periods respecting the prerequisites.
objective: minimize $objective$ which represents the maximum load for all periods, or in some variants, minimize $objective$ which represents the difference between the maximum and minimum load for all periods.
