The Balanced Academic Curriculum Problem
The problem is about creating a schedule for a set of university courses with prerequisites, where each course has a certain number of credits. 
The main variables are: 
- the period (or semester) to which each course is assigned
- the number of courses in each period
- the total number of credits in each period
These variables represent the schedule of courses and the distribution of credits across periods.
The basic restrictions or rules of the problem are: 
- each course must be assigned to a period
- a course cannot be scheduled before its prerequisites
- the number of credits in each period must be within certain limits
- the number of courses in each period must also be within certain limits
The goal of the problem is to assign a period to every course while satisfying these rules and minimizing the maximum number of credits in any period.
