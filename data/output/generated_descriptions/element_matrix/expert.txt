name: Element Matrix Problem
description: The Element Matrix Problem is a constraint satisfaction problem that involves finding the value of a specific element in a given matrix. The problem is defined by a set of constraints that specify the structure of the matrix and the relationships between its elements.
variables: $index_i, index_j, matrix, value$ - These decision variables represent the row index, column index, the matrix itself, and the value of the element at the specified position in the matrix, respectively.
constraints: $cp2d(matrix, array2d(1..max_i,1..max_j, [4,1,7,1,0,8,3,2,1,0,0,6])) \wedge index_i = 1 \wedge index_j = 3 \wedge value = 7 \wedge element\_matrix(index_i, index_j, matrix, value)$ - These constraints specify that the matrix must be equal to a given matrix, the row and column indices must be 1 and 3, respectively, the value of the element at the specified position must be 7, and the element at the specified position in the matrix must be equal to the given value.
objective: The objective of the problem is to satisfy all the given constraints, which means finding a matrix that satisfies the specified relationships and has the given element at the specified position. The problem does not involve minimizing or maximizing a specific value.
