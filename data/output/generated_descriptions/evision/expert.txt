name: Equal Vision puzzle
description: The Equal Vision puzzle is a classic problem in constraint programming where the goal is to place watchmen in a grid such that each watchman can see a certain number of vacant cells, while minimizing the total number of vacant cells. The puzzle is defined on a square grid of size 'size' where each cell can either be occupied by a watchman or be vacant.
variables: $x_{i,j}, n_{i,j}, sumx$ 
These variables represent the state of each cell in the grid, where $x_{i,j}$ is 1 if the cell at position $(i,j)$ is vacant and 0 otherwise, $n_{i,j}$ is the number of vacant cells visible to the watchman at position $(i,j)$, and $sumx$ is the total number of vacant cells in the grid.
constraints: $\forall i,j \in S: \sum_{m \in S, m \neq i, m-i+j \geq 1, m-i+j \leq size} x_{m,m-i+j} + \sum_{m \in S, m \neq i, i+j-m \geq 1, i+j-m \leq size} x_{m,i+j-m} + \sum_{m \in S, m \neq i} x_{m,j} + \sum_{m \in S, m \neq j} x_{i,m} = n_{i,j}$, 
$\forall i,j \in S: n_{i,j} \geq cvis - 99x_{i,j}$, 
$\forall i,j \in S: n_{i,j} \leq cvis + 99x_{i,j}$
These constraints ensure that the number of vacant cells visible to each watchman is correctly calculated, and that the number of visible vacant cells is within the allowed range.
objective: minimize $sumx$ 
The objective of the problem is to minimize the total number of vacant cells in the grid, which is represented by the variable $sumx$.
