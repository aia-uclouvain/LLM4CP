Discrepancy Problem
The Discrepancy problem is about finding an assignment of values to a set of variables such that a specified number of variables take on values from their respective sets of "bad" values.
The main decision variables are: 
- `variables`: an array of variables representing the values to be assigned, 
- `bad`: an array of sets representing the "bad" values for each variable, and 
- `k`: an integer representing the target number of variables that should take on values from their respective sets of "bad" values.
The key constraints are: 
- each variable can only take on a value from a specified range (in this case, 1 to 5), 
- the `bad` sets are predefined for each variable, and 
- the `discrepancy` constraint ensures that exactly `k` variables take on values from their respective `bad` sets.
The objective of the problem is to satisfy all the given constraints, specifically to find an assignment of values to the `variables` such that exactly `k` of them take on values from their respective `bad` sets, where `k` is given as 2 in this instance.
